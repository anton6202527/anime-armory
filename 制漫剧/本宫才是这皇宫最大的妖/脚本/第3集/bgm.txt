# 本宫才是这皇宫最大的妖_第3集_BGM与音效_精修版_v1

## 整体情绪曲线

夜半试术的屏息诡异（潜行成了）→ 同源相斥反噬的一记刺痛警告 → "吃一个强一分"想通规则的沉静笃定 → 潜行夜探、暗金纹发烫的危机潜伏 → 老槐树下三妖分食死猫的惊悚奇观 → "收租=吃人""张才人被当租子"的真相寒意 → 冷静盘算只吃一只 → 集尾走出树影、平淡宣战的杀气定格（动作钩）。

## BGM 风格（按镜头段落）

- 镜头1-3（夜半试术·潜行成）：承第2集集尾兽吼引子余韵，极简——单根古琴/箫的清冷长音 + 心跳般的低频点，屏息感。镜头3"潜行·成了"给一记轻而亮的能量微光音，作小爽点，不要做大。
- 镜头4-5（同源相斥反噬）：BGM 骤然一紧，弦乐刺起一记短促刺痛音；系统重低音机械古音入场（带兽吼底噪），"贪……则反噬"尾音压住、留毛骨悚然气口。
- 镜头6-7（想通规则·吃一个强一分）：节奏由紧转沉稳，低弦缓推，"吃一个，强一分"两短句逐句落定，立"克制取舍"的笃定感（非燃爽，是清醒）。
- 镜头8-9（潜行夜探·妖气发烫）：换悬疑潜行底色，低频脉动 + 细碎环境夜声（风过枯枝、远处更漏）；镜头9暗金纹发烫处叠一记低频嗡鸣，危机潜伏抬头。
- 镜头10-11（三妖分食·惊悚奇观）：阴森诡异主题——不和谐弦乐 + 啃咬撕咬的湿黏音效垫底，妖气幽幽。给三妖外形特写留惊悚气口。
- 镜头12-13（三妖对话·收租）：BGM 退到最低，突出三妖沙哑非人怪声对白；"收租""轮到姓张的才人"几个词点出，不抢词。
- 镜头14-15（真相·被当租子）：弦乐一记下沉重音托住"收租"二字；镜头15"是被当成了租子"——所有声音骤收、留一拍死寂，寒意爬上来（情绪点，不靠音量靠留白）。
- 镜头16（冷静盘算）：低频脉动微微推进，像在数数，克制、压着，蓄势。
- 镜头17（集尾宣战·动作钩）："这租子，本宫替张才人，退了"——前半句平淡近乎耳语，"退了"二字落下瞬间一记低沉杀气重音 + 金芒亮开的能量音，戛然而止留静默，勾第4集开打。

## 关键音效点

- [镜头2] 无声起身、踩过碎瓷却"零声响"——刻意抽掉脚步声，只留布料极轻摩挲（潜行的听觉表现）。
- [镜头3] "潜行·成了"——一记轻亮的能量微光音 + 影子淡化的虚化音。

- [镜头5] 系统重低音机械音入场 + 古老兽吼底噪，"贪则反噬"尾音悬住。
- [镜头9] 暗金妖纹发烫——低频嗡鸣一线，妖气逼近的预警。
- [镜头10-11] 啃咬撕咬的湿黏怪响 + 妖气幽幽底噪。
- [镜头12-13] 三妖沙哑非人怪声对白，"收租"二字咬重。
- [镜头15] "是被当成了租子"后——全声收住、留一拍死寂。
- [镜头17] "退了"落下：低沉杀气重音 + 金芒亮开能量音，戛然静默（第4集开打引子）。
