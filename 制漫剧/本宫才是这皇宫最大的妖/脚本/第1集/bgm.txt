# 本宫才是这皇宫最大的妖_第1集_BGM与音效_精修版

## 整体情绪曲线

冷开场毒酒压迫(死亡倒计时) → 铜镜认脸的异样死寂 → 原身已死/懿旨仍到的信息钩 → 逼酒抵唇的窒息 → 一笑留白憋住 → 万妖血脉觉醒重低音 drop(本集最大爽点) → 金瞳宣言后硬断留钩。

## BGM 风格（分段，对齐镜头）

- 镜头1-2（冷开场·0-6s）：**无前奏**。低频弦乐一记闷压下来，叠白绫摩擦、匕首金属冷响、酒壶轻放三声。一上来就死亡压迫，不给铺垫。
- 镜头3-6（认脸·信息钩）：国风暗黑氛围乐，古琴单音 + 远处夜风 + 铜镜轻颤嗡鸣。节奏极慢，留白突出"这张脸不是我的"与"原身已死"两记钩子。
- 镜头7-13（倒计时·张力阶梯）：心跳式低鼓逐拍加速，弦乐持续上行。小禾报信→柳娘进门→逼酒推进，鼓点越来越密、越来越近，像催命倒计时。
- 镜头14-16（憋·抽真空）：音乐骤然抽低，只留高频耳鸣 + 残烛爆芯声，给"鸩酒""再杀一次""你笑什么"三句留窒息停顿。**爽点前的留白**。
- 镜头17-19（觉醒·💥爽点 drop，01:41.400-01:55.603）：上古妖兽低吟 + 重低音 drop 砸下。镜头19"可吞噬、可炼化、可夺其术"是全集最大爽点落点，鼓点与暗金妖纹炸亮**同帧**。
- 镜头20-24（宣言·集尾钩，01:55.603-02:16.803）：drop 余韵转低沉悬念弦乐单音。镜头22宣言尾字"算"做低频重音，镜头24"晚了"后留 0.8s 静默，再接下集预告音效（兽吼一线）。

## 关键音效点

- [镜头1] 托盘三连：白绫轻落 → 匕首冷响 → 酒壶咚声。
- [镜头2] 近距离急促呼吸 + 心跳第一下突起。
- [镜头3] 铜镜轻颤、镜面嗡鸣。
- [镜头4-5] 记忆潮涌耳鸣 + 远处宫门闷响。
- [镜头7] 门被撞开 + 小禾踉跄跪地、衣料摩擦。
- [镜头8] 宫门外脚步逼近 + 尖细传唤拖尾音。
- [镜头9] 深蓝宫装袖摆扫过门槛，托盘上三件器物轻晃。
- [镜头10-11] 小禾被拖开、闷哭被手掌捂住。
- [镜头13-14] 鸩酒壶嘴抵近唇边、酒液轻晃；苦杏仁味用一记高频细响"点"出来。
- [镜头15] 音乐抽真空，残烛"啪"地爆芯。
- [镜头17] 系统音入场：重低音机械声 + 古老兽吼底噪。
- [镜头18] 妖纹滋滋流光、血脉灼烧低鸣，疤口裂开"咔"一线脆响。
- [镜头19] 💥爽点卡点："夺其术"三字落下时 BGM drop + 暗金妖纹炸亮同帧。
- [镜头20] 柳娘倒吸冷气，瓷壶脱手坠地碎裂。
- [镜头22] 宣言尾字"算"低频重音 + 烛火被无风压灭一瞬。
- [镜头24] "晚了"后 0.8s 静默 → 下集预告兽吼一线。

## 卡点备注（供 n2d-compose）

- **最大爽点**：镜头19，系统"可吞噬、可炼化、可夺其术" → 实际时间码 01:52.012-01:54.903（占屏到 01:55.603）。BGM drop / 重音效优先对齐"夺其术"尾音。
- **集尾钩**：镜头24"晚了" → 实际时间码 02:13.763-02:16.803，**不展示反杀动作**（反杀+噬妖留第2集冷开场）。
- 镜头14-16 的留白别填满，憋住才显镜头19的爆。
