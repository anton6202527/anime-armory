# 特效定妆 prompt — 本宫才是这皇宫最大的妖（共享层 · ⚙️玄幻题材）

> 特效（妖纹/光效）定**颜色 / 形状 / 路径 / 强度**，和角色脸同等对待——会漂就锁。
> 含特效的镜头一律**多图参考派生**（角色定妆 + 特效定妆 + 上一关键帧）防漂。
> 图AI=Codex，视频AI=即梦 → 末尾拼即梦锚定句。
