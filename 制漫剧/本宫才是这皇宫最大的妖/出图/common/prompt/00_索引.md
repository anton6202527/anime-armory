## 工作流约定（新角色/场景/道具/特效如何加入共享库）

1. 在本索引对应表追加 ⬜ 行（ID/名称/首现/复用范围）。
2. 在 `角色|场景|道具|特效定妆.md` 追加完整 prompt 块（格式见 `references/prompt_format.md §1.2`，**必含底部「生成后自检」栏**）。
3. 生成 PNG → 逐张过自检 → 过则 PNG 落 `出图/common/`，本表状态改 ✅ 并填路径。
4. 后续集自动复用，不再重做。

> **硬闸门**：本集分镜 PNG 生成前，本集引用的所有共享定妆必须 ✅（= 过自检）。缺图或自检未过 → 禁止生成本集镜头图。
