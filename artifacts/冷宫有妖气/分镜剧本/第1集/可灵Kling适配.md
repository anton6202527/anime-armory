# 冷宫有妖气_第1集_可灵Kling适配（跨平台落地样例）

> **演示：平台无关的 `故事板.md` → 可灵Kling 落地。**
> **不变（直接复用）**：`分镜剧本.md` / 角色卡 / 场景卡 / `voiceover.txt` / `字幕_中文.srt` / `字幕_英文.srt` / `bgm.txt`。
> **只换"视频生成"适配层**：即梦=写一段「人物运动+镜头运动+动态细节」动态 prompt；可灵=**首帧图 + 尾帧图 + 运动笔刷 + 运镜**（图生视频）。
> **角色一致性**：每个 Clip 的「首帧图」先用**角色定妆照做参考**（在即梦/可灵图生图出好关键帧）→ 作为可灵图生视频的首帧 → 锁脸锁妆造，与即梦同一套锚点。

## 本项目·可灵Kling 档案要点（摘自 references/platforms.md）
- 画幅 9:16；单 Clip 5~10s（可灵支持较长，可把即梦的两个短 Clip 合并）
- 一致性：**首帧图**（复用角色卡定妆锚定）→ 图生视频；剧烈动作再加**尾帧图**定终点
- 运动用**运动笔刷**框定主体轨迹；**镜头运镜**（推/拉/摇/移/环绕）单独写
- 负面词写进 prompt；画风词依 `global_style.md`
- 锚点简写：沈念常=「乌黑长发披散·凤眼薄唇·肤白·左腕淡疤·月白粗布旧宫装」；沈念觉醒=「双臂暗金图腾妖纹·金色竖瞳」；小禾=「双环髻·洗白青宫装·泪痕」；柳娘子=「深蓝端庄宫装·温和假笑眼神阴冷」；场景=「破败冷宫寝殿·暗褐木梁挂蛛网·雕花木榻·残烛·青灰冷调+暗金烛光」；画风=「国风写实漫剧·电影级光影·9:16」

## Clip 1（8s）觉醒环顾 — 即梦镜头1+2
**首帧图**：特写，沈念（沈念常锚）紧闭双眼眉头微蹙、即将惊醒，背景破败冷宫寝殿木梁蛛网虚焦、残烛微光，国风写实漫剧电影级光影9:16
**尾帧图**：中近景，沈念已半坐起于雕花木榻、睁眼茫然环顾四周，手抓粗麻被，烛光在斑驳墙面投影
**运动笔刷**：① 眼睑→睁开；② 上半身→由卧到坐、头部缓缓转动环视
**运镜**：固定微推 → 缓慢后拉
**负面**：低幼Q版、画风漂移、多余手指、现代物件

## Clip 2（6s）铜镜认脸 — 即梦镜头3
**首帧图**：近景，沈念扑到角落旧铜镜前双手扶镜，镜中映出陌生绝美的脸（凤眼薄唇肤白），烛光在铜镜面流动
**尾帧图**：特写，沈念抬手抚上自己的脸、指尖停在唇边，瞥见左手腕旧疤，神情惊恐颤抖
**运动笔刷**：手→抬起抚脸至唇边
**运镜**：近景缓慢推向特写

## Clip 3（6s）身世闪回 — 即梦镜头4（建议拆 2 个 3s 短 Clip 更稳）
**3a 首帧/尾帧**：少女受封贵妃华服回忆画面（冷色颗粒噪点）→ 皇后宫中搜出巫蛊人偶；**运镜**：硬切+轻微缩放
**3b 首帧/尾帧**：皇帝拂袖震怒 → 冷宫大门「砰」重重合拢；**运镜**：快速推近门缝→黑场
**说明**：可灵对"多画面蒙太奇"不擅长，闪回宜拆成数个独立短镜，靠剪辑串联；每镜仍用首帧图引导。

## Clip 4（8s）报信+柳娘子登场 — 即梦镜头5+6（建议拆 2 个 Clip）
**4a（5s）**首帧：房门「砰」撞开、小禾（小禾锚）哭着冲入；尾帧：小禾扑到沈念身前死死抓住她手臂、浑身发抖。**运动笔刷**：小禾→由门口冲到近前扑抓。**运镜**：中景固定
**4b（5s）**首帧：门帘被掀开一角；尾帧：柳娘子（柳娘子锚）假笑步入、身后两太监端托盘（白绫/匕首/毒酒）。**运动笔刷**：柳娘子+太监→自门帘缓步入画。**运镜**：门帘处缓慢横移
**说明**：一个 Clip 内"两组人物分别动作"对图生视频偏难，拆成两镜各自首尾帧更稳。

## Clip 5（7s）赐死逼酒 — 即梦镜头7+8
**首帧图**：低角度仰拍，柳娘子俯视沈念、温和假笑居高临下、手持酒壶
**尾帧图**：特写，酒壶嘴递近沈念唇边、鸩酒晃动泛苦杏仁味，沈念咬紧牙关眼神疾转；旁侧太监捂住小禾的嘴
**运动笔刷**：① 酒壶→递向沈念唇边；② 沈念→头部微侧
**运镜**：低角度仰拍 → 推近特写

## Clip 6（7s）血脉觉醒 — 即梦镜头9+10（爽点）
**首帧图**：特写，沈念绝望神情，胸口位置一点暗金热流微光初现
**尾帧图**：近景，暗金图腾妖纹自心口蔓延爬满双臂指尖、双眼化为金色野兽竖瞳、发丝被妖气掀起；对面柳娘子假笑骤然凝固
**运动笔刷**：① 暗金妖纹→自胸口向双臂指尖蔓延；② 瞳孔→收缩变金色竖瞳；③ 发丝→被气劲掀起
**运镜**：急推 → 环绕半圈
**说明**：妖纹蔓延+瞳变是"同主体渐变"，正是可灵首尾帧+运动笔刷的强项。

## Clip 7（6s）一拳反杀 — 即梦镜头11（核心爽点·剧烈动作）
**首帧图**：沈念觉醒态（沈念觉醒锚）出拳前一瞬、身影带残影逼近柳娘子
**中间帧（可选）**：拳头贯穿柳娘子胸口的瞬间、黑烟自体内迸出
**尾帧图**：柳娘子身体如纸片碎裂、化作漫天黑色灰烬飘散，地上残留污血与空荡深蓝宫装；沈念立定收拳
**运动笔刷**：① 拳→直线贯穿柳娘子胸口；② 柳娘子身体→碎裂为黑烟灰烬向外飘散
**运镜**：动态特写 + 轻微甩镜跟拳；0.4s 慢动作强调贯穿
**说明**：剧烈打击+碎裂，首尾帧差异大；建议加中间帧或降低单镜时长（5s）、必要时用"运动幅度"参数调高。

## Clip 8（7s）收尾钩子 — 即梦镜头12
**首帧图**：中景，两太监瘫倒连滚带爬逃出画面、小禾瞪大眼僵立，沈念垂眸看着暗金纹路缓缓消退的手
**尾帧图**：近景，金瞳渐隐恢复常态、眼神转冷冽、缓缓握拳，余烬在烛光中飘落
**运动笔刷**：① 手→暗金纹消退；② 手指→缓缓握拳
**运镜**：中景固定 → 近景缓推

## 即梦 vs 可灵 同一 Clip 写法对照（以 Clip 6 为例）
| | 即梦AI（默认） | 可灵 Kling |
|---|---|---|
| 输入 | 一张关键帧（或文生）+ **一段动态 prompt** | **首帧图 + 尾帧图** + 运动笔刷 + 运镜 |
| 运动表达 | 文字描述"妖纹蔓延、瞳孔变竖瞳、发丝掀起" | 首帧=觉醒前 / 尾帧=觉醒后，**运动笔刷**框定妖纹蔓延轨迹 |
| 一致性 | 角色参考图（定妆照）→ 图生图 | 首帧图（同一定妆照出的关键帧）→ 图生视频 |
| 时长 | 5~8s | 5~10s（可合并短镜） |
| 海外/英文 | — | 英文 prompt 直接套 `素材清单.md` 的英文锚定句（亦可转投 Veo） |

> **结论**：跨平台只动这一层——把"一段动态 prompt"改写成"首帧/尾帧关键图 + 运动笔刷/运镜"。`分镜剧本.md`（叙事节拍）、角色/场景卡（一致性锚点）、`字幕_中/英.srt`（投放）全部零改动。Veo/海外则改用英文锚定 + 英文镜头术语，同理只换适配层。
