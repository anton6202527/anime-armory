## ① LOC_01 冷宫寝殿 ⏳

**首现**：第1集 / **复用**：第1-5集主要场所（沈念冷宫蛰伏期）
**目标存档**：`出图/定妆_冷宫寝殿.png`
**参考图来源**：无（**强制清空所有参考图**）

### 即梦设置
- 模型：图片3.0
- 比例：9:16
- 参考图：**完全清空**（最关键的一步）
- 数量：4

### 正向 prompt（中文）

```
破旧阴暗的古代冷宫寝殿全景空镜，无任何人物，暗褐色木质横梁布满蛛网交错垂落，雕花木榻上散落霉斑粗麻被褥，墙面斑驳脱落露出深褐土砖、布满霉斑与水渍痕，靠角落一面老旧布满灰尘的圆形铜镜，案台上一截燃烧过半的残烛烛芯偏黑、烛火微弱摇曳投出诡异晃动阴影，地面薄尘，空气中似有飘浮微尘，空旷死寂压抑，宽阔环境构图远景，国风写实漫剧，电影级光影强烈明暗对比，暗黑宫廷氛围，青灰冷调加暗金烛光点缀，高细节，竖版9:16
```

### 正向 prompt（英文备用）

```
empty wide-shot of a dilapidated dim ancient Chinese cold-palace bedchamber, NO PEOPLE, dark-brown wooden beams heavily cobwebbed and dangling, carved wooden couch with mildewed coarse bedding, mottled walls peeling to reveal dark earthen bricks with mildew and water stains, an old dusty round bronze mirror in the corner, half-burnt candle on an old table with blackened wick faint flickering shadows, dust on floor, suspended dust motes in air, empty deathly silent oppressive, wide environment shot, cinematic Chinese ancient-fantasy webcomic, dramatic chiaroscuro, dark palace atmosphere, cool grey with dark-gold candlelight accents, ultra detailed, vertical 9:16
```

### 负向 prompt（必须严防"画进人 + 画成新房"）

```
人物、人脸、女子、男子、宫女、太监、贵妃、剪影、人影、明亮、华丽、整洁、新房、富丽堂皇、彩色、暖色调、阳光、白天、户外、家具新颖、低幼Q版、卡通、画风漂移、多余文字水印、现代物件、霓虹、廉价滤镜
```

### 7 条检查清单
1. ✅ **真的没人**（即梦经常偷偷画进剪影）
2. ✅ 蛛网、霉斑、破败感到位
3. ✅ 唯一光源是烛火
4. ✅ 雕花木榻作为画面主体或前景
5. ✅ 角落有铜镜（镜头3 要用）
6. ✅ 案台 + 残烛（氛围空镜要用）
7. ✅ 整体青灰冷调 + 烛光暗金，不是黄褐暖色

### 后续复用说明
分镜出图时**可以**把这张作为"风格参考"（注意不是"角色参考"）传上去，强度调低（0.3-0.5）即可。

## 跨 AI 用法

场景图作为"空镜"，理论上不需要"东方面孔"那一条锚定。跨 AI 时只保留"国风写实漫剧 + 暗黑宫廷氛围 + 9:16"部分即可：

```
中文：国风写实漫剧风格，电影级光影，暗黑宫廷氛围，竖版9:16
EN: cinematic Chinese ancient-fantasy webcomic aesthetic, dramatic chiaroscuro, dark palace atmosphere, vertical 9:16
```
