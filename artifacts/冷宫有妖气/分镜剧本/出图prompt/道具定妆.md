## ① 赐死三件套 ⬜

**首现**：第1集 镜头6/8B / **复用**：仅第1集
**目标存档**：`出图/定妆_赐死三件套.png`
**参考图来源**：无（**强制清空**）

### 即梦设置
- 模型：图片3.0
- 比例：9:16
- 参考图：**完全清空**
- 数量：4

```

## ② 万妖血脉系统光幕（HUD/系统面板）⬜

**首现**：第1集 镜头9（觉醒系统音）/ **复用**：全篇升级/查面板/收物时
**目标存档**：`出图/定妆_系统光幕.png`
**参考图来源**：无

### 即梦设置
- 模型：图片3.0
- 比例：9:16
- 参考图：**清空**
- 数量：4

### 正向 prompt（中文）

来源：`global_style.md` 第27-28行（已经过精修的世界观锚定，全篇统一）

```

```
a translucent jade-green floating UI panel hovering before her eyes (medium close-up), ancient seal-script vertical Chinese text on the panel, dark-gold rune borders with faint flowing glow, panel content like "Demon Bloodline forced activation" / "Killed minor demon (Madam Liu), +10 XP", aesthetic blend of arcane HUD and Chinese-fantasy plaque (NOT modern game UI, NOT pure ancient tablet), panel semi-transparent showing blurred character silhouette and background, blurred dilapidated cold palace bedchamber with candlelight, cinematic Chinese ancient-fantasy webcomic, dark palace, cool tone with dark-gold demonic aura, ultra detailed, vertical 9:16

### 负向 prompt

```

### 6 条检查清单
1. ✅ 半透明青绿色（青绿+暗金，不是纯绿）
2. ✅ 古朴**篆字** / 竖排排列（不是现代字体）
3. ✅ 边缘暗金描边（妖术 HUD 感）
4. ✅ 系统提示内容写中文（不是英文 placeholder）
5. ✅ 背景虚化，光幕是前景
6. ✅ "妖术 HUD"而非游戏 UI

### 复用说明
这是**全篇反复出现**的视觉元素（每次升级、每次系统播报）。出一张稳定版本后，所有"系统消息"镜头都用这张作参考图，强度 0.5+，文字内容在 prompt 里替换即可。

## 跨 AI 用法

道具图作为静物，"东方面孔"那一条锚定不需要。跨 AI 时只保留"国风写实漫剧 + 9:16"部分：

```
中文：国风写实漫剧静物，电影级光影，暗黑宫廷氛围，竖版9:16
EN: cinematic Chinese ancient-fantasy still-life, dark palace atmosphere, vertical 9:16
```

> 系统光幕这种"妖术 HUD"特殊视觉，跨 AI 时建议**保留**完整原 prompt，因为它的"国风+科技+暗金妖气"组合很难自动锚定。
