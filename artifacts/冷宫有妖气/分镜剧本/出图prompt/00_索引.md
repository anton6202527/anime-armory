# 出图 prompt 索引（全篇共享）

> 本文件夹放**全篇共享**的定妆 prompt（角色 + 场景 + 道具）。
> 每集的"分镜出图 prompt"放在该集自己的 `第N集/出图prompt/分镜出图.md`。
> **新角色/场景/道具首次出现** → 在本索引追加 ⬜ 行 → 跑出图 → PNG 落到 `分镜剧本/出图/` → 改状态 ✅。
> **后续集复用** → 直接读 `分镜剧本/出图/定妆_xxx.png`，无需重做。

## 工作流约定

```
1. 新角色/场景/道具在 第N集 出料时被发现
   ↓
2. 在本索引 追加 ⬜ 行（含首现/复用范围）
   ↓
3. 在 角色定妆.md / 场景定妆.md / 道具定妆.md 追加完整 prompt 块
   ↓
4. 跑即梦（或 CLI 自动）→ 挑图 → PNG 落 `出图/`
   ↓
5. 改本索引状态为 ✅，填 PNG 路径
   ↓
6. 后续集分镜出图时直接引用此 PNG 作参考图
```

进度更新可参考 `_进度.md` 的 `出图prompt` + `出图` 列。
