# 冷宫有妖气_第5集_BGM与音效

整体情绪曲线：压迫开场 → 爆发屠斗（高燃战斗）→ 险象（毒雾）→ 碾压处决（爽点顶点）→ 晋升神圣感 → 清剿肃杀 → 立威威压 → 冷冽余韵（钩子）

BGM 风格：
- 镜头1-2：低频压迫嗡鸣 + 一记妖气爆发重音，蓄势引爆
- 镜头3-6：国风战斗弦乐+重低音鼓点全开，密集打击节奏（每记重拳/尾扫一个 punch）
- 镜头7-8：节奏短暂收紧（毒雾险境）→ 清凉能量音色（抗毒）化险为夷
- 镜头9-11：鼓点推向最高潮，处决瞬间一记最重爆音 + 戛然留白
- 镜头12：转**神圣/史诗**音色（晋升），银白冰霜质感的清越高频 + 磅礴铺底
- 镜头13：低频肃杀脚步节奏（清剿）
- 镜头14：一记威压重音 + 死寂留白（立威）
- 镜头15：收束、转幽冷意味深长的弦乐单音收尾（钩子）

[镜头1] 巨蟒腥风 + 低频压迫

[镜头9] 石片破空尖啸 + 噗命中
[镜头10] 蛇妖惨叫 + 竹林压塌 + 颚骨裂响
