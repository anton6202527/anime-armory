## 制漫剧 / 制MV（生视频·生图）
- 制作模式: 先出视频后配音   # 用户 2026-06-07 指定的新默认(后配音)。仅作"不选时的推荐"——新作品首跑拆集仍必给菜单选一次(影响全程的出片顺序)。当前推荐：先按估算/现有时间轴把画面推起来，合成前补真音并拟合(适合配音素材/音色/凭证未就绪、或快节奏量产 demo)。代价仍在：时长靠估算锁死→后期补真配音可能对不上→需回头重定时/局部重出视频，正式成片/付费投放前务必换真实音色重跑 voice→finalize→阶段2。可选 配音先行(真实配音时长驱动镜头·音画最准·返工少)
- 单集时长: 前长后短         # 区间·验收窗口：第1集 120–180s / 其余集 60–120s。最终时长由爽点/钩子重切落进区间(字数仅切粗胚≈区间×9)。可选 均衡60–120s / 快节奏45–75s / 长集120–180s / 自定义
- 生视频AI: 即梦
- 生图AI: Codex only       # 统一只用 Codex 内置生图/官方 OpenAI 路由；禁止 Dreamina/即梦生成图片（即使旧项目写“即梦/同视频AI”也改回 Codex only）
- 出视频规格: 预算一般       # 调生视频AI前每次告知当前档。三档各预设 分辨率·帧率·每Clip跑几条挑稳·质量档：充足=1080p/30fps/关键镜2-3条·普通镜2条/高质量档；一般=720p/24-30fps/关键镜2条·普通镜1条/标准档；不够=720p/24fps/全1条/省积分档
- 视频分辨率: 720p          # 通常由 出视频规格 档预设；首次问一次→记入项目 _设置.md→之后沉默沿用，可单独覆盖
- 漫剧画幅: 9:16            # 竖屏短剧
- MV合成画幅: 16:9          # 横屏 MV
- 重抽预算策略: 预算充足      # 出图自检后抽到三项全过为止，主要人物脸/妆造零漂移容忍。可选 预算一般(保主要人物·配角差不多) / 预算不足(全程差不多·最省credit)
- 生成粒度: 逐个            # 仅作默认建议——每次出图/出视频前 skill 都会弹四档菜单让你再选一次(花钱/token敏感点)。可选 逐个/小批(默认5)/按场景分批/整集。任何档都先过共享定妆硬闸门
- 生成优先序: 关键镜优先     # 🔑爽点/反转/钩子/封面候选先出，先定基调先锁主角脸。可选 分镜顺序(Clip1→N) / 先易后难。共享定妆库永远最先
- 脚本批次: 逐集            # 物料脚本量产时每1集停下报边界决策+物料清单再继续(文本不烧钱,问一次后沉默沿用)。可选 小批(默认5集) / 整批。第1集永远单独逐集
