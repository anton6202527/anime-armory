## 制漫剧 / 制MV（生视频·生图）
- 制作模式: 配音先行         # 推荐：真实配音时长驱动镜头(音画准·返工少)。可选 先出视频后配音(快速demo·不推荐：时长靠估算锁死→后期补配音对不上→音画不同步/返工，仅纯视觉demo用)
- 单集时长: 前长后短         # 区间·验收窗口：第1集 120–180s / 其余集 60–120s。最终时长由爽点/钩子重切落进区间(字数仅切粗胚≈区间×9)。可选 均衡60–120s / 快节奏45–75s / 长集120–180s / 自定义
- 生视频AI: 即梦
- 生图AI: Codex only       # 只用 Codex 内置生图/官方路由；禁止 Dreamina/即梦 CLI 兜底
- 视频分辨率: 720p          # 首次问一次→记入项目 _设置.md→之后沉默沿用
- 漫剧画幅: 9:16            # 竖屏短剧
- MV合成画幅: 16:9          # 横屏 MV
- 重抽预算策略: 预算充足      # 出图自检后抽到三项全过为止，主要人物脸/妆造零漂移容忍。可选 预算一般(保主要人物·配角差不多) / 预算不足(全程差不多·最省credit)
- 生成粒度: 逐个            # 仅作默认建议——每次出图/出视频前 skill 都会弹四档菜单让你再选一次(花钱/token敏感点)。可选 逐个/小批(默认5)/按场景分批/整集。任何档都先过共享定妆硬闸门
- 生成优先序: 关键镜优先     # 🔑爽点/反转/钩子/封面候选先出，先定基调先锁主角脸。可选 分镜顺序(Clip1→N) / 先易后难。共享定妆库永远最先
- 脚本批次: 逐集            # 物料脚本量产时每1集停下报边界决策+物料清单再继续(文本不烧钱,问一次后沉默沿用)。可选 小批(默认5集) / 整批。第1集永远单独逐集
