# 出图 prompt 两层架构 — 标准格式

把要出的图展开成**开箱即用**的 prompt 块——出图阶段直接复制粘贴到即梦/Gemini 等图 AI（或喂给 CLI）。**分两层**：复用资产（定妆 = 角色/场景/反复入镜道具）放共享，单次资产（分镜）放本集。

---

## 1. 共享层 — `出图/common/`（全篇一份，扁平：PNG + prompt 同目录）

```
出图/common/
├── 00_索引.md               全篇定妆清单 + 状态 + 首现/复用范围 + PNG 路径
├── 角色定妆.md              所有角色（含形态变体）实战 prompt
├── 场景定妆.md              所有场景实战 prompt
├── 道具定妆.md              关键道具实战 prompt
├── 定妆_<角色>.png          定妆 PNG（与 prompt 同目录，扁平）
├── 定妆_<角色>_<形态>.png
├── 定妆_<场景>.png
└── 定妆_<道具>.png
```

### 1.1 `00_索引.md` 必含字段

- 三张表（角色 / 场景 / 道具），每条含 **ID**、名称、形态、首现集、复用范围、状态（✅/⏳/⬜）、PNG 路径
- **跨 AI 锚定句速查**（三大视频 AI 的中英锚定句）
- **全集统一负面词**
- **即梦统一参数**（模型 / 比例 / 精细度 / 数量 / 参考强度档位）
- **工作流约定**（新角色/场景/道具如何加入共享库）

### 1.2 `角色|场景|道具定妆.md` — 单张 prompt 块标准格式

```markdown
## ① <ID> <名称>·<形态>（✅/⏳/⬜）

**首现**：第N集 / **复用**：第N-M集
**目标存档**：`出图/common/定妆_xxx.png`
**参考图来源**：（已完成共享图 / 无需参考图）

### 即梦设置
- 模型 / 比例 / 参考图 / 参考强度 / 数量

### 正向 prompt（中文）
\`\`\`
（含外貌 + 妆造 + 气质 + 画风词；跨 AI 时末尾追加视频 AI 锚定句）
\`\`\`

### 正向 prompt（英文备用）
\`\`\`
...
\`\`\`

### 负向 prompt
\`\`\`
（统一负面词 + 本项目专属堵漏词，如 "铠甲、护臂" 防止觉醒态被画成装备）
\`\`\`

### N 条检查清单
1. ✅ 关键视觉特征 1
2. ✅ 关键视觉特征 2
3. ...

### 已知瑕疵（接受）/ 设计选择 / 备注
（如有：用于记录"原设定 vs 实际产出"的有意偏差，及如何在分镜阶段圆）
```

## 2. 本集层 — `出图/第N集/`（每集一份，扁平：PNG + prompt 同目录）

```
出图/第N集/
├── 00_总览.md               本集图清单 + 引用共享库 + 本集分镜进度
├── 01_分镜出图.md           本集 N 张分镜实战 prompt（一镜一图，复杂镜可拆 NA/NB）
└── 镜头N_<描述>.png         本集分镜 PNG（与 prompt 同目录，扁平）
```

### 2.1 `00_总览.md` 必含字段

- **本集图数统计**（共享复用 vs 本集分镜，分别列）
- **进度**（已完成 / 总数）
- **已完成图归档**（共享库链接 + 本集出图链接）
- **出图依赖图**（哪些分镜需要哪些定妆作参考）
- **镜头 → 必传参考图 速查表**（路径指向共享库 PNG）
- **不重复**写定妆 prompt（指向共享 `00_索引.md` 链接即可）

### 2.2 `01_分镜出图.md` — 单镜头格式

```markdown
## 镜头 N（节拍标签：如"前3秒冲突"/"反转"）

**剧本描述**：（复制自 分镜剧本.md 画面视觉描述）

**参考图**：
- `出图/common/定妆_<角色>.png`（强度 0.8）
- `出图/common/定妆_<场景>.png`（强度 0.5）

### 正向 prompt（中文）
\`\`\`
（同一少女[参考图人物] + 本镜动作 + 表情 + 景别构图 + 电影级光影 + 画风词 [+ 视频AI 锚定句]）
\`\`\`

### 正向 prompt（英文备用）
\`\`\`
...
\`\`\`

### 负向 prompt
\`\`\`
（统一负面词 + 本镜专属堵漏词）
\`\`\`

### 检查清单
1. ✅ 脸型与定妆一致
2. ✅ 服装配色一致
3. ✅ 景别符合分镜要求
4. ✅ 表情符合本镜情绪

### 备注
（特殊处理、降级方案等；如"本镜运动复杂可改为静帧 + 后期添加动态模糊"）
```

## 3. 工作流（新角色/场景/道具首次出现时）

```
第N集 出料阶段发现新角色"X"（n2d-script 阶段已建 characters/X.md 含 ① 定妆 prompt）
  ↓ n2d-image 阶段：
① 出图/common/00_索引.md 追加 ⬜ 行（ID/名称/首现/复用）
  ↓
② 出图/common/角色定妆.md 追加完整 prompt 块（本文 §1.2 格式）
   （来源 = characters/X.md 的 ① 定妆 prompt + Stage 2 的实战包装：即梦设置 + 检查清单）
  ↓
③ 跑即梦 CLI（或手动指导）→ 挑图 → PNG 落到 出图/common/
  ↓
④ 索引状态改 ✅，填 PNG 路径
  ↓
⑤ 出图/第N集/00_总览.md 引用此 PNG
  ↓
⑥ 后续集自动复用，不再重做
```

---

## 4. 与 Stage 1 文件的关系

| 文件 | 谁写 | 角色 |
|---|---|---|
| `脚本/第N集/分镜剧本.md` | n2d-script | 提供"画面视觉描述"原文（艺术层）|
| `脚本/第N集/素材清单.md` | n2d-script | 简版图清单（理论） |
| `common/characters/<X>.md` 的 ① Prompt | n2d-script | 角色定妆 prompt 源头 |
| `出图/common/角色\|场景\|道具定妆.md` | **n2d-image** | **实战定妆 prompt**（含 CLI 调用 + 检查清单 + 已知瑕疵） |
| `出图/第N集/01_分镜出图.md` | **n2d-image** | **实战分镜 prompt**（含参考图 + 操作） |
| `出图/第N集/00_总览.md` | **n2d-image** | 本集图清单 + 引用共享 + 进度 |
| `脚本/第N集/故事板.md` | n2d-script | Clip 表，**Stage 3 视频生成**用，不在 image prompt 文件夹中 |

**一句话**：
- `素材清单.md` = **简版图清单**（理论，Stage 1 出）
- `出图/common/` = **全篇定妆库**（按角色/场景维度组织，跨集复用，Stage 2 实战包装）
- `出图/第N集/` = **本集分镜库**（按集 + 镜头维度组织，Stage 2 出）
- 跨 AI 时（图 AI ≠ 视频 AI）每个 image prompt 末尾要拼视频 AI 的"图像风格锚定句"，详见 `platforms.md`。
