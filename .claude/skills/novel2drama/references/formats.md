# 素材标准格式模板

所有每集素材都按下列模板填写。**提示词一律中文为主 + 英文备用**。
> 下文 prompt 示例以**默认 = 视频AI=图AI=即梦** 的写法为准；切换 可灵 / Seedance / Veo 等视频 AI 时，prompt 的语言/画幅/Clip时长/角色一致性机制按 `references/platforms.md` 对应档案调整（核心分镜/卡片不变）。

> **跨 AI 强制规则**：若**图 AI ≠ 视频 AI**（例：Gemini 出图 + 即梦出视频），所有 image prompt 末尾**必须**拼接目标视频 AI 的"图像风格锚定句"（见 `references/platforms.md` 各档案）。本文档示例中默认未拼锚定句（同 AI 场景），跨 AI 时自行追加。

## 1. 角色卡（characters/角色名.md，全篇唯一，首次出现即建卡）

```markdown
# 角色卡 — {姓名}（ID: CHAR_01）

- 姓名 / 年龄 / 性别：
- 身份：
- 性格关键词：（3~5个）
- 固定外貌：发型、发色、脸型、肤色、特征（疤/痣等）
- 固定服装：款式、颜色、材质、配饰
- 固定配色：（主色+辅色，便于一致性）
- 固定表情风格 / 动作习惯：

## 妆造拆解（锁一致性用）
- 发型/发色/发饰：（如：乌黑长发半披，无华饰 / 双环髻）
- 妆容：（如：素颜清淡、唇色淡 / 烟熏冷艳 / 病态苍白）
- 服装：（上下身款式、领口、袖型、材质、新旧、腰带）
- 配饰：（簪、耳饰、玉佩、腰牌等；没有也写"无"）
- 色卡：主色 + 辅色 + 点缀色（便于跨集统一）
- 形态变体：（如沈念 常态/觉醒态；记录差异点，避免漂移）

**① 定妆照 / 角色参考图 Prompt（中文）：** ← 先出这张，作为后续所有镜头与视频首帧的"角色参考/图生图"锚点
角色定妆设定图：{姓名}，{年龄性别}，全身正面站姿（A-pose）+ 半身脸部特写，{发型妆容服装配饰色卡}，干净浅灰纯色背景，柔和均匀打光、无强阴影，五官清晰、服装完整可辨，国风写实漫剧角色设定图，高细节，竖版9:16
（备注：定妆照用"干净背景+均匀光"以便锁脸锁妆造，不用电影级强阴影；进入分镜后再套全局电影光影。）

**① 英文（备用）：**
character design / reference sheet: {name}, full-body front A-pose + bust close-up, {hair makeup outfit accessories palette}, clean light-grey solid background, soft even lighting no harsh shadows, clear face and full outfit, Chinese ancient-fantasy webcomic character sheet, ultra detailed, vertical 9:16

**② 即梦图片 Prompt（中文·常态出镜）：**
（角色锚定描述，含外貌+妆造+气质+画风词，用于实际分镜出图）

**② 英文（备用）：**
（同上英文）
```

> 流程：先用 ① 定妆照锁定角色 → 在即梦里把这张设为「角色参考图 / 图生图」→ 之后每个分镜与视频首帧都基于它生成，保证脸和妆造不漂移。
> 后续所有镜头严格复用角色卡，禁止外貌/发型/服装/年龄漂移，除非剧情明确要求并在卡上记录"变体"。

## 3. 分镜剧本（第N集/分镜剧本.md）— 逐镜头脚本

每集 8~15 个镜头。每镜头：

```markdown
## 镜头 N
**画面视觉描述：** 时间/光线/环境 + 人物动作 + 表情 + 构图/景别/机位/运镜。
**台词 / 音效 / 旁白：** 台词（角色·情绪）：「…」 | 旁白：「…」 | 音效：[…]
```

节奏硬约束：前3秒有冲突，前15秒出现矛盾，中段情绪提升，结尾留钩子，每集至少一次反转。

---

## 4. 故事板 Clip 表（第N集/故事板.md）— **即梦视频生成输入**

把相邻分镜合成 **片段（Clip）**，每片段 1~2 个分镜、时长约 5~8 秒，作为即梦"一次视频生成"的单元。

```markdown
## Clip 表（分镜连贯性校验后）

**片段1（Clip 1）：时长：7秒**
**场景**：{场景名}（夜晚/内）
**分镜1：0-4s**
镜头：{景别}，{距离}，{机位角度}，{运镜}。
{画面动态描述：人物运动 + 镜头运动 + 动态细节，如烛火摇曳/晨雾流动}
**分镜2：4-7s**
镜头：…
{…}

## 物品补充设计清单
（本集出现但未建卡的关键道具，列出并补 prompt；无则写"无遗漏"）
```

> 视频 prompt 必须显式描述**人物运动 + 镜头运动 + 动态细节**。避免超复杂打斗、多人混战、大量人群、高频切换等 AI 难生成动作。

可选同步输出机器可读 `storyboard.json`：
```json
{ "episode": 1, "clips": [
  { "id": "EP01_CLIP01", "duration": 7, "scene": "冷宫寝殿/夜/内",
    "shots": [ { "t": "0-4s", "lens": "全景·推镜", "desc": "...", "video_prompt": "..." } ] }
]}
```

## 5. 素材清单（第N集/素材清单.md）— **即梦图片生成输入**

```markdown
英文 Prompt：

### LOC_01 {场景名}
中文 Prompt：
英文 Prompt：

## 关键道具
### 残烛
中文 Prompt：
英文 Prompt：
```

## 6. 配音文案（第N集/voiceover.txt）

按镜头顺序逐条，标注角色与情绪，便于 AI 配音：
```
[镜头1·沈念·内心独白·茫然] 这不是我的寝室……
[镜头2·旁白·低沉] 林婉儿，户部侍郎嫡女……
```

## 7. BGM 与音效（第N集/bgm.txt）

```
整体情绪：阴森压抑 → 高潮爆发
BGM风格：国风暗黑 / 弦乐渐强 / 低频鼓点
关键音效点：[镜头4 诡异铜铃] [镜头6 重低音心跳] [镜头7 妖气爆裂]
```

## 8. 封面 / 首图（第N集/封面.md）

一张高点击率竖版封面，含本集最大爽点或钩子、主角清晰正脸、强情绪、可叠加标题留白。给中文+英文 prompt。

## 9. 字幕（第N集/字幕_中文.srt + 字幕_英文.srt）— 中英双语

标准 **SRT** 格式，可直接导入剪辑软件 / 上传平台。**两个文件同一套时间码**：`字幕_中文.srt` 国内投放，`字幕_英文.srt` 海外投放（TikTok / ReelShort / YouTube / 北美短剧）。

**时间码来源**：把 `voiceover.txt` 的每条台词/旁白，按 `故事板.md` 各 Clip/分镜的累计时长依次排进时间轴；一镜多句则在该镜窗口内平分。

**英文翻译要点**：口语化、贴短剧语境、长度适配字幕（每行≤≈42字符、≤2行）；保留人名音译统一（沈念 Shen Nian、林婉儿 Lin Wan'er、柳娘子 Madam Liu）；系统提示用游戏化英文（如 "Demon Bloodline activated."）。

```srt
1
00:00:00,000 --> 00:00:03,000
这里……不是我的宿舍。

2
00:00:03,000 --> 00:00:08,000
粗麻、霉味、蛛网……这是哪儿？
```
英文版同一时间码：
```srt
1
00:00:00,000 --> 00:00:03,000
This... isn't my dorm room.

2
00:00:03,000 --> 00:00:08,000
Coarse cloth, mildew, cobwebs... where am I?
```

## 10. 出图prompt — **两层架构**（共享 + 本集）

把要出的图展开成**开箱即用**的 prompt 块——出图阶段直接复制粘贴到即梦/Gemini 等图 AI。
**分两层**，复用资产（定妆 = 角色/场景/反复入镜道具）放共享，单次资产（分镜）放本集。

### 10.1 共享层 — `分镜剧本/出图prompt/`（全篇一份）

```
分镜剧本/
├── 出图/                    全篇共享【定妆 PNG 库】
│   ├── 定妆_<角色>.png
│   ├── 定妆_<角色>_<形态>.png
│   ├── 定妆_<场景>.png
│   └── 定妆_<道具>.png
└── 出图prompt/              全篇共享【定妆 prompt 实战库】
    ├── 00_索引.md           全篇定妆清单 + 状态 + 首现/复用范围 + PNG 路径
    ├── 角色定妆.md          所有角色（含形态变体）实战 prompt
    ├── 场景定妆.md          所有场景实战 prompt
    └── 道具定妆.md          关键道具实战 prompt
```

**00_索引.md 必含字段**：
- 三张表（角色 / 场景 / 道具），每条含 ID、名称、形态、首现集、复用范围、状态（✅/⏳/⬜）、PNG 路径
- 跨 AI 锚定句速查（三大视频 AI 的中英锚定句）
- 全集统一负面词
- 即梦统一参数（模型 / 比例 / 精细度 / 数量 / 参考强度档位）
- 工作流约定（新角色/场景/道具如何加入共享库）

**角色/场景/道具定妆.md 单张 prompt 块标准格式**：

```markdown
## ① <ID> <名称>·<形态>（✅/⏳/⬜）

**首现**：第N集 / **复用**：第N-M集
**目标存档**：`出图/定妆_xxx.png`
**参考图来源**：（已完成共享图 / 无需参考图）

### 即梦设置
- 模型 / 比例 / 参考图 / 参考强度 / 数量

### 正向 prompt（中文）
```...```

### 正向 prompt（英文备用）
```...```

### 负向 prompt
```...```

### N 条检查清单
1. ✅ 关键视觉特征 1
2. ...

### 已知瑕疵（接受）/ 设计选择 / 备注
（如有）
```

### 10.2 本集层 — `分镜剧本/第N集/出图prompt/`（每集一份）

```
第N集/
├── 出图/                    本集**分镜出图** PNG（不含定妆，定妆在共享库）
│   └── 镜头N_<描述>.png
└── 出图prompt/
    ├── 00_总览_出图清单.md  本集图清单 + 引用共享库 + 本集分镜进度
    └── 01_分镜出图.md       本集 N 张分镜实战 prompt（一镜一图，复杂镜拆 NA/NB）
```

**00_总览.md 必含字段**：
- 本集图数统计（共享复用 vs 本集分镜）
- 进度（已完成 / 总数）
- 已完成图归档（共享库链接 + 本集出图链接）
- 出图依赖图
- 镜头 → 必传参考图 速查表（路径指向共享库 PNG）
- **不重复**写定妆 prompt（指向共享 `00_索引.md`）

**01_分镜出图.md 单镜头格式**：

```markdown
## 镜头 N（节拍标签）

**剧本描述**：（复制自 分镜剧本.md 画面视觉描述）

**参考图**：（必须传哪些共享定妆 + 强度）

### 正向 prompt（中文）
```...```

### 正向 prompt（英文备用）
```...```

### 负向 prompt
```...```

### 检查清单
1. ✅ ...

### 备注
（特殊处理、降级方案等）
```

### 10.3 工作流（新角色/场景/道具首次出现时）

```
第N集 出料阶段发现新角色"X"
  ↓
① 共享 出图prompt/00_索引.md 追加 ⬜ 行（ID/名称/首现/复用）
  ↓
② 共享 出图prompt/角色定妆.md 追加完整 prompt 块（10.1 格式）
  ↓
③ 跑即梦（或 CLI 自动）→ 挑图 → PNG 落到共享 出图/
  ↓
④ 索引状态改 ✅，填 PNG 路径
  ↓
⑤ 第N集 出图prompt/00_总览.md 引用此 PNG
  ↓
⑥ 后续集自动复用，不再重做
```

### 10.4 与其他素材文件的关系

| 文件 | 角色 |
|---|---|
| `分镜剧本.md` | 提供"画面视觉描述"原文（艺术层）|
| `素材清单.md` | 简版图清单（按角色/场景/道具列简版 prompt）|
| 共享 `出图prompt/角色\|场景\|道具定妆.md` | **实战定妆 prompt**（含即梦操作 SOP / 检查清单 / 已知瑕疵）|
| 本集 `出图prompt/01_分镜出图.md` | **实战分镜 prompt**（含参考图 + 操作）|
| 本集 `出图prompt/00_总览.md` | 本集图清单 + 引用共享 + 进度|
| `故事板.md` | Clip 表，**视频生成**用，不在 image prompt 文件夹中 |

> 一句话：
> - `素材清单.md` = **简版图清单**（理论）
> - 共享 `出图prompt/` = **全篇定妆库**（按角色/场景维度组织，跨集复用）
> - 本集 `出图prompt/` = **本集分镜库**（按集 + 镜头维度组织）
> - 跨 AI 时（图 AI ≠ 视频 AI）每个 image prompt 末尾要拼视频 AI 的"图像风格锚定句"，详见 `references/platforms.md`。
