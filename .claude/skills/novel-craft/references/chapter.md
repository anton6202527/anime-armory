# 单章写作守则

适用于任何 novel-* 写章场景；子代理 prompt 模板也在此。

## 人称选择

- **第一人称（我）** —— 沉浸感最强，对口吻一致性要求极高；只在视角角色孤独/内省时用。
- **第三人称限定** —— 默认选择。视角始终在该角色身上，只写他能感知到的。
- **第三人称全知** —— 通常**不推荐**用于视角作品。

## 视角纪律（第三人称限定下）

只写视角角色能感知的：
- ✅ 看到 / 听到 / 闻到 / 想到 / 回忆。
- ❌ 别人的内心活动（除非可由表情/动作合理推测，且写成推测）。
- ❌ 视角角色不在场的事件。
- ❌ 全知评注（"殊不知此事日后…"）。

## 不要 OOC

- **语速 / 句长**：保持原设。
- **关心什么**：原设关心 X，新章别突然关心 Y。
- **能力边界**：原设做不到的事，新章别突然会。

## 不大段复刻

派生作品场景（spinoff / expand）：
- 超过 ~30 字的原作原句 → 必删，重写。
- 多段稍作改动后照搬 → 也必删（洗稿降质）。
- 5-10 字内的角色口头禅 / 标志短句可保留。

## 节奏

- 每章一个情节小目标，开头点明、中段冲突、结尾达成或翻车。
- 单章字数随项目定：漫剧友好 4000-6000 字 / 网文长篇 5000-8000 / 短篇 6000-10000。
- 纯对话 ≤ 1/3。
- 长景物 / 心理 / 回忆 ≤ 200 字一气；要长就拆。
- 章末必有钩子。

## 衔接

- 子代理拿到上一章末尾 ~500 字。
- 本章开头**承接**而非复述。
- 时间跳切可以（"三日后……"）。

## 子代理 prompt 模板

```
你是<原作名>的派生作者，正在写从 <视角角色> 视角的 <模式> 作品《<书名>》。

## 全局设定
[贴：角色卡.md 全文，含"口吻习惯/不要做的事"段]

## 世界观
[贴：世界观.md 全文]

## 章纲（全本）
[贴：章纲.md 全文]

## 文风样本（必须对齐）
[贴：章节/第01章.md（Demo 第 1 章）全文]

## 你要写的：第 N 章 《<标题>》
- 本章目标字数：[XXXX-XXXX]
- 主线事件：[章纲里的本章 outline]
- 视角情绪基调：[一句话]
- 涉及的锚点（必须对齐）：
  - 锚点 X：原作第 Y 章 Z 事件 / 在场 W / 已知 K / 未知 L
  - 对齐方式：用事件骨架对齐，不要照抄原作原文
- 上一章末尾衔接：
  [上一章最后 ~500 字]
- 本章末尾要留的钩子：[一句话]

## 写作守则
[贴：本文件全文 + 项目本地的额外口吻补充（如果有）]

## 输出
直接输出本章 markdown：第一行 `# 第 N 章 <标题>`，之后正文段落。不加额外注释或元信息。
```

## 写完自检

- [ ] 字数在目标区间
- [ ] 视角全程限定，没穿帮
- [ ] 涉及锚点对齐事件骨架，未照搬原文
- [ ] 没让视角角色"知道"原作后文才揭示的事
- [ ] 章末有钩子
- [ ] 标题和首行 `# 第 N 章 <标题>` 一致
