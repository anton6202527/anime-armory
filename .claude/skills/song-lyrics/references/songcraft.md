# 作词工艺知识库（song-lyrics 的工艺基元）

> 不强制流程，给"怎么写词"的工艺。song-lyrics 主流程引用本库；也可独立摘录回答工艺问题。

---

## §访谈 — 几个字 → 可写的词

一次问一组、带默认建议（用户选/改即可），别一次性轰炸：

1. **主题 + 情绪**：写什么、什么情绪顶点（思念/热血/释怀/孤独/狂欢…）
2. **曲风 + 平台**：国风 / 流行 / 说唱 / 民谣 / 电子 / 抖音神曲；平台决定时长与 hook 密度
3. **叙事 vs 抒情**：讲一个故事，还是抒一种情绪
4. **视角**：第一人称"我" / 对唱/对话 / 旁观叙述
5. **结构**：默认流行结构（见 §结构），可按曲风调整
6. **韵脚 / 关键意象**：想押的韵母（如 ang/ai/ou）、必用意象/词

> 四样硬项（主题、曲风、结构、副歌 hook 方向）定了就能开写，其余边写边定。

## §结构 — 段落骨架（先搭再填）

| 段落 | 作用 | 写法要点 |
|---|---|---|
| **Intro** | 入场氛围 | 可纯乐器/短句，立调 |
| **Verse 主歌** | 叙事/铺垫 | 信息推进，情绪压着，为副歌蓄力 |
| **Pre-chorus 预副歌** | 蓄力爬升 | 1-2 句把情绪推到副歌门口，常变韵/升 key |
| **Chorus 副歌** | **核心 + hook** | 全曲记忆点，情绪顶点，**反复出现**，最先写 |
| **Bridge 桥段** | 反转/升华 | 换视角/换和声/最高潮，避免审美疲劳 |
| **Outro** | 收尾 | 重复 hook 淡出，或留白 |

常用流行结构：`intro–verse1–pre–chorus–verse2–pre–chorus–bridge–chorus–outro`。
抖音神曲：**前 8 秒必出 hook**（黄金开头），结构更短、副歌前置。

## §副歌 & Hook（先写这个）

- **副歌是全曲的脸**：最先写。定情绪顶点 + 韵脚 + 那句**最抓耳、可重复、易跟唱**的 hook。
- Hook 三要素：**短、顺口、有记忆点**（常含重复、对仗、或一个钩子意象）。
- 副歌反复出现 → 每次可微调词尾制造递进，但 hook 句保持不变（记忆锚）。

## §押韵

- **一段一韵脚**（或规律换韵），别每行乱押。中文按韵母大类押（如 中华十三辙）。
- 句尾落在**开口/响亮韵**更好唱（ang/an/ai/ao/ou 比闭口 i/ü 更易飙）。
- 押韵服务顺口，不为押而凑——**宁可不押也别拗口**。
- 内韵/头韵可加分（句中也呼应），但别堆。

## §字数贴旋律（可唱性核心）

- **每行音节数对齐乐句长度**：旋律一句几个音 → 词大致几个字。给定旋律填词时尤其严格。
- 没有旋律时，**同段各行字数相近**（如主歌每行 7-9 字），副歌可更短促有力。
- **重音/长音落在情绪词上**：旋律的长音/高音处放想强调的字，别落在虚词（的/了/吗）。
- 避免一个音塞太多字（赶）或一个字拖太多音（散）。

## §平台 / 曲风套路（速查）

| 曲风/平台 | 词的特征 |
|---|---|
| 国风 | 意象密、用典、半文半白、画面感；韵脚雅 |
| 流行情歌 | 口语、直给情绪、副歌强 hook、人称"我/你" |
| 说唱 | 密集押韵(多押/连环韵)、flow 节奏、内容叙事/态度 |
| 民谣 | 叙事白描、生活意象、克制、留白 |
| 电子/舞曲 | 短句、重复、hook 至上、词服务律动 |
| 抖音神曲 | 前奏短、副歌前置、洗脑重复 hook、8 秒抓人 |

## §可唱性自查清单（交付前逐条核）

- [ ] 副歌有 1 句最抓耳、可重复的 hook？
- [ ] 一段一韵脚、押得自然不拗口？
- [ ] 每行字数贴乐句（或同段字数相近）？
- [ ] 长音/高音处落在情绪实词、不落虚词？
- [ ] 口语顺口、念一遍不打结？
- [ ] 结构完整（主歌蓄力→副歌爆发→桥段反转）？
- [ ] 平台/曲风套路对上（如抖音前 8 秒出 hook）？
- [ ] 情绪有起伏曲线，不是从头平？

> 一句话：**副歌先行、可唱优先、一段一韵、字贴旋律、hook 抓耳。**
