# 创作蓝图 — 歌《仗剑下山》

> 这首歌的"宪法"。写歌线产出成品歌交给 制MV 做视频。

## 一句话主题 / 情绪
少年仗剑下山，闯荡江湖——意气风发与一路孤独并行，历尽风霜仍是当年那张笑脸。

## 曲风 / 平台 / 基调
- 曲风：国风/古风武侠（竹笛 + 古筝 + 战鼓 + 弦乐，副歌加电吉他推力）
- 目标平台：抖音（前奏短、副歌前置、hook 抓耳）
- 情绪基调：燃 · 少年意气，底色一抹孤独

## 歌曲结构（段落骨架）
- [intro]（竹笛+古筝，山雾起，~8s 短前奏）
- [verse1] 下山·别师
- [pre-chorus] 蓄力·转身
- [chorus] 仗剑下山·hook（情绪顶点）
- [verse2] 初入江湖·人心
- [pre-chorus] 孤独·再上路
- [chorus]
- [bridge] 多年后·回望（最高潮，转调）
- [chorus]（末句拔高）
- [outro] 清唱留白

## 演唱（song-compose 后端）
- 演唱音色：男声·少年感（清亮带一点沙），**自有/授权/合成**，不用未授权真人嗓
- 后端：Suno（云·最快）或 ACE-Step（本地，Mac 可跑）
- **Suno style 提示**：`Chinese guzheng+bamboo flute wuxia epic, youthful male vocal, war drums, soaring chorus with electric guitar, cinematic, key Am, ~86 BPM, 抖音 hook`

## 风格卡
- 文风/咬字：古风白话相间、画面感强、燃中带孤
- 句子节奏：主歌叙事铺垫，副歌短句有力、可跟唱
- 禁忌：太口水、现代俚语、拗口堆典
- hook：「我仗剑下山，闯一闯这人间」
