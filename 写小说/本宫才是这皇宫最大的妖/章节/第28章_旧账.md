# 第二十八章 旧账

那行小字，是原身的手迹。

沈念把玉佩翻到内侧，借着窗下那点天光，一笔一画看了三遍。字是簪花小楷，娟秀，却在最后一捺上微微一抖——写的人那一刻，心是乱的。

她不认得这笔字。可那具身子认得。

指腹一贴上去，原身那点散在血脉里的残念，毫无征兆地翻涌上来。

不是记忆，是碎片。像隔着一池浑水，看水底沉着的几样旧物。

——一座灯火通明的偏殿，案上摊着她写到一半的字。

——一个深蓝宫装、发髻一丝不苟的姑姑，端着茶，笑意温和，"贵妃娘娘的字，越发好了。"

——那姑姑替她研墨，袖口一拂，一只小小的、绣着并蒂莲的香囊，悄无声息落进了她的妆奁。

沈念的指尖顿住。

那姑姑的脸，她见过。

不是在原身的记忆里见过。是在那座废戏台底下的地窖里——

一身银灰长袍，眉目清隽，一双瞳仁是凝住的赤红。

赤睛。

碎片还在翻。沈念没有挣，任那池浑水一层层往上泛。

——巫蛊案发那夜，禁军踏破偏殿，从她的妆奁底下，搜出三枚扎满银针的桐木小人，每一枚背后，都刻着永宁帝的生辰。

——满殿的人指着她，那个研墨的姑姑站在最前头，垂着泪，"奴婢亲眼看着贵妃娘娘，把那香囊收进妆奁的……"

——白绫，冷宫，三年。

水底的旧物，看清了。

沈念缓缓松开玉佩，靠回榻边的冷墙。她闭了闭眼，等那股翻涌的残念退下去，才睁开。眼底没有泪——原身的泪，三年前在偏殿里就流干了。她替原身，把这笔账，算了一遍。

栽赃的桐木小人，不是原身放的。是那个研墨的姑姑，借研墨递香囊那一拂，神不知鬼不觉塞进妆奁的。而那个姑姑——

是赤睛假扮的。

沈念冷笑了一声，笑得没有半分暖意。

她原以为，巫蛊案是皇后构陷一个得宠贵妃的寻常宫斗手笔。下毒、栽赃、坐实、赐死，一套走完，干净利落。如今才知道，那一手栽赃里，藏着一只妖的爪子。

三年前，赤睛就插了手。

那只在地窖里替"主子"分赃的赤睛，那只腰悬银牌、与她同阶的妖——三年前，就披着一张姑姑的人皮，钻进了她入冷宫的局里。

她忽然想通了一桩一直想不通的事。

皇后构陷一个贵妃，要的是除掉碍眼的人。可一个寻常贵妃，犯不上动用一只银牌妖位的妖，亲自下场栽赃。除非——皇后要的，从一开始就不只是除掉林婉儿。

是要把林婉儿，送进冷宫。

送进那座养妖、收租、定期开宴的猎场。

"原来如此。"沈念极轻地说。

她入冷宫，从来不是皇后宫斗输赢的余波。是皇后与那位藏在空主位上的"主子"，合谋下的一步棋。她这具身子，从被搜出桐木小人那一刻起，就是要被送进冷宫、当成一份"供奉"的活物。

巫蛊冤案，是局。打入冷宫，是局。三年苟延残喘等着被吃干净，也是局。

而布这局的两双手——一双是坤宁宫里的皇后王氏，一双是地窖空主位上的妖师。

地窖里那一窖供奉，从冷宫一路递向后宫深处的猜测，到今夜，落了实。皇后那点宫斗手腕底下压着的更深的东西，到今夜，露了形。

养在冷宫的妖主子，与坤宁宫的主子，本就是一双手——不，是两双手，攥着同一根绳。

沈念低头，看着掌心那枚玉佩。

簪花小楷的那行小字，是原身写给那个男人的。写得娟秀，写得心乱。三年了，原身把这枚玉佩贴身藏着，藏过白绫，藏过冷宫，到死都没舍得丢。

她忽然觉得这原身可怜。也只是觉得。

可怜归可怜，账归账。

林婉儿替别人背的黑锅、受的三年活罪、最后那口没喝下去的毒酒——这笔账，皇后欠，妖师也欠。一个借妖栽赃送她入局，一个披人皮亲手把她推下井。

她把玉佩收进袖中。

"这笔账，"她对着空荡荡的冷宫，声音冷得像井底的水，"本宫要连本带利，一文一文，讨回来。"

讨债，得先有讨债的本钱。

她比谁都清楚，眼下她这身银牌妖位，搁进那一窖妖里不过中游；赤睛与她同阶却驯化更深，那位空主位上的主子，更是一道妖气就能压得她膝盖发软的五品妖师。硬碰，是把命白白送上席。

可账，迟早要硬碰着讨。

她得把手里这点东西，磨得更利。

——

那一夜起，沈念连炼了三晚血。

她没贪吃新妖。贪噬反噬的滋味，她在槐树下尝过一回，在地窖口又被境界压制堵过一回，记得清清楚楚。这三晚，她炼的是旧术——把已经入体、却还生疏的两门，往骨血里再淬一淬。

第一门，是潜行往下沉出来的那点血提纯。

她原以为潜行只能收没脚步、淡没身形。这三晚她盘膝枯坐，一遍遍把那股暗金热流逼进血脉最深处，逼着它把每一缕掺在血里的杂气都滤出去——血越纯，潜行越深。到第三晚，她试着借这门提纯过的潜行，在小禾眼皮底下从榻这头挪到那头，那忠心的小丫头守了半宿，竟一回都没瞧见她动过。

潜行成了潜踪。这一门，是为了来日摸进坤宁宫、摸进那座地窖，谁也察觉不出。

第二门，是覆鳞那身鳞甲。

她要的本就是扛得住打。这三晚，她把那层冷青的硬壳，从手背一寸寸往周身漫——漫过小臂，漫过胸腹，漫过后心。鳞甲越铺越广，可代价也越压越沉。

到第三晚下半夜，沈念睁开眼时，皮下那道鳞影，烫得发疼。

不是炼血时的热。是一种隐隐的、贴着骨头烧的疼，像血脉深处有一簇火，一直没熄。她垂眼看自己的手背，鳞甲状的暗纹缓缓游动，纹路比三日前，深了一线。

连炼三晚两术，血脉在过载的边缘绷了三天。她能清楚地觉出，那条隔在人性与妖性之间的界，薄了。

薄到她偶尔会冒出一个念头——若此刻去那窖里，把十三只妖一口口全吃了，会怎样。

念头一冒出来，她就掐住了掌心。

很疼。也正是这点疼，把那念头摁了回去。

她在心里冷冷记下：两术并淬，妖纹三日不退；再炼，怕是要伤己。代价她认。可这界薄了，她不认——主子也得是她。

下不为例。

她松开掐进掌心的手，指甲在皮肉上留下四道月牙似的红痕。窗外天色将明，冷宫破败的院墙外，那座吃人的皇城，灯火一盏盏熄了。

讨债的本钱，磨了三晚。潜踪能摸进去，鳞甲能扛得住——这两样，够她在硬碰那位主子之前，先把这条命，稳稳地攥在自己手里。

至于那笔旧账，连本带利，她记在心上，一文不少。

——

第三晚，五更将尽。

沈念没睡。三晚连炼，人是乏的，妖纹却还烫，睡不沉。她索性起身，借着窗下那点将明未明的天光，对着案头那面斑驳铜镜，慢慢梳头。

乌黑长发一缕缕从指间滑过。镜里那张脸，凤眼薄唇，肤白近透明，左腕一道淡疤——这些日子看惯了，她竟也快当成是自己的了。

木梳齿划过发尾。镜中人也划过发尾。

她梳得很慢，心里盘着那笔账：皇后那一头，眼下动不得，得等出征边关的棋走完；妖师那一头，循味没夺成，线断在地窖那张空主位上……

木梳停了。

镜子里，那双沉静的凤眼，眼底毫无征兆地，泛起一线红。
