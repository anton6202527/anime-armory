# 世界观 — 摘自《仙界闭关小能手》

> 只摘原作已确立的规则，不发明能推翻原作的新规则。基于采样章节（28-30、129、471、末段）建立，未读章节涉及的细节后续按需精读补充。

## 大世界框架

- **凡间** + **修真界** + **灵界**（曾经的飞升目的地）+ **神界**（最终极目的地，王敦和贺平生最终都到了这里）。
- 修真界顶层规则不再允许向灵界自由飞升——化神四层会被强制飞升，所以现在的修真者基本卡在元婴 / 化神。
- 数千年前规则不同：化神期任何层都可飞升灵界，没有强制门槛。

## 修炼境界（由低到高，含补读后的完整阶梯）

修真界 → 灵界 → 仙界 → 神界 的境界阶梯：

1. **炼气期** 1–13 层
2. **筑基期** —— 需吞筑基丹突破；多数人卡在这里
3. **金丹期** —— 渡道障结金丹
4. **元婴期** —— 金丹破而生婴；天道誓言对元婴期及以上才有约束力
5. **化神期** —— 化神四层（中期）会被强制飞升灵界
6. **渡劫期**
7. **金仙 / 大罗金仙 / 太乙金仙** —— 仙界 / 神界过渡区段；王敦中后期境界
8. **道元境** —— 多层（十一、十二…）；王敦达过道元十二
9. **混元境** —— 道元之后的瓶颈，王敦后期突破
10. **神将境 + 飞升神界顶层** —— 终局，得神格后跃迁

## 修炼基础

- **灵根**：决定修炼难度。单灵根最优，双灵根次之，五行灵根（金木水火土俱全）最难，被称为"费钱货"——需要大量资源才能堆出来。
- **灵力 / 灵气**：从外界灵气中炼化为体内灵力，按属性分类（金/木/水/火/土）。
- **灵力品质**：四品色阶——**赤红（下品）→ 黄（中品）→ 金（上品）→ 紫（极品）**。这套色阶贯穿全书，也是花粉、灵植、符箓的品阶约定。
- **道障**：金丹期开始出现，是境界突破的硬门槛。元婴期之后道障更难破。
- **机缘**：除修炼资源外，机缘是突破高境界的必要条件。无机缘即便狂吞丹药也无法突破。

## 太虚门（王敦所属宗门）

- **结构**：外门 + 内门
  - **外门**：初入修真者的容身处。贺平生原是外门子弟（父辈外门弟子）。
  - **内门**：山上诸峰，各自有长老 + 弟子，含**秀竹峰**（贺平生入内门所在峰，玉宁是峰主）等多个主峰。
- **杂役班**：各峰下设杂役，负责挑水 / 伙食 / 跑腿。贺平生从秀竹峰挑水杂役起步。
- **灵药谷**：太虚门下属的一个灵植种植区，**枯木真人**为长老，王敦是这里的执事弟子。
  - 不允许建宫殿房屋（怕走水失火），所有人住山洞
  - 阵法供应火属性灵力，需主修火灵根修士长期注入
  - 主要灵植：烈阳花（二品，火属性，560 株一片）
  - 花粉用途：磨成粉末入兽血制**符墨**，是绘制【烈火符】（二品符箓）的主料

## 灵植 / 丹药 / 符箓 品阶

- **灵植**：一品至九品；一品灵植只能炼一品丹药，依此类推。
- **符箓**：一品至九品（晚期出现五品的【天风符】【厚土符】，是上古遗物级别）。
- **丹药**：分品阶之外还分下品/中品/上品/极品（用色阶对应）。
- **筑基丹**：突破筑基期用；极品筑基丹几乎稳定突破。

## 既定关键人物（王敦的人际网络）

- **贺平生**：原作主角，王敦的师弟，五行灵根，太虚门秀竹峰内门弟子，最终飞升神界。王敦命运的核心系泊者。
- **枯木真人**：太虚门灵药谷长老，王敦的直属上司。脸皮如枯木须发皆白，性格暴躁，赏罚分明。
- **玉宁**：秀竹峰峰主，贺平生的师尊。
- **侯慕贤**：贺平生的大师兄（玉宁门下），出身**侯家**（太虚门旁系修真家族）。
- **田小青**：贺平生的同峰师姐。
- **振羽老祖 / 皇家老祖**：梁国（修真界一个国家）的两位高阶老祖；中后期对贺平生有助益。
- **姜禹**：贺平生的化神期仇人。
- **乔慧珠**：贺平生的妻子，中段失踪。
- **重华**：王敦的道侣（晚期出现，原作未细写）。
- **长生**：王敦和重华的孩子。
- **太真**：重华的姐妹（晚期出现）。
- **中极**：和贺王同代的飞升候选者，十二人中**死于飞升前/中**，未到神界。

## 不要做的事（修正版）

- **不要在外传里发明能推翻原作设定的新规则**（如：不允许设定"其实化神期可以飞升只是大家没找到方法"）。
- **不要把王敦写成单纯陪跑者**——补读后已确认他自己是天才（紫薇徒 + 王敦密码发明者 + 任务系统宿主）。要呈现"装拙的天才"，反差是核心。
- **但也不要让王敦盖过贺平生**——王敦的天才弧是平行而非超越。两人最终肩并肩飞升神界。
- **不要写贺平生不存在的事件**——所有涉及贺平生的事件都要和原作锚点对齐。
- **不要把"王敦密码"具体内容编出来**——原作没解释这个密码是什么内容；续写也只写它"存在 + 有用 + 带来功德"即可，避免编一个推翻原作设定的具体内容。
