# 角色卡 — 王敦

> 基于源采样章节（第 28-30、129、471、末段约 104000-110341 行）建立的画像。未读章节的细节后续在章纲编织 / 子代理 prompt 阶段按需补充。

## 来源
- 原作：《仙界闭关小能手》
- 模式：parallel（并行外传）
- 原作主角：贺平生（与王敦是师兄弟关系，王敦为兄）
- 王敦原作首次出场：第 28 章
- 原作末尾王敦状态：已飞升神界，得神格，达神将境

## 基本信息

- **称谓**：王敦；贺平生称"王师兄"；他自己也常用"我师兄"自况（晚期）。
- **门派 / 身份**：太虚门内门弟子，归灵药谷长老**枯木真人**管辖。具体岗位：灵药谷的执事弟子，负责招募火灵根新人 + 日常管理烈阳花田 + 上报采收。
- **修为**（出场时）：炼气期六层。比同期贺平生（炼气期四层）高一阶，但在整本修真等级体系里位于**中低层**。
- **修为终局**：神将境（神界飞升后，由贺平生赠神格强化所致）。

## 外观

- 身材肥胖（关键体貌——与瘦削的贺平生形成视觉反差）。
- 身穿青色道袍（太虚门内门弟子常服）。
- 笑容时"笑呵呵"，紧张/不满时"脸色顿时一黑"。

## 性格底色

- **务实而非清高**：典型的"中层管理"心态。上面有枯木真人这种暴脾气长老压着，下面要拉着新人完成任务。
- **友善但有戒心**：贺平生第一次问他"紫色能管几日"，他脸色一黑——不是不愿说，是**他自己不知道**，又不愿暴露知识盲区。这是底层小师兄典型的"装懂"。
- **乐于做人情**：贺平生上道，他立刻笑呵呵；主动提"米粮缺少我那里多得是"——会用小恩小惠经营关系。
- **怕事**：枯木真人面前一脸恭敬，避开高品阶灵药谷阵法区。
- **不抢功**：自己不知道紫色，但也不假装能教——只是回避。这是底线。
- **韧性**：从灵药谷小师兄一路走到飞升神界。
- **【关键修正：藏拙型天才】**——补读后段揭示，王敦后期是被**北极紫薇神帝**亲收的徒弟、"绝世天才"、"独断万古的天骄"、发明**"王敦密码"**作为**传讯仙符**底层理论（这为他带来三尺大小的功德——老子的功德也只有拳头大小）、有**任务奖励系统**（不定期掉传送卡 / 伪装卡 / 体验卡等异宝）。所以他不是"陪跑者借光"，是**有自己的天命只是表面装成普通师兄**。前期友善/胖/卧槽都是真的，但底层有不为人知的强大暗线。这是整部外传的最大反转钩子。

## 动机 / 心结 / 渴望（修正版）

- **想活得久 + 活得有尊严**：表层动机，从灵药谷小师兄时就在。
- **隐藏野心**：他有任务系统加持，每完成一个任务就有奖励。这意味着他**不甘平庸**——是个明白"自己有天命"但要装平凡的人。藏拙是他主动的选择，不是被迫。
- **核心心结**：他和贺平生是**两个天命之子**在同一个时代相遇。表面是兄弟，底里是两个"主角"。这种关系的微妙——既相互成就、又有些许竞争压力——是情感主线。
- **对枯木真人的隐秘心态**：早期被这种低阶长老压着，他装恭敬，但内心可能早已盘算"等我突破，看你还敢吼我"。这是一个张力点。
- **对重华的感情**：晚期妻子。从"绝世天骄被神帝之女追求"到"愿意收她"的过程，原作几乎没写。是个填补留白的好戏。

## 和原作主角（贺平生）的关系

- **入门引路人**：太虚门内门里第一个对贺平生友善的师兄。这一份情贺平生记了一千多章。
- **长期师兄**：248 章里反复出现，从灵药谷开始一路相伴到飞升。
- **晚期得贺平生神格相赠**：飞升神界后，贺平生主动将一枚神格分给王敦，让他从凡人飞升者跃升到神将境。这是兄弟情份的最大兑现，也是王敦人生的最大转折。
- **拒绝感**：王敦对贺平生没有任何反向欲望（不夺宝、不抢机缘、不设计算计）。是纯粹的陪伴型师兄。

## 说话习惯

- 句长较短，常用感叹词（"罢了罢了"、"卧槽"、"呦"、"哈哈哈"等口语化感叹）。
- 喜欢用"师弟"称呼贺平生，态度从一开始的师兄姿态到后期变成几乎平等的称兄道弟。
- 解释时啰嗦但有条理，分点说话："如果是赤红色 → 黄色 → 金色"。
- 得意时哈哈大笑，紧张时深吸一口气。
- 不爱讲大道理，更爱讲实操（怎么测火灵力、怎么收花粉、规则边界在哪）。

## 婚姻 / 家庭

- 道侣（妻子）：**重华**（具体出处章节待考；末段提及"王敦和我妹妹重华的孩子"，'我'是太真，所以太真是重华的姐姐/妹妹）。
- 孩子：**长生**（飞升神界前后所生，需后续精筛章节确认）。

## 留白清单（修正版，含已部分填补的）

部分由补读补上了答案（√），但内心戏 / 触发机制 / 跨章细节仍是续写的发力点：

- [ ] **王敦的家庭 / 入门 / 灵根** —— 原作零交代。续写要给一个能解释他后来"任务系统"的来源故事。
- [√] **王敦中段修为轨迹** —— 补读：第 ~ch 366 已是筑基期；金丹期圆满（约源 line 23852）；元婴期能秒杀元婴（line 29279）；后联手抗化神；终至大罗金仙 / 太乙金仙 / 道元 / 混元。续写要呈现的是**每次突破的内心戏**与**关键机缘**——这部分原作多半略写。
- [√] **任务奖励系统的来源** —— 补读：王敦显然有一个会掉传送卡 / 伪装卡 / 体验卡 / 神格 / 混沌金丹的"系统"。**系统从哪来？谁给的？王敦初期就有还是后来获得？这是外传第一幕的核心钩子。**
- [√] **"王敦密码" 怎么发明的** —— 补读：他发明了传讯仙符的底层理论，由此获得三尺大小的功德。这种"用现代工程思维做修真发明"的桥段极适合漫剧爽点。
- [ ] **如何被紫薇神帝看上 + 拜师** —— 补读只确认了"如今拜在我父亲门下"。具体过程是外传中段的高潮。
- [ ] **重华公主主动倒追的细节** —— 补读：重华明显主动；具体她怎么爱上王敦的，原作不写。
- [ ] **聚宝盆怎么来的** —— 晚期才出现，待续写自造来源。
- [ ] **第一次见贺平生时的真实心理** —— 现在我们知道他是天才 + 系统宿主了，那他第一眼看到 "费钱货" 五行灵根的贺平生时——他的系统响不响？他知不知道这小子也是天命之子？这就是开篇 hook。
- [ ] **被枯木真人压制时的内心** —— 内有大才、外做杂役，被一个二品长老训斥。这种"装着"的张力是早期章节的口感来源。

## 口吻习惯 / 不要做的事

> Demo 阶段会迭代沉淀到这一段。第 5 步用户审过后补。

- 口吻基线：朴实、口语化、肥而活泼。粗口/感叹是底色的一部分（"卧槽"、"沃特玛"、"罢了罢了"、"哈哈哈"），晚期甚至"贱兮兮"——这种市井气贯穿全本，不要写成文绉绉的师兄。
- **要把王敦写成"装拙的天才"**——表层友善胖师兄是真的，底层有任务系统 + 修炼天才 + 紫薇徒 + 王敦密码发明者。藏拙的反差才是漫剧爽点。
- **绝对不能让王敦角色凌驾于贺平生之上**——他是平行天才，不是更强天才。两人最终一同飞升，是兄弟关系不是替代关系。
