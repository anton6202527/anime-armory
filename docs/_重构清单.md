## 关键事实
- import 形态 = 各脚本 `sys.path.insert(0, _COMMON_DIR)` 后**裸模块名导入**（`import disclosure` / `from settings import ...`），不是 `from common import`。
- vendor 手法 = 复制模块到 `<line>/_lib/`，把脚本里的 `_COMMON_DIR` 改指向本线 `_lib`。模块本身零改动。
- 各线实际只依赖 2–5 个**小纯 Python 工具模块**。
