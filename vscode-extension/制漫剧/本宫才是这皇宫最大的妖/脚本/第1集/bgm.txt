# 本宫才是这皇宫最大的妖_第1集_BGM与音效

## 整体情绪
冷宫惊悚宫斗开局，前半段压低、窒息、带审判感；后半段从“赐死”突然翻成“血脉觉醒”。音乐要服务留存：开头三秒给异样，赐死段持续压迫，觉醒段用暗金低频和短促重击把爽点抬起来。

## BGM风格
- 主体：低频弦乐 drone + 古琴断音 + 细碎铃声/金属摩擦。
- 节奏：前段 70-80 BPM 慢压；柳娘子递酒后加入低鼓心跳；觉醒瞬间短暂停拍，再落重鼓。
- 音色：残烛、冷风、衣料摩擦、远处宫铃；避免热血鼓点过早进入。
- 情绪关键词：冷、旧、窒息、妖异、反杀前一秒的安静。

## 关键音效点
| 节点 | 音效 / 音乐处理 |
|---|---|
| 铜镜换脸 | 轻微镜面嗡鸣 + 一声极低反向吸气 |
| 小禾撞门 | 木门撞击、急促脚步、宫铃远响 |
| “送您上路” | BGM压低一拍，留出台词压迫 |
| 托盘三件套 | 白绫布料摩擦、匕首轻碰托盘、酒壶瓷声 |
| 毒酒递唇 | 低鼓心跳加快，苦杏仁气味处用细小弦擦 |
| 系统音出现 | 人声低频叠层，不做电子科幻音 |
| 左腕疤裂光 | 皮肤细裂声 + 暗金能量低鸣 |
| 毒酒落地 | 瓷碎重音，随后短暂静默 |
| 金瞳觉醒 | 低频冲击 + 宫铃倒放 + 一声兽类低吼感 |
| “我说了算” | BGM抽空半拍，尾字落低鼓，留1秒冷场 |
