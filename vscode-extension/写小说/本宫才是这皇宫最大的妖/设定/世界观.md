# 世界观 — 《冷宫有妖气(精修版·名待定)》（改写后·现行）

> 写的是**改写后**的现行世界(原作底色 + 新设定覆盖)。

## 力量 / 修炼 / 魔法体系（现行）
- 人间修炼薄弱,**妖患潜伏**(宫廷尤甚——冷宫是妖的"自助餐厅")。
- 女主独有**噬妖夺能**(妖皇血脉):炼化妖→夺一项神通,**有代价**(同源相斥/贪噬反噬/境界压制)。是全篇升级线与张力核心。
- 妖有品阶:小妖→银牌妖位→…→五品妖师→妖王;以"逆阶斩杀"立威。

## 政治 / 势力格局（现行）
- **帝(林渊)—后(王氏/王家)—太后** 三方制衡;朝堂为宫斗主战场。
- 女主轨迹:冷宫废妃 → 妖患破局者 → 朝堂变量 → "我说了算"。
- **战争/外患线服务于朝堂博弈**(②收束):每次军功都回收到"逼皇后/立威于帝"。
- 玄天阁:炼妖/丹方的外部知识方,既是线索来源也是冲突源。

## 地理
- 冷宫(起点猎场)→ 后宫/朝堂 → 边境(外患·战争线,但镜头始终牵回宫廷)→ 枯井地下封印(血脉来历)。

## 时间线（关键节点·重写后)
1. 穿越觉醒 + 首次噬妖夺能(反杀柳娘,夺潜行)〔第1-3章·①差异化开篇〕
2. 冷宫立足、连斩群妖、银牌妖位立威
3. 皇后正面交锋;林渊起疑亲探(CP暗线起)〔③〕
4. 外患/边境危机——**作为逼宫立威的手段**,军功回收朝堂〔②第40-60章收束〕
5. 妖王/妖皇血脉真相;帝后终局博弈
6. 兴学/女性平权落点 + CP克制收束〔呼应2026"女性觉醒"趋势〕

## 术语表（以现行为准）
- **噬妖夺能**:炼化妖躯夺其一项神通的血脉能力(有代价)。
- **妖纹**:施展时体表浮现、随所夺之妖变化的符号。
- **贪噬反噬**:连续炼化致妖性压人性的代价态,靠清醒压制。
- **妖位 / 妖师 / 妖王**:妖的品阶。
