# 改动spec — 《冷宫有妖气》→《冷宫有妖气(精修版·待定名)》

> 这部改写的"宪法"。本次=**去同质化精修**,不是推倒重来:**绝大多数章节/事件保留**,只动评分报告点名的三处短板(开篇差异化 / 中段主线收束 / 加CP暗线)。每条具体可判定。

## 一句话改动方向
把"穿越冷宫废妃+系统反杀(满街同质化)"的故事,精修成"**靠『吃妖夺神通』这一独特血脉、清醒杀出冷宫**的故事"——**强化辨识度、收束中段、补一条克制情感暗线**,内核与主体情节不变。

## ① 保留的内核(魂,不许丢)
- **主角人设内核**:沈念清醒、冷静果决、**不圣母不恋爱脑不降智**、步步为营——这是魂,任何改动不许动。
- **情感主线**:沈念"从任人宰割的废妃 → 我说了算"的**掌控感与复仇/逆袭爽感**;主仆羁绊(小禾)。
- **世界观底色/基调**:大周朝 + 后宫宫斗 × 玄幻斩妖**双线并行**;冷宫"从囚笼变猎场"。
- **升级爽感链**:杀妖 → 炼血 → 夺神通 → 一路变强(保留,但见 ③ 把"夺神通"做实做独特)。
- **标志性桥段**:穿越觉醒 + 一拳反杀伪善赐死姑姑(柳娘子);夜斩三妖;屠百年蛇妖立威。**保留**。
- **完本结构 + 女性兴学/平权的精神落点**(第100章)——保留(且这正踩 2026"女性觉醒"趋势,是加分项,前置呼应)。

## ② 改的部分(事件 / 设定 / 结局)
- **开篇(第1-3章)主线不改、写法改**:从"泛泛系统提示+反杀"改成**把『吃妖夺神通』当场演示**——首杀柳娘子时不仅"经验+10",而是**炼化其妖躯、夺得它的一项神通/特质**(如柳娘的"伪人皮"潜行),让女主当场尝到"吃一个、强一分、得一术"的独特爽点。文案/简介同步重写(见下)。
- **中段(约第40-60章)主线收束**:原作此处从宫斗**跳**到徐州守城/家国战争(镇妖使),女频核心被稀释。改为:**战争线服务于宫斗主轴**——出征是"打脸朝堂/逼皇后/立威于帝"的手段而非另起炉灶;**保留并穿插宫斗与羁绊**(皇后/太后的暗手、小禾、CP暗线不断线);军功最终**回收到"这皇宫我说了算"**的爽点上,而不是变成纯战争爽。删/压缩与主轴无关的战争支线铺陈。
- **改的设定**:把"万妖血脉"从"打怪升级的系统"明确为"**妖皇血脉·噬妖夺能**"体系(详见 新设定.md 登记)。
- **结局**:不改大走向(兴学/平权),仅让 CP 暗线在结局有一个克制的收束(不落俗套大婚)。

## ③ 加的新料(清单;详细体系进 新设定.md)
- **新金手指做实**:`噬妖夺能`——炼化妖躯可**夺其一项神通/特质**(有**代价/上限**:同源相斥、贪噬反噬),把"无敌流"从"万能"改成"**有取舍的强**",治无敌流通病(④人设维度抓手)。
- **CP暗线(克制·不恋爱脑)·男主=永宁帝·林渊**〔已定〕:当年信谗言贬她入冷宫者。基调=**势均力敌的试探与制衡**,她始终在主导位,不为爱降智;全书只给"暗线"密度(眼神/博弈/关键时刻互为底牌),**不写恋爱脑桥段**;结局克制收束(不落俗套大婚)。
- **新人物**:无须大增;CP线复用既有林渊。
