# 冷宫有妖气_第6集_BGM与音效

整体情绪曲线：压境威压（开场仪仗）→ 紧张悄声（小禾劝阻+换装铜镜变身）→ 行进低压（宫道权力梯度）→ 殿内幽冷暗涌（妖气感知禁术伏笔）→ 言辞机锋（试探→翻脸摔杯·情绪顶点）→ 反转阴笑（皇后突然换脸笑·派人）→ 暗潮（翠微妖气可见）→ 诡异静默（子时木桩夜探）→ 冷冽收尾（钩子留白）

本集为**文戏对峙**主导：BGM 整体走"低频暗涌 + 弦乐机锋"，几乎不用打击鼓点；唯一两个动态重音=**Clip 7 摔杯**与 **Clip 9 翠微妖气可见叠化**。

BGM 风格：
- 镜头1-2（仪仗压境/宣旨）：低频管乐压迫嗡鸣 + 一记沉重定音鼓，威压开场
- 镜头3（小禾劝阻）：短促紧张的小弦乐颤音，制造悄声悬念
- 镜头4（铜镜变身）：渐变的清越弦乐——从颓→挺、由凉→冷，伴随沈念神态变化
- 镜头5（宫道行进）：低音弦乐缓铺底 + 远远的宫门钟磬尾韵，行进感、权力梯度
- 镜头6（入殿·禁术感知）：檀香烟雾质感的呼吸式低吟 + 一根高频不和谐弦音（标记禁术伏笔）
- 镜头7-8（首见皇后·试探）：弦乐机锋——皇后线高贵但冷、沈念线静水深流，对位推进
- 镜头9（摔杯·情绪顶点）：一记锐利重音 + 戛然留白，瓷裂飞溅清晰可闻
- 镜头10（沈念回击）：留白半秒 + 一根弦乐轻颤（暗示她占了上风）
- 镜头11（反转笑）：温柔却阴冷的弦乐反转音色——比刚才更可怕
- 镜头12（翠微妖气可见）：一段诡异墨绿色调的高频提琴和声 + 低吟妖气音效
- 镜头13（回冷宫低语）：弦乐压低暗涌，环境安静突出对话
- 镜头14（子时木桩夜探）：**几乎全静音**——只留极远夜风、无呼吸的诡异感（以静制诡，这是钩子的关键音色策略）
- 镜头15（钩子收尾）：幽冷弦乐单音收尾留白，意味深长

关键音效点：
[镜头1] 沉重脚步整齐落地 + 铁门腥锈风 + 定音鼓
[镜头2] 绢帛轻响 + 冷风过门
[镜头3] 衣袖摩擦 + 小禾抽噎 + 紧张弦颤
[镜头4] 铜镜微响 + 簪入发声 + 整衣襟
[镜头5] 整齐脚步在长道里回响 + 远远宫门钟磬尾韵
[镜头6] 檀香低吟 + 极轻的不和谐弦音 + 系统光幕闪现的清越电子音
[镜头7] 宽袖垂落拂地 + 檀香烟雾轻响
[镜头8] 金护甲叩瓷"叮" + 茶汤微响
[镜头9] **啪——白瓷砸案瓷裂 + 茶汤迸溅 + 一记低沉重音 + 戛然留白**（情绪顶点）
[镜头10] 死寂半秒 + 一根弦轻颤
[镜头11] 檀香烟轻散 + 温柔却阴冷的弦乐
