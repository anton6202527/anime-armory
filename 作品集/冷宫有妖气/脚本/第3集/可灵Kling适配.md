# 冷宫有妖气_第3集_可灵Kling适配（跨平台落地样例）

> **不变（复用）**：`分镜剧本.md` / 角色卡 / 场景卡 / `voiceover.txt` / `字幕_中/英.srt` / `bgm.txt`。
> **只换"视频生成"适配层**：可灵=**首帧图 + 尾帧图 + 运动笔刷 + 运镜**。一致性=首帧图复用角色定妆照。
> 锚点简写：沈念常=「乌黑长发束起·凤眼薄唇·肤白·左腕疤·深色夜行劲装」；沈念觉醒=「双臂暗金图腾妖纹·金色竖瞳·暗金妖气」；三小妖=「洗白旧宫女衣·鳞片妖(黄绿竖瞳)/枯枝指妖(细长倒刺指)/裂口妖(嘴裂到耳根锯齿牙)」；蛇妖化形=「白衣广袖·乌黑长发披散·碧绿竖瞳似笑非笑」；系统光幕=「半透明青绿·古篆·暗金描边」；场景 LOC_03=「夜·荒院老槐树·树冠遮天·稀薄月光」、LOC_04=「夜·枯井·青苔暗红血痕·碧绿妖气升腾」；画风=「国风写实漫剧·电影级光影·9:16」。
> 本集特点：**多组近身打杀（剧烈、首尾帧差异大）+ 蛇妖破井腾空（大位移）**——剧烈动作镜统一压到 5~6s、加中间帧、调高"运动幅度"；**觉醒态妖纹蔓延、蛇妖破井是渐变/特效=强项**。

## Clip 1（7s）夜行出击 — 镜头1+2（建议拆 2 镜）
**1a**首帧：沈念深色劲装推门出偏殿、回眸眼底暗金一闪；尾帧：迈步入夜、夜风掀发摆。运动笔刷：人→出门迈步。运镜：中景固定
**1b**首帧：沈念穿过荒草绕过倒塌假山；尾帧：走到遮天蔽日老槐树下、妖气淡光绕身。运动笔刷：人→行走；草叶→夜风拂动。运镜：跟拍

## Clip 2（7s）三妖分食+揭吃人 — 镜头3+4
**首帧图**：中近景，老槐树下三只人形小妖（三小妖锚）围地分食一只死黑猫、妖气幽幽
**尾帧图**：裂口妖咧锯齿笑、鳞片妖舔唇；切沈念立于假山阴影眼神骤冷
**运动笔刷**：小妖→撕咬/抬头；妖气→飘动
**运镜**：缓推 → 切假山阴影
**说明**：三妖同框静→动幅度小，单镜可行；表情狰狞靠首尾帧。

## Clip 3（7s）现身对峙+嗅出异常 — 镜头5+6
**首帧图**：中景，沈念缓步出阴影，三妖齐刷抬头、六瞳锁定她
**尾帧图**：近景，裂口妖鼻子抽动惊变后退半步，沈念面无表情继续逼近
**运动笔刷**：三妖→起身回头；裂口妖→后退；沈念→缓步前移
**运镜**：中景固定 → 微推

## Clip 4（5s）秒杀1·一拳爆胸 — 镜头7（剧烈动作）
**首帧图**：沈念觉醒态（沈念觉醒锚）瞬移到鳞片妖面前、出拳前残影
**中间帧**：拳轰中鳞片妖胸口、肋断冲击
**尾帧图**：鳞片妖如炮弹倒飞撞老槐树、树皮崩裂树冠剧震、黑血迸溅
**运动笔刷**：① 拳→直线轰击胸口；② 小妖→倒飞撞树
**运镜**：高速跟拳 + 轻甩镜；0.4s 慢动作强调命中
**说明**：剧烈打飞，务必加中间帧、时长 5s、运动幅度调高。

## Clip 5（6s）觉醒态威压 — 镜头8
**首帧图**：近景，另两妖僵立后退；沈念暗金妖纹刚自双臂浮起
**尾帧图**：妖纹爬满双臂脖颈幽幽发光、金色竖瞳、嘴角冷笑、发丝被妖气掀起
**运动笔刷**：妖纹→蔓延铺满；发丝→被妖气掀起
**运镜**：环绕半圈
**说明**：妖纹蔓延=渐变强项，一镜到底。

## Clip 6（6s）秒杀2+3 — 镜头9（剧烈动作，建议拆 2 镜）
**6a**首帧：裂口妖扑咬沈念咽喉；尾帧：沈念侧闪、右手掐住其喉、小妖化黑烟噗散。运动笔刷：手→掐喉；妖→碎成黑烟。运镜：动态特写
**6b**首帧：枯枝指妖转身逃；尾帧：沈念追上一拳砸后脑、小妖前栽倒地尘土扬。运动笔刷：沈念→追击挥拳；妖→栽倒。运镜：跟拍

## Clip 7（6s）收势+升级 — 镜头10
**首帧图**：沈念立定，暗金妖纹与竖瞳正在褪去
**尾帧图**：恢复常态，夜风卷血腥味衣摆轻扬，眼前浮现青绿系统光幕（经验80/100）
**运动笔刷**：妖纹/竖瞳→褪去；光幕→浮现
**运镜**：缓拉

## Clip 8（6s）枯井召唤 — 镜头11
**首帧图**：俯视井口，沈念立于枯井边低头望向漆黑井底，碧绿妖气缕缕升腾
**尾帧图**：近景沈念启唇说话，青苔与暗红血痕在月光下若隐若现
**运动笔刷**：妖气→自井底升腾
**运镜**：俯视井口 → 推近沈念

## Clip 9（7s）蛇妖破井登场 — 镜头12（大位移特效·强项）
**首帧图**：枯井口漆黑，一道白光自井底破空冲起
**尾帧图**：百年蛇妖（蛇妖化形锚）长发与广袖白袍翻飞、稳稳落在沈念面前、碧绿竖瞳似笑非笑、森冷妖气如涟漪扩散
**运动笔刷**：白影→自井底冲天而起并落地；妖气→涟漪扩散
**运镜**：仰拍冲天 → 落地定
**说明**：腾空亮相=可灵擅长的特效大动作，首帧（井底白光）→尾帧（落地亮相）效果好。

## Clip 10（8s）对峙+钩子 — 镜头13+14
**首帧图**：正反打近景，蛇妖碧瞳掠贪婪轻笑、沈念无惧微笑回应
**尾帧图**：蛇妖转身纵跃没入井中黑暗，沈念独立井边、侧脸嘴角微微上扬定格
**运动笔刷**：蛇妖→转身跃入井中
**运镜**：正反打 → 中景→侧脸特写缓推
**说明**：含对话停顿，可拆"对峙近景"+"跃入井+沈念定格"两镜。
