# 导演视角 Prompt 八维 / MV 出图·出视频 prompt 装配规范（mv 线共用）

> **核心信念**：MV 的画面，**画师**问"主角美不美、画面炫不炫"；**导演**问"这一镜在替**这段音乐**说什么"。
> MV 没有连续剧情，靠的是**情绪 + 节奏 + 视觉冲击**——所以八维里 **⑥光影**（演出光/色胶/霓虹/逆光剪影）和 **⑦情绪·张力**（绑段落+卡点）权重比漫剧更高，**②机位**（大运动/换机位）是副歌高能的命。
> 本文件是 MV 线自包含版（**不复用 n2d 的同名 doc**），mv-image(出图)、mv-video(出视频) 都按它装配 prompt。与 `视觉蓝图.md`(段落↔画面) + `beatgrid.json`(卡点) 配合落地。

---

## 一、八维：画师视角 → 导演视角（MV 版）

| 维度 | 画师视角（停在这）| 导演视角（还必须回答，MV 重点）| prompt 体现 |
|---|---|---|---|
| **① 镜头** | 来个主角特写 | 景别为何是它 + 焦段/景深（副歌大特写冲击·verse 中景叙事·intro 大远景氛围）| `大特写，长焦浅景深` |
| **② 机位** | 正面平视 | 角度+高度+运动起点；**MV 副歌靠大胆机位**（低角度仰拍、荷兰角倾斜、俯冲）——机位即能量 | `荷兰角倾斜，低角度仰拍` |
| **③ 人物** | 主角好看 | 锚点句锁脸锁造型（跨段不漂）+ 此刻状态（妆造随段落变：intro 素→chorus 浓）| `同一歌手[锚点句]，副歌浓妆亮片` |
| **④ 动作** | 摆 pose | 静帧=情绪最强一瞬；视频=人物运动+表情，**踩段落**（chorus 大动作/甩发、verse 微表情）| 视频：`随鼓点甩发→张口嘶吼` |
| **⑤ 场景** | 背景对 | 场景锚 + 时间/环境（舞台/街景/天台）+ 氛围（雨/雾/烟/光斑），跨段可换景做反转 | `霓虹街口，夜，雨后地面反光` |
| **⑥ 光影** | 打亮 | **MV 的灵魂**：演出光(追光/频闪/染色光)+方向(逆光剪影/侧光轮廓)+调性(高对比/暗调)+**色胶**(品红/青/橙) | `品红+青双色胶，逆光剪影，高对比` |
| **⑦ 情绪** | 表情对 | 段落情绪 + 张力(克制/蓄力/爆发/释放，**对齐 beatgrid**) + 色调情绪——驱动运镜幅度与卡点疏密 | `chorus=爆发，张力拉满，暖橙高饱和` |
| **⑧ 画质** | 高清 | 画风词统一(依视觉蓝图 global_style，全曲不跳)+渲染级(MV 强色彩/胶片/故障感)+画幅+负面词 | `MV 强色彩，胶片颗粒，9:16` |

> MV 最易漏也最关键：**②机位（别全程正面平视）、⑥光影（别均匀打亮——MV 就活在戏剧光里）、⑦张力（必须对齐 beatgrid 段落）**。

## 二、八维 × 段落（MV 的装配主轴）

MV 的八维不是按剧情走，是**按段落 + 卡点走**。每段的"导演取向"已在 `prompt_format.md` 的段落表里，八维据此填：

| 段落 | ⑦张力 | ②机位/运镜 | ⑥光影/色调 | ④动作 | 卡点疏密 |
|---|---|---|---|---|---|
| intro | 克制 | 固定/极缓推、大远景 | 暗调/单色、氛围光 | 入场/静态 | 疏 |
| verse | 叙事 | 缓推/跟、中景 | 自然光/低饱和 | 微表情/小动作 | 中 |
| pre-chorus | 蓄力 | 渐快推近、机位逼近 | 渐亮/升饱和 | 蓄势 | 中→密 |
| **chorus** | **爆发** | **快推/环绕/轻甩/荷兰角、大特写** | **追光/频闪/强色胶、高对比** | **大动作/甩发/嘶吼** | **密(踩鼓点)** |
| bridge | 反转 | 换机位/换景、大运动 | 换色/冷暖反转 | 转变 | 中 |
| outro | 释放 | 缓拉远/定格、大远景 | 调暗/留白 | 收束 | 疏 |

## 三、图 prompt 装配模板（mv-image）

```
[①镜头][②机位]，[③歌手·锚点句·本段妆造] 正在 [④情绪最强一瞬],
置身 [⑤场景+环境氛围]，[⑥演出光+色胶+调性]，
整体 [⑦本段情绪+张力+色调]，[⑧MV画风+渲染级+画幅]
负面：[统一负面词 + 本镜堵漏]
```

> 定妆图仍是**中性档案**（正面/均匀光/无戏，只锁脸锁造型）——**戏剧光/色胶只上分镜图**，定妆图打 MV 浓光会污染下游参考。

## 四、视频 prompt（mv-video）：八维 + 卡点

①③⑤⑥⑧ 由首帧 PNG 锁死，视频只升级：

- **④动作 → 人物运动+表情**：踩段落（chorus 大动作随鼓点、verse 微动）。
- **②机位 → 运镜**：服务⑦张力（副歌快推/环绕/轻甩、verse 缓推跟、bridge 换机位）；**关键运动对齐 downbeat**。
- **⑦张力**每条带词（克制/蓄力/爆发/释放），决定运镜幅度 + clip 时长（副歌碎切、verse 缓）。
- **动态细节**：发丝/光斑/光束扫动/烟雾/衣摆 ≥1 条。

> 视频里**不重定**①镜头/③人物/⑤场景/⑥光影/⑧画风（首帧已锁），改了=与首帧打架=闪烁漂移。

## 五、导演视角自查（每条 MV prompt 落笔前）

- [ ] **①镜头**景别有理由（副歌大特写/intro 大远景）？
- [ ] **②机位**非默认正面平视？副歌用了大胆机位/运动？
- [ ] **③人物**锚点句已拼 + 妆造随段落？
- [ ] **④动作**踩了段落（chorus 大动作/verse 微动）？视频写了运动+表情？
- [ ] **⑤场景**锚+环境氛围齐？
- [ ] **⑥光影**用了演出光/色胶/调性（**不是均匀打亮**）？
- [ ] **⑦情绪·张力**对齐 beatgrid 段落？色调情绪一致？
- [ ] **⑧画质**画风依视觉蓝图全曲统一？画幅对？
- [ ] **三维复检**：②机位、⑥光影、⑦张力——有没有偷懒回到画师视角？

> 一句话：**八维全想到，MV 才从"一串好看的主角图"变成"替音乐叙事的镜头流"。** 配合段落表 + beatgrid 一起过。
