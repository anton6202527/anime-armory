# 出图 prompt 格式（ad-image）

## §1 三层定妆库（`出图/共享/prompt/`）

- **角色/代言人**：标准三视图（正/侧/背），≥1024px，3–6 角度。prompt 锁五官/发型/服装/气质 + 锚点句。
- **场景**：关键场景四视图，锁结构/光环境/色温。
- **产品（hero）**：包装正/侧/背 + logo/材质细节特写。prompt 写死品牌色 HEX、logo 位置、包装文字、禁改清单（见 `产品一致性checklist.md`）。

## §2 视觉一致性契约总览（`出图/分镜/prompt/00_总览.md`）

继承 `storyboard.json.visual_contract`：
- 品牌色（主/辅 HEX）、基础视觉风格、光位锚、镜头与构图策略、运动边界、风格禁忌。
- 把"视频改不动、要烤进首帧像素"的决策（色调/光位/轴线·视线/景别/品牌色）写清，供 `ad-video` 继承。

## §3 逐镜 prompt（`出图/分镜/prompt/镜头N.md`）

每镜八维（导演视角）+ 资产绑定：
```
镜头N | 景别 | 角度 | 主体动作 | 场景 | 光位/色温 | 情绪 | 完成度
资产身份注册：CHAR_xx（角色）
资产引用注册：PROD_main（产品定妆参考组）/ LOC_xx（场景）
品牌色：#E60012（主色出现在 …）
锚点句：与定妆参考①同一人/同一包装
身份锁定句：同一张脸 / 同一 logo 同一品牌色
起幅：首帧=起幅非动作顶点，为运镜留构图余量
安全框：主体/文字留 title-safe 余量（多比例 reframe 用）
负向：不要偏色 / 不要改包装文字 / 不要变形 logo
need_end_frame：是否出尾帧（接力）
```

## §4 尾帧接力

`storyboard.json` 标 `continuity.need_end_frame` 的接缝出 `镜头N_end.png`（= 下一镜首帧构图），`ad-video` 双帧引导焊接点。命名：首帧名 + `_end`。
