# 拆集法 / 集边界决策规范（阶段1·拆集必读）

> 目的：定**集与集之间的边界**——一部小说切成多少集、每集起止在哪。
> 这是 `导演节奏.md` 的**前置**：`导演节奏` 管「一集**内部**怎么走（留存曲线）」，本文件管「**集与集之间**的刀切在哪」。先切对边界，集内节奏才有意义。
> **一句话**：集边界由**结尾的悬念断点**决定，不由字数决定。`节拍盖过字数，配音盖过估算`。

---

## 拆集优先级（从最先下刀到最后微调，逐级让位）

下刀按 P0→P6 顺序判断；**低优先级永远让位于高优先级**——字数（P4）冲突时，永远服从断点（P0）/闭环（P1）。

### P0 — 先找「结尾」，不找「开头」（第一性原理）

一集的边界由它**结尾的悬念断点**决定。拆集第一步不是数字数，是**在原文里扫所有"最吊胃口、可硬断"的点**，那些点就是集的**右边界候选**：

- 危机悬置（刀架脖子上 / 门被推开 / 倒计时归零前一瞬）
- 真相半露（抖出一半，关键那句不说）
- 反转预告（一个眼神 / 一封信 / 一道系统音 = 要变天的信号）

把候选断点标出来，**集 = 两个相邻强断点之间**。`--by-chapter` 让粗胚先贴章界，但**章界 ≠ 断点**，必须人工把右边界 snap 到最近的强断点（可 ±一段）。

> 反例：某集结尾停在「她走进了大殿」——这不是断点，是半场戏。往后挪到「大殿门在身后合上，龙椅上坐着的人，是她以为已经死了的人」。

### P1 — 每集装下一个完整「憋—放」闭环

每集至少含一个 **冲突→爽点(或反转)** 的微弧——观众一集**出一次气**（或留一个强到足以抵消的钩）。

- **拆集第一大忌：整集全是铺垫、零出口**。宁可把相邻铺垫段并进有爽点的那集，也不要切出一个"纯憋着"的集。
- 例外：当本集的"憋"本身**断在爽点引爆前一帧**（P0 的危机悬置），这个 cliffhanger 就是它的出口——允许。
- 爽点要有"憋—放"距离：跨集时，"憋"可放在上集尾、"放"在下集头，但**别把一个完整爽点劈成两半各放半个**（上集没爽、下集莫名其妙）。

### P2 — 切点要让**下一集**能 0-3s 冷开场

边界不只服务"本集怎么收"，还服务"**下集怎么起**"。

- 切点之后的第一段，必须能当**冷开场钩**（最炸的画面/台词）起。
- **不许切出"以纯过渡/交代戏开头"的集**（下集第一句是"话说三天后……"= 切错了）。
- 检验：把切点想象成下集第 0 秒——这一秒抓得住人吗？抓不住，把边界往前/后挪到能抓人的地方。

### P3 — 钩子密度达标，杜绝「纯过渡集」

中段每 15-20s 一个钩子（悬念/欲望/反差/信息/危机，见 `导演节奏.md §二`）。

- 一段原文太平（长环境描写 / 大段内心戏 / 流水账交代），**两条路**：① 压缩进相邻集；② 与相邻动作集**合并**，让平淡段被高张力段"带飞"。
- **绝不**让平淡段独立成集——那是必划走的"纯过渡集"。
- 若某段戏信息量撑不起一集、又删不掉，就**并集**（一集可跨多章）。

### P4 — 时长是「验收区间」，不是目标点

时长只做**容量约束**，是 P0-P3 切完后的**验收窗口**，不是下刀依据。

- **前长后短**（默认）：第1集 120-180s 拉满留存；其余集 60-120s 碎快追下集。
- 字数 band 只用来切**粗胚**（估总集数+每集容量），换算是高方差代理（对白/打斗压缩少、环境/心理压缩多）。
- 落进区间靠 **P0-P3 节拍重切 + 配音实测校准**，不靠凑字数。某集为保完整闭环而超/欠区间，**P1 优先，区间让位**（记一笔，配音后再看要不要拆/并）。

### P5 — 尽量切在场景 / 视角 / 时间的自然幕界（生产成本）

在满足 P0-P4 的前提下，**优先把断点踩在**：

- 换场景（出宫→入府）、换视角（A 的戏→B 的戏）、时间跳跃（当夜→次日）处。
- 好处：跨集断点踩在自然幕界，能**降低跨集崩脸/场景漂移**，且利于**定妆库与分镜复用**（同一集内尽量同场景/同主角集合，少切换 = 少漂移）。
- 当强断点（P0）正好落在幕界——最理想，优先采用。

### P6 — 角色登场节奏 + 首集加权

- **新角色分摊登场**：别一集塞太多新面孔；每个重要角色给一个"立人设"的高光时刻（主角登场本身就是好集头钩）。新角色集中爆发 = 观众记不住 = 弃剧。
- **首集单独加权**：首集除自身闭环外，还要扛 **世界观 + 主角核心欲望 + 系列总钩（追完整部的那个大悬念）**——所以更长（120-180s），留存投资最高。重切时把更多开篇内容并进第1集。

## 拆集自查清单（开拆后逐集核对·配合 `导演节奏.md` 集内自查）

- [ ] **P0** 每集结尾都落在一个强断点（危机悬置/真相半露/反转预告），没有一集停在半场戏/平淡处？
- [ ] **P1** 每集都有至少一个完整"憋—放"闭环（或断在爽点前一帧的合法 cliffhanger）？没有"纯铺垫集"？
- [ ] **P2** 每集切点之后能 0-3s 冷开场？没有"以纯过渡/交代戏开头"的集？
- [ ] **P3** 没有"纯过渡集"？平淡段都被压缩或并入相邻动作集？
- [ ] **P4** 各集时长落进 `_设置.md` 的验收区间（首集 120-180s / 其余 60-120s）？为保闭环超区间的已记录、待配音复核？
- [ ] **P5** 断点尽量踩在换景/换视角/时间跳跃处？同一集内场景/主角集合尽量收敛（利于复用、防漂移）？
- [ ] **P6** 新角色分摊登场、各有立人设时刻？首集承担了世界观+主角欲望+系列总钩？
- [ ] **章 ≠ 集**：没有机械"一章=一集"？节拍跨多章的已合并、长章的已上/下集？

> 一句话：**`拆集法` 让"切在哪"对，`导演节奏` 让"切出来的每一集"追得停不下来。** 先过本清单（集边界），再过 `导演节奏` 清单（集内留存）。
