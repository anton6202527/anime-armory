# 仙侠场面工艺 / 御剑飞行·追逐·渡劫·炼丹炼器·大阵·大场面·斗法对轰·神魂（仙侠玄幻必读）

> `打斗分镜.md` 管"打斗这一种戏怎么拆才又顺又燃"；**本文件管打斗之外的几类仙侠招牌硬场面**。
> 这几类都和打斗一样：**是核心卖点 / 爽点，绕不过去，但 AI 直接生必崩**（飞行飘、追逐丢空间、雷劫乱闪、丹火漂、阵图歪、大远景糊）。
> 共用同一套核心方法论，按场景类型分别落地。阶段2 设计含这些场面的集时，`分镜剧本.md`/`故事板.md` 按本文件拆；下游 n2d-image/n2d-video/n2d-compose/n2d-review 各取所需。

## 〇、核心方法论（和打斗共用的一句话）

> **把一段奇观拆成关键帧 → 用图把关键帧锁死 → 用首尾帧让视频只做"帧间插值/受控运动"（不让 AI 自由发挥）→ 用剪辑+音效+特效在后期补"感"（速度感/危机感/打击感/恢弘感）。**

这是 n2d「先出图锁死、图生视频只控运动」铁律在所有奇观戏上的延伸。崩，根因都是"让 AI 自己编"；解法都是"形态由关键帧图决定，视频只负责受控地动起来"。

**这几类场面共同的三大死因 → 解法（速查）：**

| 死因 | 现象 | 解法（落到哪一步）|
|---|---|---|
| 让 AI 编大动态 | 飞行飘忽、追逐丢距离、雷击乱闪 | **关键帧出图锁死 + 首尾帧/受控运动引导视频**（image+video）|
| 奇观元素漂 | 飞剑/云/劫雷/丹火/阵图每镜不一样 | **奇观元素进共享定妆库**（image，和角色脸/法宝特效同等）|
| 没"空间/递进交代" | 看不懂谁在哪、强度没层次 | **establish 广角 + 节奏曲线分段**（script）|

**和"动背景不动人"的关键技巧**：飞行/追逐这类"高速移动"，**别让 AI 动人物**（一动就崩姿态崩脸），而是**锁住人物姿态、让背景/云层/速度线高速流动 + 镜头跟随**——速度感来自背景与镜头，不来自人物自身变形。这是本文件最值钱的一条。

## 一、御剑飞行 / 飞行类（御剑·御物·腾云·踏空·轻功掠空）

### 1.1 难点
飞行 AI 最易：人物**飘**（姿态不稳像悬浮）、剑/云**不稳**（每帧形态变）、**没速度感**（像定在原地）、背景**贴图感**（人与景两层皮）、镜头不知怎么动。

### 1.2 关键帧拆解（飞行三态）
| # | 态 | 画什么 | 景别/运镜 |
|---|---|---|---|
| 1 | **起飞** | 落剑/踏空/腾身，剑光亮起、衣袂气劲鼓荡、足下生云 | MS/LS·由下仰拍（显腾起）|
| 2 | **巡航**（定妆级）| **标准飞行姿态**：御剑（前倾/负手立剑）或腾云（盘坐/踏云），背景云层/山河高速后掠 | LS/跟拍·侧飞或对冲 |
| 3 | **机动** | 转向/俯冲/急停/穿云——飞行里的"爽点" | CU/主观·快推或下坠 |

**铁律**：
- **巡航姿态定一张"飞行定妆"复用**（如 `定妆_主角御剑巡航.png`），全程同姿态，只换背景与镜头——这是治"飘"的根。
- **速度感 = 背景流动 + 镜头跟随 + 速度线/拖光**，不靠人物动。
- 飞剑/法器/云的形态**进共享定妆库**（`定妆_青锋飞剑.png`、`定妆_脚下祥云.png`），锁颜色/形状/拖尾。

### 1.3 节奏曲线
establish 远景（交代"在飞·飞向哪"，长镜 2-3s）→ 跟拍中景（姿态+速度，1-2s）→ 机动特写/主观（俯冲穿云=爽点，碎切快）→ 抵达/落地（留白缓收）。**赶路别匀速长飞**，给一个机动爽点（穿云/俯冲/对冲镜头）才不闷。

### 1.4 视频要点（n2d-video）
- **首尾帧锁姿态**：首帧=起飞，尾帧=巡航；或首尾都用巡航帧（姿态几乎不变），运动全交给背景与镜头。
- prompt 写：**"人物姿态几乎保持不变；云层/山河向后高速掠过；镜头侧向跟飞（或对冲快推）；衣袂、剑穗、发丝向后飘"**——明确"动的是背景不是人"。
- 机动镜头才让人物动（转头/俯身下压），且给镜头同步运动掩护。

### 1.5 迷你示例
```
Clip 御剑追赶（1.6s · 巡航→俯冲）
首帧：出图/第N集/御剑_巡航.png（前倾御青锋，负手，云海背景）
尾帧：出图/第N集/御剑_俯冲.png（同人同剑，身体前压、剑尖下指）
视频 prompt：人物姿态由前倾转为前压下俯（小幅）；脚下青锋稳定（参考 定妆_青锋飞剑）；
            云海向后高速翻涌、地表山峦快速下移并放大；镜头跟飞并下压跟俯冲；
            衣袂剑穗发丝全向后疾飘、叠速度线拖光。
```

## 二、追逐（地面轻功追 / 御剑空中追 / 法术追逐：剑气法宝在后追人）

### 2.1 难点
追逐 = 连续运动 + 空间关系，AI 易丢 **谁追谁 / 距离感 / 方向**；双人同框跑必崩；背景重复单调；**轴线乱**（一会左→右一会右→左，看着像迎面对跑）。

### 2.2 关键帧拆解（追逐四拍）
| # | 拍 | 画什么 | 镜 |
|---|---|---|---|
| 1 | **发现 / 拉开** | 逃者狂奔、回头瞥见追者；交代初始距离 | 逃者镜（主观回头）|
| 2 | **逼近** | 追者迫近、距离缩小（剪辑制造"越追越近"）| 追者镜 |
| 3 | **险情** | 障碍/分叉/绝路/反击——追逐的"心跳点" | 环境/正反打 |
| 4 | **结果** | 甩开（喘息）或被擒（接下一段戏）| 收束镜 |

**铁律**：
- **正反打交替代替双人同框**（追者镜 ↔ 逃者镜），**靠剪辑制造距离变化**，别强求一镜两人追。
- **轴线一致**：追逐方向全程同向（都左→右），换向要给一个明确的转向镜过渡，否则观众方向感崩。
- **速度感同飞行**：镜头跟拍 + 背景流动 + **前景遮挡物快速划过**（廊柱/树干/人群掠过镜头）+ 偶发障碍。
- 路线环境锚点入库（过同一座桥/林/街），别每镜换地图。

### 2.3 节奏曲线
紧（快切碎，跟拍）→ **险情顿一下**（障碍/绝路放大危机，半拍慢）→ 更紧（提速）→ 结果（甩开=留白喘息 / 被擒=硬切定格）。追逐就是"紧—险—更紧—放"。

### 2.4 视频要点
- 单人跟拍 clip（追者一段 + 逃者一段），别一镜两人；用**主观镜头**（逃者回头看追者迫近 / 追者视角锁定背影）拉代入。
- prompt：**"人物原地跑动姿态循环；背景与前景遮挡物向后高速流动；镜头跟拍（侧随/背跟）"**——又是"动背景"。

### 2.5 迷你示例
```
Clip 巷中追逃（险情拍）
逃者镜(0.7s·快)：少女狂奔回头，惊惶；前景廊柱快速划过镜头
追者镜(0.6s·快)：黑衣人贴地疾掠迫近，画面同向（左→右）
险情镜(0.9s·半顿)：前方死巷/横出障碍——危机放大，少女急刹
后期：脚步/破风快节奏 + 险情瞬间 BGM 抽一下 + 心跳低频
```

## 三、渡劫 · 突破（天劫雷云 / 渡劫雷电 / 境界突破光柱）

### 3.1 难点
雷电 / 天劫光柱 AI 乱闪、不连贯、**雷与人的接触点假**（雷不知劈在哪）、光效漂、缺"**层层加码**"的递增节奏（一道雷和九道雷拍得一样就不燃）。

### 3.2 关键帧拆解（渡劫五段·递增）
| # | 段 | 画什么 | 节奏 |
|---|---|---|---|
| 1 | **天象异变** | 乌云汇聚、天地变色、罡风起（establish）| 长镜·缓推，憋 |
| 2 | **首劫·试探** | 第一道劫雷劈下，人物凝神承受 | 中速 |
| 3 | **连劫·递增** | 一道比一道猛、越来越密越来越亮 | 逐道加速 |
| 4 | **死生一劫** ⭐ | 最凶一劫（爽点顶点）：雷柱轰顶、护体几碎 | **顿帧·CU·最长** |
| 5 | **突破** | 撑过 → 光柱冲天 / 气势暴涨 / 境界跃升 | 爆发后留白 |

**铁律（和打斗命中帧同构）**：
- **雷击命中帧必出独立图**（雷劈中人/护体罩的接触帧）——和打斗"命中帧"一个地位，80% 的"劈到了"在这一帧。
- **层层加码**：劫雷一道道来，每道间留"喘息+蓄力"，最后一道最长、最亮、顿最久。别九道雷平均用力。
- 劫雷形态 / 护体光 / 突破光柱**进共享定妆库**（`定妆_紫劫雷.png`、`定妆_突破金光柱.png`），多道雷复用同一定妆不同角度/粗细。

### 3.3 视频要点
- 雷落用**首尾帧**（将落：雷蓄于云 → 击中：雷柱触体），AI 只插值雷的下落，不自由乱闪。
- 人物承受：**锁姿态**（凝立/单膝撑地），光效闪烁/雷光交给特效与后期，别让 AI 动人。
- 突破：光柱**上升运动** + 人物气势张开（缓抬头/睁眼），镜头缓拉远显恢弘。

### 3.4 后期（n2d-compose）
- 雷击命中 = 打斗命中同款四件套：**顿帧 + 炸雷重低音 + 强闪白（比打斗更强，可满帧白）+ 震屏**。
- 递增感：每道雷的音量/闪白强度逐道加大，最后一道顶。
- 突破：光柱起 + **BGM 推到全曲最高潮** + 持续亮白过曝 0.3s + 长留白让"成了"沉下来。

### 3.5 迷你示例
```
Clip 死生一劫（命中帧）
首帧：渡劫_紫雷蓄云.png   尾帧：渡劫_雷柱轰顶.png（参考 定妆_紫劫雷）
视频：紫雷自云端劈下、击中头顶护体金罩；人物单膝撑地姿态不变；镜头微推命中转极慢
后期：命中顿帧5帧 + 炸雷重音 + 满帧闪白1帧 + 震屏3帧；随后护体罩裂、BGM 抽真空 0.5s
→ 接 突破Clip：金光柱冲天、缓抬头睁眼、镜头缓拉远、BGM 推顶
```

## 四、炼丹 · 炼器（丹炉火候 / 法器淬炼 / 灵纹流转）

### 4.1 难点
过程戏 AI 易拍成**静态**；火焰 / 灵光流转**不一致**；丹炉 / 材料 / 成品形态**漂**；缺"**过程递进 + 成败悬念**"的节奏（开炉前不憋一下，成丹就不爽）。

### 4.2 关键帧拆解（炼制五拍·过程蒙太奇）
| # | 拍 | 画什么 | 景别 |
|---|---|---|---|
| 1 | **备料** | 灵材逐一入镜、品相特写 | CU 特写 |
| 2 | **起炉控火** | 灵力注入、丹火升腾、手诀控火候 | MCU 手/炉 |
| 3 | **异变 / 险情** | 走火 / 丹纹紊乱 / 炉鸣异响——**成败悬念** | CU·顿 |
| 4 | **凝结成形** | 丹胚凝聚 / 器形显现、灵纹缠绕 | CU |
| 5 | **开炉** ⭐ | 炉盖掀开、成丹/成器光华迸发（成）或炸炉（败）| CU→MS·爆发 |

**铁律**：
- **以特写驱动**（手 / 炉 / 火 / 材料 / 成品），少全身——AI 友好，也聚焦"过程的精致"。
- 丹炉 / 火焰色 / 灵纹 / 成品**进共享定妆库**（`定妆_青焰.png`、`定妆_九转金丹.png`），火焰全程同色同质。
- **过程用蒙太奇快切压缩时间**（投料→控火→凝丹一串快切），**开炉前憋一拍**（成败未卜），开炉爆发。

### 4.3 视频 / 后期要点
- 火焰、灵光走"受控流动"（升腾、旋绕），首尾帧锁火形；开炉是爆发镜（光爆+成品升起）。
- 后期：炉鸣 / 控火嗡响 / **开炉成丹清越一响（爽点音效）**；失败=炸炉（同打斗爆发：闪白+震屏+轰响）。
- BGM：过程低回专注 → 开炉前一滞 → 成丹叮地一亮 / 炸炉一炸。

### 4.4 迷你示例
```
Clip 开炉成丹（成·爽点）
凝丹蒙太奇(快切3镜·各0.4s)：投灵草→青焰裹丹胚→丹纹缠绕（参考 定妆_青焰）
憋一拍(0.6s·顿)：炉身微颤、主角凝神——成败未卜
开炉(0.8s·爆发)：炉盖掀飞、九转金丹腾起旋转流光（参考 定妆_九转金丹）
后期：开炉清越一响 + 短光爆 + 主角眼神一亮；BGM 一记上扬
```

## 五、大阵 · 法阵（万剑归宗 / 阵法起势 / 灵光几何阵图）

### 5.1 难点
阵图几何 AI **画不准**（对称崩、符文乱）、起阵过程不连贯、阵法**跨镜不一致**（同一个阵每镜长得不一样）。

### 5.2 关键帧拆解（起阵四层）
| # | 层 | 画什么 | 视角 |
|---|---|---|---|
| 1 | **阵眼 / 阵基** | 阵眼点亮 / 阵旗插定 / 剑阵剑落位（局部）| CU 特写 |
| 2 | **符文连缀** | 符文逐次亮起、灵线连成图 | MS·光线流动 |
| 3 | **全阵激活** ⭐ | 整阵成形（俯视 establish 大场面）、光罩/光柱起 | **俯视/航拍 LS** |
| 4 | **阵法威能** | 困敌 / 攻击 / 防护——阵的"用途爽点" | 阵内人物视角 |

**铁律**：
- **阵图必须进共享定妆库**（`定妆_周天剑阵.png`、`定妆_困灵八卦阵.png`），定**几何 + 符文 + 色**；**全阵俯视图当锚**，局部镜从它多图派生——几何对称 AI 必崩，靠"一张定好的阵图复用"治。
- 视角组合：**俯视/航拍 establish 交代阵型** + 阵内人物视角（被困者仰看光罩）。
- 同一阵**不中途换生图工具**（几何一致性税最贵）。

### 5.3 节奏 / 后期
布阵铺垫（慢）→ 逐符点亮（加速）→ 全阵激活（光柱/罩爆发·爽点）→ 威能。后期：起阵**低频嗡鸣渐强** → 激活一记**轰然光爆 + BGM 推顶** → 阵纹流转持续音。

## 六、大场面 establish（山门洞府 / 秘境奇观 / 万人朝拜 / 仙山云海）

### 6.1 难点
大远景 AI 易**糊**、世界观**贴图感**、跨镜**地理不一致**、**人/景比例失调**（仙山该巍峨却显小）。这不是动作戏，是"**空间交代**"，最容易被随便一镜带过而显廉价。

### 6.2 关键帧拆解（进入三镜·由远及近）
| # | 镜 | 画什么 | 运镜 |
|---|---|---|---|
| 1 | **大远景·气势** | 全貌：仙山 / 宗门 / 秘境全景，云海衬底 | ELS·缓摇 / 航拍缓降 |
| 2 | **中景·地标** | 标志性局部（山门匾、主峰、巨阵）| LS·缓推 |
| 3 | **人物入画** | 把人放进世界（小人 / 小舟入画）| MS·人景同框 |

**铁律**：
- **比例尺：以小衬大**——放一个小人 / 小舟 / 飞鸟在画里，衬出仙山之巍峨（不放参照物就显小）。
- **地理锚点入库**：标志性地点建"**场景定妆**"（沿用角色卡/场景卡机制，大场面也要锚），**跨集复用同一张 establish**，别每集换山门。
- **三镜递进，别一镜糊一片**：一镜既要恢弘又要每个细节清楚 = AI 必糊；分"远全貌→中地标→近人物"三镜走。
- 运镜慢（缓推 / 缓摇 / 航拍下降），让观众"看清世界"。

### 6.3 后期
配**空灵 BGM + 环境音**（风 / 钟 / 鹤唳 / 诵经底噪），慢节奏，给世界"呼吸感"。establish 镜不抢戏、给足时长（2-4s），是"喘息 + 沉浸"位。

## 七、斗法对轰（法术对撞 / 僵持顶法 / 压制反压）

### 7.1 难点
两人放法术对轰 = 双主体 + 光束相撞 + **力量对比**，AI 易：光束乱飘、撞点不知在哪、僵持没张力、谁压谁看不懂、双人同框崩。
> 与打斗的区别：打斗（`打斗分镜.md`）是**近身命中**（五帧·命中接触）；斗法对轰是**远程能量对冲 + 僵持顶法**，多一个"撞点位置较劲"的力量对比段——故独立成节。

### 7.2 关键帧拆解（对轰五拍）
| # | 拍 | 画什么 | 镜 |
|---|---|---|---|
| 1 | **双方蓄法** | 各自掐诀/凝灵力、法术将出（正反打）| 甲镜 / 乙镜 |
| 2 | **对轰·相撞** ⭐ | 两道法术在中间撞上，撞点炸开能量团/顶住 | 撞点 CU/双人极远 |
| 3 | **僵持·顶法** | 撞点僵在中间、能量团较劲；**撞点偏谁谁弱** | 撞点 + 表情特写切 |
| 4 | **压制 / 反压** | 撞点推移——一方占上风，光墙推向另一方 | 撞点移动 + 占优方 |
| 5 | **破防·轰飞** | 撞点击穿，败方护体碎、被轰飞 | 败方 MS·快 → 留白 |

**铁律**：
- **撞点 = 这场的"命中帧"**（同打斗命中帧地位）：撞点独立出图，画能量对冲炸团/挤压、放射状速度线。
- **别一镜塞两人对轰**：用正反打（甲放招镜 → 乙放招镜 → 中间撞点镜），撞点镜可双人极远或只给能量团。
- **双方法术光束/能量进共享定妆库**（同 §〇 入库 + `打斗分镜.md §五`），**两色区分**（如金 vs 黑）全程一致。
- **僵持靠"撞点位置移动"叙事**（偏谁谁弱）+ 表情特写（咬牙 / 吐血 / 狞笑）快切，别让 AI 自由演较劲。

### 7.3 节奏 / 视频 / 后期
蓄法（憋）→ 相撞（炸）→ 僵持（持续紧，可叠对话/较劲台词）→ 压制（撞点推移、提速）→ 破防（顿 + 轰飞 + 留白）。
- 视频：撞点镜用**首尾帧**（相撞 → 撞点偏移）锁能量走向；放招人**锁姿态**（顶法姿态不变）只动光效。
- 后期：撞点持续嗡鸣轰响、光团呼吸明灭；压制时音量推高；破防 = 炸雷级 impact + 闪白 + 震屏 + BGM 抽真空再起。

### 7.4 迷你示例
```
Clip 顶法破防（僵持→破防）
相撞镜(双人极远·0.8s)：金掌劲 vs 黑魔焰在中央撞成光团（参考 定妆_金掌劲/定妆_黑魔焰）
僵持切(各0.4s)：甲咬牙稳住 / 乙狞笑加力——撞点缓偏向甲（甲渐弱）
反压镜(0.7s)：甲爆喝、撞点骤推回乙侧，光墙压过去
破防(0.6s·顿+爆)：乙护体黑光碎裂被轰飞 → 0.8s 留白缓拉
后期：撞点轰鸣持续 + 反压瞬间 BGM 上扬 + 破防炸响+闪白+震屏
```

## 八、神魂场面（神识探查 / 元神出窍 / 夺舍夺体 / 神魂攻击）

### 8.1 难点
神魂/神识是"**看不见的东西可视化**"，AI 易：元神形态**漂**（每镜不一样）、**"二我"穿帮**（肉身+元神同框不像同一人）、神识探查拍成**空镜没信息**、夺舍的"两魂相争"**看不懂**。

### 8.2 关键帧拆解（按子类型）
| 子类型 | 拆解 | 要点 |
|---|---|---|
| **神识探查** | 眉心光点 → 神识波纹扩散（俯视掠过山河/人群）→ 锁定（波纹回缩聚焦）| 主观 + 俯视给信息，少人物动作 |
| **元神出窍** | 肉身入定（盘坐闭目）→ 元神虚影自天灵升起（半透明、与肉身同貌）→ 元神行动 | **"二我"同框**：元神=肉身的半透明发光态 |
| **夺舍 / 夺体** | 入侵魂（黑雾/异色元神）扑向目标 → 两魂相争（体表光色拉锯）→ 夺舍成（睁眼变神色/瞳色变⭐）| 成败帧是爽点，单独出图 |
| **神魂攻击** | 神魂之力击中元神 → 魂体震荡/裂纹（命中帧同打斗）| 命中帧独立出图 |

**铁律**：
- **元神/神识形态进共享定妆库**（`定妆_主角元神.png` 半透明态、`定妆_神识波纹.png`），治"魂漂"。
- **"二我"同框**：元神**必须用肉身定妆做参考派生**（半透明发光态，强度 0.75-0.85，同形态变体规则），确保"是同一个人的魂"——否则肉身和魂变两个人。
- **神识探查靠"波纹 + 主观 + 俯视"给信息**，别拍成纯空镜（观众得"看见神识扫过哪里"）。
- **夺舍"两魂相争"靠体表光色拉锯 + 表情狰狞切**，别要求一镜画两个魂打架。
- 神魂攻击命中帧同打斗命中帧（独立出图 + 顿 + 音效）。

### 8.3 节奏 / 后期
神识=波纹扩散（缓）→聚焦（顿）；元神出窍=入定（静）→升起（缓、神圣感）→行动；夺舍=扑入（快）→相争（拉锯紧）→成败（顿 + 变色爽点）。
- 后期：神识=空灵嗡鸣 + 波纹音；元神出窍=缥缈 BGM + 升起气音；夺舍相争=低频压迫 + 两音色拉锯，夺舍成=一记定音 + 瞳色变特写；神魂攻击命中=闷响 + 魂体闪。
- 视频：元神升起用**首尾帧**（肉身 → 元神升起）；"二我"**锁两者姿态、只动半透明度/光**；夺舍扑入用首尾帧。

### 8.4 迷你示例
```
Clip 元神出窍探查（入定→升起）
首帧：神魂_肉身入定.png（盘坐闭目）
尾帧：神魂_元神升起.png（同貌半透明元神自天灵升起，参考 定妆_主角元神）
视频：肉身姿态不变；半透明元神缓自天灵浮起、透明度渐显；微光气流上升；镜头缓推
后期：缥缈 BGM 起 + 升起气音；元神成形一记轻鸣
```
