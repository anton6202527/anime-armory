# 分镜语法 / 镜头语言规范（阶段2·分镜设计必读）

> 目的：把 AI 漫剧从「能看」做到「专业」。**镜头衔接/运用/梳理的廉价感，80% 来自分镜随意**（随机切镜、没景别节奏、跳轴、缺转场逻辑）——不是单图不够漂亮。本规范是阶段2 设计 `分镜剧本.md` / `故事板.md` 时的硬约束。

---

## 一、景别系统（先有词汇，才有节奏）

| 景别 | 缩写 | 画什么 | 干什么 |
|---|---|---|---|
| 大远景 | ELS | 环境为主，人很小 | 交代时空/氛围（建制镜头）|
| 全景 | LS | 完整人物 + 环境 | 交代人物与空间关系 |
| 中景 | MS | 膝/腰以上 | 对话、动作主体 |
| 近景 | MCU | 胸以上 | 情绪、台词重点 |
| 特写 | CU | 脸/手/物 | 强情绪、关键道具、爽点 |
| 大特写 | ECU | 眼/局部 | 极致张力（反转/觉醒瞬间）|

## 二、景别节奏铁律（治「镜头梳理」）

1. **有进有出**：一个戏剧节拍 = 远→近 推进（establish→approach→detail）或 近→远 释放。**忌连续 3 镜同景别**（看着像 PPT 翻页）。
2. **节拍配景别**（速查）：
   | 戏剧功能 | 推荐景别/运镜 |
   |---|---|
   | 新场景开场 | **建制镜头**：ELS/LS 先交代空间 |
   | 环境/压迫氛围 | LS 固定或缓慢横移 |
   | 对话/对峙 | MS↔MCU 正反打 |
   | 情绪累积 | MCU **缓慢推近** |
   | 台词重点/内心 | CU |
   | 爽点/反转/觉醒 | **CU/ECU + 切 + 轻甩镜** |
   | 收尾钩子 | 留白后 CU 定格 |
3. **每集至少 1 个建制镜头**（开场或换场处），否则观众丢失空间感。

## 三、连贯性三规则（治「镜头衔接」）

1. **轴线（180°）**：对话/对峙双方连一条假想轴，**机位只在轴一侧**。跳轴 = 两人突然「换边」，观众瞬间晕。换轴必须用一个过轴镜头（推到正面/越肩）过渡。
2. **30° 规则**：同主体同景别的**相邻两镜，机位角度差 >30°**，否则像跳帧（jump cut）。要么换景别，要么换角度。
3. **视线 / 出入画方向一致**：A 看向画右 → 下一镜 B 应在画右、看画左（eyeline match）。人物出画走右 → 下一镜从左入画（方向连续）。**方向乱 = 空间逻辑崩**。

## 四、转场逻辑（治「镜头运用」）

- **动作匹配剪辑（match cut）**：上镜动作收尾 = 下镜动作起点（如「抬手」接「手落」）。这是最顺滑的硬切。
- **首尾帧连贯**：合成相邻 clip 时，**clip K 的尾帧 = clip K+1 的首帧**——用 `frames2video` 双帧引导（n2d-video）；阶段2 设计时就把相邻镜头的首尾构图对齐。
- **视线引导转场**：A 看向某处 → 切到 A 所看的内容（eyeline cut）。
- **声音先行（J-cut）**：下一句台词/系统音/环境声可提前 0.15-0.4s 进入上一 Clip 的尾部，让画面切换不突兀。只用于旁白、系统音、背身/侧脸说话或转场声；正面口型特写禁用。
- **情绪转场**：硬切给冲击（反转/爽点）；叠化/慢转给抒情、时间流逝、回忆。
- **空镜过渡**：换场之间插一个空镜（烛火、落叶、门）缓冲节奏，避免镜头硬怼。

## 五、运镜（克制原则）

- **一个 clip 一个主运动**：推 OR 拉 OR 跟 OR 环绕 OR 固定，别叠。
- **AI 友好排序**：固定 ≈ 缓慢推/拉 > 横移/跟 > 环绕 > 复杂甩/手持。**复杂运镜 AI 最易崩**——爽点可用轻微甩镜，其余从简。
- **运镜服务情绪**：推近=逼近/聚焦、拉远=释放/孤独、跟=代入、环绕=高光时刻、固定=克制/压迫。

## 六、构图（治「图片不准确」一半）

- **单人优先**：多人/复杂打斗 AI 最易乱脸乱手 → 能拆镜就拆（正反打代替双人同框混战）。**但打斗是核心爽点（仙侠武侠）时不是简单回避**，按 `打斗分镜.md` 五帧拆招 + 命中帧出图。**御剑飞行/追逐这类高速移动按 `仙侠场面分镜.md`「锁姿态、动背景」**——别让 AI 动人物（一动崩姿态崩脸），锁住姿态让背景/云层/速度线流动 + 镜头跟随；追逐另用正反打 + 轴线一致代替双人同框。
- **明确前后景**：写清"前景X / 后景Y"，避免 AI 把人物糊在一起。
- **三分法 + 留白**：主体不要永远死正中；竖版 9:16 给头顶/字幕区留空间。
- **景别要够**：要 CU 就写"极致面部特写"，别让 AI 出成中景（即梦常把特写画大）。

## 六·五、一致性高危镜（分镜阶段就规避，治崩脸）

2026 公认最易崩脸/角色断层的三类镜——**跨图出图和 image2video 都最难锁脸**，分镜设计能避就避，避不开就标记让出图/出视频加码：

- **极端角度**（大俯/大仰/纯背/极端透视）：定妆多是正/侧脸，极端角度无据 → 优先常规角度叙事；非用不可先在定妆组补对应角度（侧/背）。
- **人物在画面太小**（大远景里小小一个人）：脸像素不足、模型自由发挥 → 关键角色尽量给中近景能辨脸；大场面靠"远全貌→中地标→近人物入画"三镜把脸交给近镜（见 `仙侠场面分镜.md`）。
- **大暗部 / 极低照度**（纯黑/剪影/强逆光）：脸埋暗部 → 留一束补光照脸（眼神光/轮廓光），或有意"剪影不露脸"（则该镜本就不锁脸，写进契约）。

> 标记法：这类镜在 `故事板.md` 该 Clip 衔接/备注写「一致性高危·提参考强度+补X角度定妆/用后端 Character-ID」，下游 n2d-image/n2d-video 据此加码。详见 `n2d-image/references/角色一致性checklist.md §二` 同条。

## 七、阶段2 落地清单（每集自查）

- [ ] 开场有建制镜头？
- [ ] 没有连续 3 镜同景别？景别有进有出？
- [ ] 对话/对峙没跳轴？相邻同主体镜头角度差 >30°？
- [ ] 视线/出入画方向连续？
- [ ] 相邻 clip 首尾帧构图对齐（用 frames2video）？
- [ ] 每个 Clip 的入点/出点/转场方式已写进 `故事板.md` 的「衔接设计」？
- [ ] 需要声音先行(J-cut)的地方避开正面口型特写，且提前量控制在 0.15-0.4s？
- [ ] 换场/跳时空/姿态难接处已有空镜缓冲，而不是期待后期硬救？
- [ ] 爽点/反转用了 CU/ECU + 切？
- [ ] 运镜每镜单一且偏简单？
- [ ] 复杂打斗/多人镜头已拆成单人正反打？含打斗集另按 `打斗分镜.md` 自查？
- [ ] 含御剑飞行/追逐/渡劫/炼丹炼器/大阵/大场面 establish 的集，已按 `仙侠场面分镜.md` 自查（飞行追逐锁姿态动背景 / 渡劫炼丹法阵爆发帧出图+元素入库 / 大场面三镜由远及近+比例尺）？

> 把以上写进 `故事板.md` 时，每个 Clip 块标注：**景别 + 运镜 + 与前后镜的衔接方式（match cut / 首尾帧 / eyeline / 空镜）**。
