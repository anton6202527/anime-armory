# 打斗分镜 / 武打·法术场面工艺（仙侠武侠必读）

> `分镜语法.md` 管"镜头怎么切"，`导演节奏.md` 管"整集节奏"，**本文件管"打斗这一种戏怎么拆才能又顺又燃"**。
> 仙侠武侠的打斗 = 核心爽点，**绕不过去**。AI 直接生"两人对打"必崩（动作跳、软绵绵、乱脸乱手、剑气乱飘）。本文件把打斗工程化。
> 阶段2 设计含打斗的集时，`分镜剧本.md`/`故事板.md` 按本文件拆；下游 n2d-image/n2d-video/n2d-compose/n2d-review 各取所需。

---

## 〇、核心方法论（一句话）

> **把一招拆成帧 → 用图把关键帧锁死 → 用首尾帧让视频只做"帧间插值"（不让 AI 自由发挥动作）→ 用剪辑+音效在后期补"打击感"。**

这是 n2d「先出图锁死、图生视频只控运动」铁律在打斗上的延伸。打斗崩，根因是"让 AI 自己编动作"；解法是"动作由关键帧图决定，视频只负责把两帧之间动起来"。

---

## 一、AI 打斗五大死因 → 对应解法（速查）

| 死因 | 现象 | 解法（落到哪一步）|
|---|---|---|
| 让 AI 编动作 | 动作跳、不连贯 | **关键帧出图锁死 + 首尾帧引导视频**（image+video）|
| 匀速插值 | 软绵绵、没力 | **变速 + 命中顿帧 + 打击音效**（compose）|
| 多人同框 | 乱脸乱手 | **拆成"一打一"序列 + 单人正反打**（script）|
| 特效/武器漂 | 剑气/法宝每镜不一样 | **法术·武器·光效进共享定妆库**（image）|
| 没"力的交代" | 看不懂谁打谁 | **五帧拆招 + 力的方向标注**（script）|

---

## 二、五帧拆招（打斗分镜的最小单元）

**一次有效攻击 = 5 个戏剧帧**。不必每招都 5 帧全给，但"命中帧"绝不能省——它是打击感的来源。

| # | 帧 | 画什么 | 景别/节奏 | 镜 |
|---|---|---|---|---|
| 1 | **起手 / 蓄力** | 拧腰、沉肩、灵力汇聚、剑出鞘半寸 | MS/MCU·**长镜慢**(蓄力感) | 攻方 |
| 2 | **发力 / 出招** | 拳/剑/法术离手，速度线、拖影 motion blur | MS·**碎切快**(运镜跟出招) | 攻方 |
| 3 | **命中 / 接触** ⭐ | 接触瞬间：冲击波/火花/凹陷/震开 | **CU/ECU·硬切 + 顿帧** | 接触点 |
| 4 | **受击 / 反应** | 被打飞、吐血、格挡、护体光破 | MS/CU·快 | 守方 |
| 5 | **收势 / 反击** | 站定喘息 / 顺势反打 / 留白定格 | MS·**留白**或接下一招起手 | 守方→攻方 |

**铁律**：
- **命中帧（#3）必出独立图**，且是慢镜/顿帧落点——80% 打击感在这一帧。
- 攻防来回 = **正反打**：攻方出招镜 → 守方受击镜，别强求一个镜头里塞两人对打。
- 连招 = 多组五帧串联，但**每组留一个命中帧**当节奏锚。

## 三、打斗节奏曲线（`导演节奏.md` 的打斗专章）

打斗不是从头快到尾——**快慢对比才燃**。一个打斗段的张力曲线：

```
张力
 ▲              💥命中(顿)
 │        ⚡连招  ╱│╲
 │       ╱碎切╲ ╱ │ ╲
 │  蓄力╱     ╲╱  │  ╲收势
 │ ───╱(长慢) 快  │   ╲留白喘息
 └──────────────────────────▶
   起手    出招   命中  收势
   长镜    碎切   顿帧  留白
```

| 段 | 镜头时长 | 运镜 | 声音 |
|---|---|---|---|
| **蓄力** | 长(2-3s) | 固定/极缓推，憋住 | 环境音收、低频蓄 |
| **连招** | 碎切(0.5-1s) | 跟拍/快推，跟着招走 | 兵器破空 whoosh 连击 |
| **命中** | **顿(0.3-0.5s 定格)** | 硬切 CU/ECU | **impact 重音 + 闪白/震屏** |
| **收势** | 留白(1-2s) | 缓拉/固定定格 | BGM 抽真空 0.5s 再起、喘息 |

> "蓄得越足、放得越狠"——别一上来连招到底，那等于没节奏。每个命中帧后给半拍喘息，下一波连招才有力。

## 四、多人混战拆解（治"乱脸乱手"）

仙侠常有"一人战群妖"。**绝不一个镜头塞混战**，拆成：

1. **1 个广角 establish**（ELS/LS）：交代"一对多"的局面（人小、群妖环绕），不要求脸清楚。
2. **N 组单人特写序列**：主角一招一个妖（每组走五帧/简化三帧），正反打。
3. **节奏靠剪辑串**：establish → 连续"主角出招 CU + 某妖受击 CU"快切 → 收势广角。
- 群妖用"同一个妖定妆 + 不同角度/位置"复用，别每只都精修脸。

---

## 五、特效 / 武器 / 法术一致性（治"剑气乱飘"）

法宝、剑气、掌风、灵力光效 = 跟角色脸一样会漂，**必须进共享定妆库**（`出图/common/`）锁形态：

- 建 `定妆_<法宝/招式>.png`（如 `定妆_青色剑气.png`、`定妆_王敦拳劲冲击波.png`），定**颜色/形状/质感/拖尾**。
- 每个含特效的镜头图，**多图参考派生**：角色定妆 + 特效定妆 + 上一关键帧，确保剑气全程一个样。
- prompt 写明轨迹方向（`剑气自右下向左上飞出，青色，拖光尾`），视频阶段才能让它"飞得对"。
- 同一场打斗**不中途换生图工具**（一致性税最贵）。

## 六、故事板字段扩展（含打斗的 Clip）

`故事板.md` 里打斗 Clip 在常规字段外，**额外标**：

```
- 招式名：<如"噬妖·裂掌">
- 帧位：起手 / 发力 / 命中 / 受击 / 收势   ← 五帧里的哪一帧
- 力的方向：<攻方→守方，左下→右上>
- 命中帧：是/否（是→标记慢镜+顿帧+impact音效落点给 compose）
- 特效轨迹：<青色剑气自右下飞出，命中炸环形冲击波>
- 关键帧图需求：起手帧PNG + 命中帧PNG（供 video 首尾帧）
```

## 八、打斗自查清单（阶段2 设计含打斗集时）

- [ ] 每次有效攻击都有**命中帧**（独立图 + 慢镜/顿帧落点）？
- [ ] 攻防是**正反打**（出招镜/受击镜分开），不是一镜塞两人对打？
- [ ] 节奏有曲线：蓄力长镜 → 连招碎切 → 命中顿 → 收势留白？不是匀速从头打到尾？
- [ ] 多人混战拆成"广角 establish + 单人序列"？没要求一镜多人脸清楚？
- [ ] 法宝/剑气/光效进了共享定妆库、跨镜一致？
- [ ] 每个打斗 Clip 标了招式/帧位/力向/命中帧/特效轨迹？
- [ ] 命中帧给 compose 标了 impact 音效 + 闪白/震屏落点？
- [ ] 复杂招式已拆成"起手 clip + 命中 clip"两短段（AI 友好）？

## 九、完整示例：王敦一招轰飞化神妖兽（script → 图 → 视频 → 合成）

> 取材《看花胖子，藏到了飞升》第52章联手抗化神。一招"噬妖·裂掌"拆 3 帧（起手→命中→受击），演示全链。

### 9.1 阶段2·故事板（n2d-script）

```markdown
## Clip 14（时长 1.8s · 镜头31+32+33）　节奏：蓄力·长镜→命中·CU硬切　张力：爆发
招式名：噬妖·裂掌　力的方向：王敦(画左)→妖兽腹(画右)
特效轨迹：王敦右掌汇暗金灵力→拍出→命中炸环形金色冲击波
命中帧：是（→compose：命中顿帧4帧 + impact重音 + 轻闪白 + 微震屏）

- 镜头31 起手帧 MCU·缓推(0.6s)：王敦沉身拧腰，右掌下沉，掌心暗金灵力汇聚成纹（参考 定妆_王敦灵力暗金.png）
- 镜头32 命中帧 CU·硬切(0.5s·顿)：掌面轰在妖兽腹部，接触点炸开环形金色冲击波，速度线放射，motion blur
- 镜头33 受击帧 MS·快(0.7s)：化神妖兽腹部凹陷、口吐黑血，向画右飞出，护体青光碎裂
```

### 9.2 出图（n2d-image）—— 出 2 张关键帧图 + 1 张受击图

```
# 共享定妆库先备：出图/common/定妆_王敦灵力暗金.png（暗金、缠纹、掌心汇聚态）

# 镜头31 起手帧 → 出图/第52集/镜头31_起手.png
MCU，王敦（<王敦角色定妆锚点句>）沉身拧腰半蹲，右臂下沉、掌心向下，
掌心暗金色灵力汇聚成游动纹路(参考 定妆_王敦灵力暗金.png)，蓄力将发未发，
表情冷厉，衣袂因气劲微扬；背景化神妖兽虚影逼近。动态定格 dynamic pose。9:16。

# 镜头32 命中帧 → 出图/第52集/镜头32_命中.png   ★打击感核心帧
CU，王敦右掌掌面轰在妖兽灰黑腹部，接触瞬间——
炸开环形金色冲击波(参考 定妆_王敦灵力暗金.png)，火花四溅、速度线向四周放射，
掌缘 motion blur 拖影；妖兽腹部皮肉凹陷。极致冲击感，动态定格。9:16。

# 镜头33 受击帧 → 出图/第52集/镜头33_受击.png
MS，化神妖兽（<妖兽定妆锚点>）腹部凹陷、仰头喷出黑血，身体向画右飞出、四肢失衡，
体表青色护体光膜炸裂成碎片；前景王敦收掌残影。动态定格。9:16。
```
> 筛图：命中帧重点看"接触点对不对位 + 冲击波形态一致 + 有力"，4 选 1。

### 9.3 出视频（n2d-video）—— 首尾帧锁动作 + 力链 + 慢镜

```markdown
## Clip 14（命中类·用首尾帧）
首帧：出图/第52集/镜头31_起手.png
尾帧：出图/第52集/镜头32_命中.png      ← 起手→命中，AI 只插值，不自由编动作

### 视频 prompt（即梦 image2video）
人物运动：王敦拧腰送肩、右掌由下沉骤然拍出（力链：腰→肩→肘→掌）；
         掌心暗金灵力随出掌成形，命中妖兽腹部时炸开环形冲击波；
镜头运动：跟随出掌方向快推 0.3s，命中瞬间转极慢（子弹时间感）；
动态细节：灵力纹流动、速度线放射、衣袂气劲鼓动；
（即梦风格锚定句）

### 降级：若一镜推不动"起手→命中→受击"全程，
拆 Clip14a(起手帧→命中帧) + Clip14b(命中帧→受击帧·镜头33为尾帧) 两段各 ~0.8s。
```

### 9.4 合成（n2d-compose）—— 后期补打击感

```
- Clip14 命中帧（镜头32 那帧）：hit-stop 定格 4 帧（≈0.13s）
- 变速：起手段 0.9x（略慢蓄力）→ 出招 1.1x（快）→ 命中瞬间 0.5x（慢镜）
- 音效：出掌 whoosh（命中前 0.1s）+ 命中帧 impact 重低音（对齐顿帧那一帧）
- 视效：命中帧叠 1 帧轻闪白 + 2 帧微震屏（幅度小，别晃晕）
- BGM：把鼓点 drop 用 BGM_OFFSET 平移，落在命中帧时间戳上
- 留白：镜头33 受击飞出后给 0.8s 缓拉远 + BGM 瞬时压低再起（喘息）
```

### 9.5 这一招为什么"燃"（复盘）
- **动作不跳**：31→32 用首尾帧，AI 只补中间，掌的轨迹连贯。
- **有打击感**：命中帧独立出图 + 顿帧 + impact重音 + 闪白震屏，四重叠加。
- **有节奏**：蓄力略慢 → 出招快 → 命中慢镜顿 → 收势留白，快慢咬合。
- **特效一致**：暗金灵力/冲击波全程引用同一张定妆，不漂。

> 一句话：**一招拆三帧、命中帧单独出图当首尾帧锚、后期顿帧+音效补力——这就是 AI 把仙侠打斗做"顺且燃"的工程链。**
