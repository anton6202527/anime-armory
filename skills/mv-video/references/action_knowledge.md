# MV 动作知识库

MV 的动作设计不是“越炫越好”，而是让动作峰值、镜头运动和视觉特效一起踩在音乐上。这个库给 mv-plan / mv-video / mv-score / mv-review 共享动作词汇，避免每次临场写“快速旋转、炫酷光效”这种空泛 prompt。

## 使用原则

- **音乐优先**：动作峰值对齐 beat/downbeat，副歌可以短促爆发，主歌要留呼吸。
- **一 clip 一个主动作**：短 clip 只能承载一个动作链；不要同时跳舞、打斗、转身、爆炸、换景。
- **图像先锁姿态**：复杂动作的关键姿态要在首帧或尾帧中出现；视频只负责运动过渡。
- **复杂接触要降级**：拥抱、拉扯、多人打斗、近身接触容易崩，MV 里优先拆成手部/眼神/道具/光效切。
- **炫技服务情绪**：动作要解释歌词或情绪，不要跟歌曲无关。

## 段落到动作强度

| 段落 | 动作强度 | 动作取向 |
|---|---:|---|
| intro | 1-2 | 入场、背影、环境唤醒、慢推 |
| verse | 2-3 | 走、回头、抬手、轻舞步、道具特写 |
| pre-chorus | 4 | 蓄力、逼近、抬头、握拳、光效预热 |
| chorus / drop | 5 | 爆发舞步、甩镜、环绕、跃起、光效冲击 |
| bridge | 3-5 | 反转、慢动作、空间变形、镜像破碎 |
| outro | 1-2 | 收束、定格、拉远、余光消散 |

## 动作家族

| 家族 | 适合段落 | 首帧准备 | 视频 prompt 关键词 | 风险 / 回退 |
|---|---|---|---|---|
| performance_pose | intro/verse | 正脸/半身/麦克风/站姿 | 轻微呼吸、抬眼、慢推、发丝摆动 | 太静可加前景光斑 |
| expressive_walk | verse/pre | 侧身或背影，留运动方向 | 跟拍、衣摆、脚步、回头 | 人脸小，结尾给手部/背影 |
| dance_hit | chorus/drop | 全身姿态清楚，四肢不要交叉 | hit、body wave、甩手、踩点定格 | 手脚易糊，短 clip 一动作 |
| camera_whip | chorus/drop | 构图留边 | whip pan、轻甩、动势模糊、闪白切 | 过度会晕，别每条都用 |
| orbit_reveal | pre/chorus | 主体居中，背景有层次 | 环绕、光线扫过、服装边缘高光 | 脸漂时改半身慢环绕 |
| prop_sync | verse/chorus | 道具清楚：剑、伞、镜、花、麦克风 | 道具举起/落下/旋转，峰值踩 beat | 道具变形时切特写 |
| vfx_burst | chorus/drop | 爆发前一瞬 | 光环、粒子、火花、花瓣、剑气、霓虹脉冲 | 特效遮脸时降低强度 |
| environment_motion | intro/outro/bridge | 空镜或小人物 | 雨、雾、云、灯牌、海浪、城市流光 | 适合补缝和留白 |
| mirror_split | bridge/drop | 镜面/玻璃/水面 | 镜像分裂、碎裂、反射错位、慢动作 | 不要生成多个不一致主角脸 |
| silhouette_action | pre/chorus | 逆光剪影 | 剪影转身、跃起、挥臂、强背光 | 脸不清时可规避崩脸 |

## 炫酷转场母题

| 转场 | 用法 | 注意 |
|---|---|---|
| 闪白 / 光效切 | downbeat 爆点、换景 | 不要每刀都闪，容易廉价 |
| 遮挡擦镜 | 衣袖、头发、伞、前景花枝掠过 | 接不上时最稳 |
| whip pan | 副歌快切、镜头甩到下一场景 | 方向要一致 |
| match action | 上一镜抬手，下一镜同方向落手 | 需要 start/end state 对齐 |
| match color | 红光爆发切红伞/红霓虹 | 适合跨景保持统一 |
| particle bridge | 花瓣/雪/火星/水滴飞过覆盖切点 | 粒子颜色跟 palette_anchor 一致 |
| mirror fracture | bridge 反转、心理撕裂 | 镜中脸要弱化，不要两个正脸抢主体 |
| shadow cut | 剪影动作切到实体动作 | 避免复杂脸部过渡 |

## 动作 prompt 模板

```text
动作家族：<action_family>
动作链：<一个主动作，最多两步>
动作峰值：<beat/downbeat 秒点或本 clip 结束前 0.2s>
镜头运动：<服务段落强度的推/拉/环绕/甩/跟>
动态细节：<衣摆/发丝/光斑/粒子/雨雾随节拍变化>
结尾落点：<end_state，给下一刀承接>
禁止：换脸、换衣、多人错脸、复杂肢体接触、生成文字/logo、生成原生人声
```

## 常见错误

| 错误 | 纠正 |
|---|---|
| 每个 clip 都写“炫酷运镜” | 具体化为动作家族 + 动作峰值 + 镜头运动 |
| 副歌每条都大环绕 | 用 dance_hit / vfx_burst / camera_whip 轮换，避免同质 |
| 主歌动作太满 | 主歌靠情绪推进，动作强度 2-3 即可 |
| 复杂多人动作直接图生视频 | 拆成单人特写、道具、剪影、反应镜 |
| 动作峰值不写时间 | 明确对齐哪一个 beat/downbeat，mv-compose 才能剪得爽 |
