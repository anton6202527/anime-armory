# 导演节奏 / 留存工程（全阶段共用·留存圣经）

> **核心信念**：红果/抖音爆款的画质很多不是行业顶级，但**镜头 + 剧情节奏**让观众"划不走、一直追"。
> 决定留存的不是单图多漂亮，而是**每一秒观众有没有理由不划走**。
> 本文件是 `分镜语法.md` 的姊妹篇——`分镜语法` 管"镜头怎么切（空间）"，本文件管"节奏怎么走（时间 + 留存）"。
> n2d-script（写词/分镜）、n2d-voice（念白节奏）、n2d-video（运镜张力）、n2d-compose（剪辑卡点）四个阶段都引用它。

---

## 一、留存曲线（一集的生命线）

一集 90~120 秒，观众的"划走冲动"在这几个点最高：**第 1-3 秒、每段戏的过渡处、中段平淡处、集尾**。导演的活就是在这些点喂"钩子/增量"。

```
张力
 ▲          💥爽点
 │   ⚡         ╱╲          🪝集尾钩子
 │  ╱ ╲   ●   ╱  ╲    ●   ╱
 │ ╱   ╲ ╱ ╲ ╱    ╲  ╱ ╲ ╱
 │╱     ●   ●      ╲╱   ●
 └────────────────────────────▶ 时间
  0-3s  前15s   中段      集尾
  冷开场 立钩子  情绪爬升   悬念断点
```

四个必抓的节点：

| 位置 | 任务 | 做法 |
|---|---|---|
| **0-3s 黄金开场** | 让人停下划动的手 | 冷开场（直接抛冲突画面/最炸的一句台词）或倒叙钩（先放后面的高潮一瞬再倒回）。**禁止**用 logo/慢摇空镜/长旁白开场 |
| **前 15 秒立钩子** | 让人决定"看下去" | 抛出核心悬念/欲望（她要复仇？他藏着什么？）+ 第一个明确矛盾 |
| **中段情绪爬升** | 不让中段塌方 | 每 15-20 秒一个"信息增量"或"微钩子"，张力阶梯式上抬，**至少 1 次反转** |
| **集尾 cliffhanger** | 逼人点下一集 | 在最吊胃口处**硬断**——别把这段戏说完 |

## 二、钩子密度（治"中段划走"）

**铁律：每 15-20 秒必须有一个"钩子"或"信息增量"**，否则观众这一段就走了。钩子不必是大反转，五种都算：

| 类型 | 作用 | 例 |
|---|---|---|
| **悬念钩** | 抛未解的问 | "这具身体的原主，到底是怎么死的？" |
| **欲望钩** | 立一个想看的目标 | "她发誓要让冷宫翻过来。" |
| **反差钩** | 打破预期 | 柔弱宫女抬头，眼里是杀意 |
| **信息增量** | 抖出新事实 | 系统提示：检测到妖族血脉 |
| **危机钩** | 倒计时/威胁逼近 | 脚步声越来越近，门要开了 |

**爽点 ≠ 钩子**：爽点是"出口气"（打脸、逆袭、揭穿），钩子是"勾着往下"。一集结构 = 钩子铺垫 → 张力累积 → **爽点引爆** → 新钩子收尾。爽点要有"憋—放"距离：憋得越足，放得越爽；上来就爽 = 不爽。

## 三、集尾 cliffhanger 三种打法（治"不追下一集"）

| 打法 | 怎么断 | 适用 |
|---|---|---|
| **危机悬置** | 在最危险的一瞬切走（刀架脖子上、门被推开） | 动作/对峙集 |
| **真相半露** | 抖出一半真相，关键那句不说 | 悬疑/宅斗集 |
| **反转预告** | 给一个"事情要变天"的信号（一个眼神、一封信、一道系统音） | 任何集 |

**绝不**在集尾把这段戏"讲完整、收干净"——那等于告诉观众"可以走了"。

## 四、镜头时长曲线（剪辑节奏，治"节奏平/像 PPT"）

节奏 = **镜头时长的变化**。等长堆叠 = 催眠；有快慢 = 有呼吸。

```
镜头时长
 ▲
长│ ▓▓▓                    ▓▓▓▓  ← 铺垫/留白：长镜，让情绪沉淀
 │ ▓▓▓  ▓▓        ▓▓▓▓
中│ ▓▓▓  ▓▓  ▓▓   ▓▓▓▓
短│ ▓▓▓  ▓▓  ▓  ▓ ▓▓▓▓   ▓ ▓ ▓  ← 爽点/对峙：碎切加速，心跳感
 └──────────────────────────────▶
   开场  对话    冲突  爽点  收尾
```

三条规则：
1. **铺垫用长镜**（3-5s/镜），给情绪沉淀和空间感；**爽点/打斗/吵架用碎切**（1-2s/镜，甚至更短），切得越快越紧张。
2. **加速到爽点**：临近爆发，镜头逐个变短，剪出"心跳加速"。爆发瞬间用 **CU/ECU + 硬切**（配合 `分镜语法` 的景别）。
3. **爽点后留白**：引爆完给一个**长留白/定格**（1-2s 不剪），让观众"喘口气、回味"——再进下一段。没有留白 = 一直紧绷 = 疲劳划走。

> 镜头时长在 **n2d-script 阶段2** 落进 `故事板.md`（每 Clip 标"加速/留白/卡点"），由配音时长打底、导演按曲线微调；**n2d-compose 必须保留这条曲线，不许把 clip 等长化**。
> **打斗有专属节奏曲线**（蓄力慢长→连招碎切→命中顿帧→收势留白），详见 `n2d-script/references/打斗分镜.md`。
> **其他仙侠奇观也各有专属节奏曲线**：御剑飞行（远景交代→跟拍巡航→机动爽点→抵达留白）、追逐（紧→险情顿→更紧→放）、渡劫（层层加码、末劫最长顿、突破后留白）、炼丹（过程蒙太奇压缩→开炉前憋一拍→爆发）、大阵（布阵慢→逐符加速→激活爆发）、大场面（三镜由远及近、慢运镜给足时长）、斗法对轰（蓄法憋→相撞炸→僵持紧→压制提速→破防顿+留白）、神魂（元神出窍：入定静→升起缓神圣→行动；夺舍：扑入快→相争拉锯→成败顿）——详见 `n2d-script/references/仙侠场面分镜.md`。

## 五、卡点（BGM/音效 × 画面）

爽点的冲击力 = 画面 + 声音**同一瞬间**砸下来。

- **爽点对齐 BGM drop**：把 BGM 的鼓点/drop/弦乐炸点，卡在爽点画面那一帧（n2d-compose 用 `BGM_OFFSET` 平移 BGM，让 drop 落在 `故事板` 标的爽点时间戳上）。
- **反转配"重音"**：揭面/觉醒瞬间配一个重低音/铃/心跳音效（bgm.txt 的"关键音效点"）。
- **留白配"抽真空"**：爆发后那个留白镜，BGM 可瞬时拉低/静默 0.5s，再起——"留白"听感最抓人。

## 六、念白节奏（配音表演，治"平读 = 出戏"）

台词写得好，平读也废。**情绪/语速/停顿**是留存的一部分：

- **快**：吵架、逼问、危机、爽点反击 → 语速上抬、句间几乎不留气口。
- **慢**：内心独白、悲伤、阴狠盘算、集尾留悬念 → 放慢、字间留停顿。
- **停顿（气口）**：关键反转词前**留一拍**（"原来……杀她的人，是你"），停顿制造紧张。
- **重音**：爽点/打脸句的"刀子词"重读。

> 这些在 **n2d-script** 写进 voiceover.txt 的情绪标注（`[镜头·角色·情绪]`），由 **n2d-voice** 映射成 TTS 的 emotion/语速/句间间隔——情绪标注不是注释，是要驱动念白的。

## 七、导演节奏落地清单（每集自查·配合 `分镜语法` 的镜头自查）

- [ ] 0-3s 是冷开场/倒叙钩，不是 logo/慢空镜/长旁白？
- [ ] 前 15 秒立住了核心悬念/欲望 + 第一个矛盾？
- [ ] 中段每 15-20 秒有一个钩子/信息增量？无超过 20s 的平淡段？
- [ ] 全集 ≥1 次反转？爽点有"憋—放"距离（不是上来就爽）？
- [ ] 集尾是 cliffhanger（危机悬置/真相半露/反转预告），没把戏讲完？
- [ ] 镜头时长有曲线：铺垫长镜 + 爽点碎切加速 + 爽点后留白？
- [ ] 爽点画面对齐了 BGM drop / 重音效（卡点）？
- [ ] 念白有快慢/停顿/重音设计，不是从头平读？

> 一句话：**`分镜语法` 让每个镜头"专业"，`导演节奏` 让整集"追得停不下来"。两者都过自查清单，才算导演级。**
