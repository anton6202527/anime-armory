# 导演视角 Prompt 八维 / 出图·出视频 prompt 装配规范（全阶段共用）

> **核心信念**：同一张定妆、同一个场景，**画师**问"画面里有什么、美不美"；**导演**问"这个镜头**为什么这么拍**、它在替剧情说什么"。
> 漫剧的廉价感，一半来自"每张图都是平视正面、打光均匀、表情到位的好看插画"——好看，但**不会叙事**。把 prompt 从画师视角升级成导演视角，就是每条 prompt 都按下面**八维**想全、按序装配。
> 本文件是 `分镜语法.md`(空间/镜头怎么切) 与 `导演节奏.md`(时间/留存) 的**落地层**：那两份决定"切什么镜、走什么节奏"，本文件决定"**这一镜的 prompt 怎么写**"。
> n2d-script(分镜剧本/素材清单)、n2d-image(出图 prompt)、n2d-video(视频 prompt) 三处都按本框架装配。

---

## 一、八维：画师视角 → 导演视角

| 维度 | 画师视角（只回答这个就停）| 导演视角（还必须回答）| prompt 里的体现 |
|---|---|---|---|
| **① 镜头** | 画全身 / 画特写 | **为什么是这个景别**（ELS/LS/MS/MCU/CU/ECU）+ 焦段/景深（广角=张力夸张·长焦=压缩偷窥·浅景深=主体抽离背景）| `中景，长焦浅景深，背景虚化` |
| **② 机位** | 正面、平视 | **角度+高度+关系**：平/俯/仰视、机位高度、过肩/正反打/视点镜；**机位即态度**——仰拍=压迫/威严，俯拍=渺小/被碾压，过肩=对峙代入 | `低角度仰拍，过肩视角，镜头略带俯冲` |
| **③ 人物** | 长得好看、穿得对 | **身份锁定**（拼角色卡『锚点句』锁脸锁妆造）+ **此刻状态**（伤痕/泪痕/发丝凌乱/衣衫破损/血污——人物随剧情"被改写"）| `同一少女[锚点句]，左颊有新伤，发髻松散` |
| **④ 动作** | 摆个静态 pose | 静帧=抓**动作最强的一瞬**（挥剑到顶点前、转身回眸瞬间）；视频=**人物运动 + 表情变化**（performance，见 §三）| 图：`剑尖抵喉的定格瞬间`；视频：`缓缓抬眼→眼神由茫然转杀意` |
| **⑤ 场景** | 背景画对 | **场景定妆锚**（跨集复用同一张）+ **时间**（昼/夜/黄昏）+ **环境状态**（雨/雾/烛火明灭/陈设凌乱）——交代时空、烘托情绪 | `冷宫内[场景锚]，深夜，烛火将熄，地面积尘` |
| **⑥ 光影** | 打亮、看得清 | **光替剧情说话**：光源动机（烛光/月光/天窗/逆光剪影）+ 方向（顶/侧/逆/伦勃朗三角光）+ 质感（硬光锐利/柔光暧昧）+ 调性（高调明快/低调压抑）| `单侧烛光，伦勃朗布光，低调，面部一半没入暗部` |
| **⑦ 情绪** | 表情到位 | **本镜情绪词**（茫然/阴狠/决绝…）+ **张力词**（克制/紧张/爆发/释放）→ 驱动运镜幅度+念白+剪辑节奏 + **色调情绪**（冷青=疏离/暖橙=温情/血红=危机）| `情绪=隐忍的恨；张力=克制；冷青色调` |
| **⑧ 画质** | 高清、精致 | **画风词统一**（依 `global_style.md`，禁漂移）+ 渲染级（电影级质感/胶片颗粒/景深）+ **画幅** + **跨AI锚定句**（图AI≠视频AI 时拼）+ **负面词堵漏** | `电影级渲染，9:16，[视频AI锚定句]；负面：低幼Q版、多手指…` |

> **顺序不是死的，但维度不能漏**。写每条 prompt 时心里过一遍这八维——尤其最易漏的**②机位 / ⑥光影 / ⑦张力**，它们正是"画师插画"和"导演镜头"的分水岭。

## 二、八维装配模板（图 prompt 通用骨架）

把八维按"先定框架（镜头/机位）→ 填主体（人物/动作/场景）→ 加灵魂（光影/情绪）→ 锁规格（画质）"的顺序写成一句连贯 prompt：

```
[①镜头] [②机位]，[③人物=锚点句+状态] 正在 [④动作]，
置身 [⑤场景+时间+环境状态]，[⑥光影]，
整体 [⑦情绪+张力+色调]，[⑧画风词+渲染级+画幅+跨AI锚定句]
负面：[统一负面词 + 本镜堵漏词]
```

**实例（画师视角 vs 导演视角，同一镜）**：

- ❌ 画师：`一个穿红衣的古装少女站在宫殿里，很漂亮，高清`
- ✅ 导演：`中景偏近、长焦浅景深；低角度仰拍过肩；同一少女[锚点句：丹凤眼/眉心朱砂/月白滚红边宫装]，鬓发微乱、指节攥白，正缓缓抬眸冷笑；冷宫正殿[场景锚]，深夜，残烛明灭、梁上积尘；单侧烛光做逆光勾边、伦勃朗三角光、低调；情绪=隐忍翻涌的杀意、张力=克制、冷青压暗红色调；电影级渲染、胶片微颗粒、9:16、[即梦风格锚定句]。负面：低幼Q版、画风漂移、多指、塑料感`

## 三、视频 prompt：八维 + 让它"动起来"

视频 prompt 在八维之上，把 **④动作** 与 **②机位** 升级为"运动版"，其余维度由首帧 PNG 已经锁定（出图阶段做完），视频阶段**只调运动**：

- **④动作 → 人物运动 + 表情变化**：写清"从什么状态→到什么状态"（抬眼/转身/挥袖/落泪），别只写静态。
- **②机位 → 运镜**：推/拉/摇/移/跟/环绕/固定 + 速度词。**运镜服务⑦情绪/张力**（见 `导演节奏.md §四`）：逼近聚焦=推近、释放孤独=拉远、高光爽点=环绕/轻甩、压迫=固定。
- **动态细节**：烛火摇曳/晨雾流动/衣袂飘动/妖气扩散——给画面"活气"。
- **⑦张力词**仍每条带一个（克制/紧张/爆发/释放），锚定本镜运动幅度与剪辑节奏。

```
人物运动：{③人物} {④动作的运动轨迹} → {表情变化}；
镜头运动：{②机位的运镜 + 速度词，服务⑦张力}；
动态细节：{⑤场景里的环境动态 ≥1 条}；
张力=｛克制/紧张/爆发/释放｝；（末尾按平台追加风格词/锚定句）
```

> 视频阶段**不重定**①镜头/③人物外貌/⑤场景/⑥光影/⑧画风——那些在出图阶段已用首帧锁死（"先在图阶段把视觉变量锁死，视频只调动作/运镜"）。视频里改了这些 = 与首帧打架 = 闪烁漂移。

## 四、各阶段取哪几维

| 阶段 | 文件 | 用到的维度 |
|---|---|---|
| **n2d-script 分镜剧本** | `分镜剧本.md` | 先定 ①②④⑤⑥⑦（导演意图层，文字描述）——每镜标景别+机位+光影动机+情绪张力 |
| **n2d-image 定妆** | `角色\|场景定妆.md` | ③人物 / ⑤场景 / ⑥光影(中性定妆光) / ⑧画质——**不锁**①②④⑦（定妆是中性参考，不带戏） |
| **n2d-image 分镜图** | `01_分镜出图.md` | **八维全上**（§二模板），首帧把视觉全锁死 |
| **n2d-video 视频** | `01_clips.md` | ④动作+②运镜+⑦张力+动态细节（§三），其余维度继承首帧 |

> 注意分工：**定妆图是"中性档案"**（正面、均匀光、无戏，只为锁脸锁造型），**分镜图才上全八维**（带机位/光影/情绪的"剧照"）。别把定妆图也打成戏剧光——那样它当参考图时会污染下游。

## 五、导演视角自查清单（每条分镜/视频 prompt 落笔前过一遍）

- [ ] **①镜头**：景别明确且**有理由**（不是默认中景）？需要时写了焦段/景深？
- [ ] **②机位**：写了角度/高度（不是默认正面平视）？机位选择**服务情绪**（仰=压迫/俯=渺小/过肩=对峙）？守轴线？
- [ ] **③人物**：拼了角色卡锚点句？写了"此刻状态"（伤痕/凌乱/服装变化），而非永远定妆态？
- [ ] **④动作**：图=抓了动作最强一瞬？视频=写了运动轨迹+表情变化（不是静态）？
- [ ] **⑤场景**：场景锚 + 时间 + 环境状态都在？
- [ ] **⑥光影**：写了光源动机+方向+调性？**光在叙事**（不是均匀打亮）？
- [ ] **⑦情绪**：本镜情绪词 + 张力词 + 色调情绪三件齐？张力词与运镜/念白一致？
- [ ] **⑧画质**：画风词依 global_style？画幅对？跨AI时拼了锚定句？负面词堵了本镜的漏？
- [ ] **最易漏三维复检**：②机位、⑥光影、⑦张力——这三维有没有偷懒回到画师视角（正面/均匀光/无张力）？

> 一句话：**八维全想到，才从"画一张好看的图"变成"拍一个会叙事的镜头"。** 配合 `分镜语法`(镜头空间) + `导演节奏`(留存时间) 三份一起过，才算导演级 prompt。
