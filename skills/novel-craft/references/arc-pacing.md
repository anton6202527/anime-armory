# 卷级大节奏守则（arc / 单卷 5-20 章的中层节奏）

`chapter.md` 管单章、`outline.md`/`split.md` 管全本三幕，二者之间缺一个**中间层**：一卷（一个 story arc，约 5-20 章）内部的情绪起伏与收束。长篇"越写越平"多半是卷级节奏塌了——每章都在动，但整卷没有蓄力—爆发的大弧。`novel-balance` 的热力图按本篇判卷级起伏。

## 一卷的标准弧线

```
卷首铺垫(1-2章) → 升温(冲突累加) → 卷中小高潮 → 转折/危机加深 → 卷末大高潮 → 卷末钩(接下卷)
```

- **卷首**：给本卷的目标/悬念（这卷主角要什么、赌注是什么），1-2 章内立住，别拖。
- **升温**：阻力逐级加码，信息差逐步揭开，蓄读者期待。
- **卷中小高潮**：一次中等兑现（小爽/小反转），防止中段疲软，也是"延迟"的中途奖励。
- **卷末大高潮**：本卷最大的兑现/翻盘/揭秘，赌注在此结算。强度必须 > 卷中小高潮。
- **卷末钩**：结算后立刻抛下一卷的钩子（强钩，见 `hooks.md`），把读者交接到下卷。

## 卷与卷之间（大节奏）

- **赌注递增**：第二卷的危机/格局必须比第一卷大（个人恩怨→门派→天下…）。原地打转 = 读者腻。
- **视野扩张同步设定揭露**：主角升级→世界变大→新设定揭露，三者绑定推进（揭露顺序进 `setting-bible.md` / wiki）。
- **张弛交替**：大高潮卷之后接一个相对舒缓的过渡卷（日常/养成/感情线推进），让读者喘口气再上强度。连续高强度会钝化。
- **不留长真空**：过渡卷也要有自己的小目标和卷末钩，不能整卷只是铺垫。

## 卷级反模式（`novel-balance` 重点扫）

- ❌ **平推无弧**：整卷每章都是匀速推进，没有可标注的卷中/卷末高潮点。
- ❌ **高潮前置**：最强场面出现在卷中，卷末反而泄气收尾。
- ❌ **卷末无钩**：一卷结束读者没有非看下卷不可的理由。
- ❌ **赌注不涨**：新卷危机量级 ≈ 上一卷，换汤不换药。
- ❌ **连续重卷**：几卷连着堆大高潮，没有张弛，读者审美疲劳。

- [ ] 本卷能标出"卷中小高潮"和"卷末大高潮"两个点，且后者更强吗？
- [ ] 卷末钩接上的下一卷，赌注比本卷大吗？
