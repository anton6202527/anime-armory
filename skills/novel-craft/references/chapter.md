# 单章写作守则

适用于任何 novel-* 写章场景；子代理 prompt 模板也在此。

## 人称选择

- **第一人称（我）** —— 沉浸感最强，对口吻一致性要求极高；只在视角角色孤独/内省时用。
- **第三人称限定** —— 默认选择。视角始终在该角色身上，只写他能感知到的。
- **第三人称全知** —— 通常**不推荐**用于视角作品。

## 视角纪律（第三人称限定下）

只写视角角色能感知的：
- ✅ 看到 / 听到 / 闻到 / 想到 / 回忆。
- ❌ 别人的内心活动（除非可由表情/动作合理推测，且写成推测）。
- ❌ 视角角色不在场的事件。
- ❌ 全知评注（"殊不知此事日后…"）。

## 不要 OOC

- **语速 / 句长**：保持原设。
- **关心什么**：原设关心 X，新章别突然关心 Y。
- **能力边界**：原设做不到的事，新章别突然会。

## 不大段复刻

派生作品场景（spinoff / expand）：
- 超过 ~30 字的原作原句 → 必删，重写。
- 多段稍作改动后照搬 → 也必删（洗稿降质）。
- 5-10 字内的角色口头禅 / 标志短句可保留。

## 节奏

- 每章一个情节小目标，开头点明、中段冲突、结尾达成或翻车。
- 每章必须推进 `设定/读者契约.md` 中至少一项：核心题旨、读者承诺、人物关系弧光、秘密揭示、能力代价或文学质感。只发生事件、但不改变读者期待的章，按注水处理。
- 单章字数随项目定：抖音漫剧/红果短剧 1000-1500 字 / 微短剧 1500-2500 / 网文中篇 3000-5000 / 网文长篇 5000-8000 / 短篇 6000-10000。权威分档见 `split.md`。
- 纯对话 ≤ 1/3。
- 长景物 / 心理 / 回忆 ≤ 200 字一气；要长就拆。
- 章末必有钩子。钩子分型 / 强度 / 失效检测见 `hooks.md`；第一章另读 `opening.md`。

## 五感与视听化写作（漫剧适配）

为了后续改编漫剧（N2D）或提升读者沉浸感，本章必须具备**高感官饱和度**：

- **五感全覆盖**：每章必须触及至少 3 种感官。除了视觉，优先加入听觉（环境音/脚步声）、触觉（温度/质感）、嗅觉。
- **空间锚点（N2D-Ready）**：描写室内或关键室外场景时，必须明确人物之间的**空间几何关系**（如：他在屏风左侧，阳光从高窗斜射，出口在他背后）。禁止只有“对话头（Talking Heads）”而无空间坐标。
- **动作细节（Cinematic）**：用“动作描述”代替“状态描述”。“她眼神躲闪，手指无意识地摩挲着茶杯边缘”优于“她很不安”。

## 文采与文笔（有节制地提升，不堆砌）

在**保住主线剧情 / 节拍 / 视角**的前提下，可以适当抬升文笔——但**文采服务叙事与人物，是手段不是目的**。

**该做（让文字有质感）**：
- 用**精准的动词和具体意象**代替笼统形容（"她攥紧袖口" > "她非常紧张"）。
- 给**有画面感的细节**（光线、气味、声响、一个小动作），让场景可感。
- **句式有节奏变化**：长句铺陈、短句收劲；爽点处短促，留白处舒缓。
- 比喻 / 通感**用在刀刃上**，一段一两处即可，且贴合该角色的认知与身份。

**别做（空洞堆砌）**：
- ❌ 形容词 / 四字成语成串叠用，华丽却不推进剧情、不刻画人物。
- ❌ 为辞藻牺牲叙事推进、人物声音或视角纪律。
- ❌ 与角色身份/口吻不符的"作者腔"抒情（视角角色不会这么想/这么说）。
- ❌ 每段都用力过猛——文采要有疏密，平实处就让它平实。

**一句话判准**：删掉这句华丽描写，剧情/人物/氛围**有损失**就留，**没损失**就是堆砌，删。

## 原创机制使用克制

凡引入原创机制（任务系统 / 卡片 / 隐藏能力 / 面板 等），**每章触发 1-2 次为佳**；超过 3 次机制就反客为主、盖过人物戏。

- **建立章（首次出现）**：可 2-3 次，让读者认识它存在。
- **中段章**：1-2 次，克制，且只在关键决策点点睛。
- **高潮章**：1-2 次，让机制反应充当反转点的"放大器"。

机制不能取代人物戏——永远给"**人物如何对这条机制做反应**"留出空间。

## 衔接

- 子代理拿到上一章末尾 ~500 字。
- 本章开头**承接**而非复述。
- 时间跳切可以（"三日后……"）。

## 子代理 prompt 模板

```
你是<原作名>的派生作者，正在写从 <视角角色> 视角的 <模式> 作品《<书名>》。

## 全局设定
[贴：角色卡.md 全文，含"口吻习惯/不要做的事"段]

## 读者契约（必须对齐）
[贴：设定/读者契约.md 全文；含核心题旨、读者承诺、好看机制、文学质感、禁偏清单]

## 世界观
[贴：世界观.md 全文]

## 章纲（全本）
[贴：章纲.md 全文]

## 文风样本（必须对齐）
[贴：章节/第01章.md（Demo 第 1 章）全文]

## 你要写的：第 N 章 《<标题>》
- 本章目标字数：[XXXX-XXXX]
- 主线事件：[章纲里的本章 outline]
- 视角情绪基调：[一句话]
- 涉及的锚点（必须对齐）：
  - 锚点 X：原作第 Y 章 Z 事件 / 在场 W / 已知 K / 未知 L
  - 对齐方式：用事件骨架对齐，不要照抄原作原文
- 上一章末尾衔接：
  [上一章最后 ~500 字]
- 本章末尾要留的钩子：[一句话]

## 写作守则
[贴：本文件全文 + 项目本地的额外口吻补充（如果有）]

## 输出
直接输出本章 markdown：第一行 `# 第 N 章 <标题>`，之后正文段落。不加额外注释或元信息。
```

## 写完自检

**硬清单（逐条必过）**：
- [ ] 字数在目标区间
- [ ] 视角全程限定，没穿帮
- [ ] 涉及锚点对齐事件骨架，未照搬原文（标志短句 ≤ 10 字可保留）
- [ ] 没让视角角色"知道"原作后文才揭示的事
- [ ] 章末有钩子
- [ ] 本章推进了读者契约里的至少一项承诺 / 题旨 / 人物弧光；没有偏成禁偏清单里的方向
- [ ] 标题和首行 `# 第 N 章 <标题>` 一致
- [ ] 文笔有质感但**不堆砌**：华丽描写删了若有损失才留（见「文采与文笔」段）

**软评估三件套（派生 / 扩写 / 续写类作品；三者同时达标才过，缺一不过）**：
- [ ] 时间线契合 ✓——锚点事件骨架对齐
- [ ] 扩写丰富度 ✓——环境 / 内心戏 / 对话细节 / 留白填补，不是文本搬运
- [ ] 人物厚度 ✓——多面 / 层次 / 镜像对照

> ⚠️ 仅对齐原著不够，仅原创扩写也不够：**对齐 + 扩写 + 人物厚度三者同时**才算合格。
