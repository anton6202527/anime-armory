# 扩写法

把短文扩成长文。**保骨架 + 加细节**。

## 加什么（按对人设/弧线影响从小到大）

1. **环境描写**（最安全）：天气、气味、光线、人物表情、室内布置。完全不影响弧线。
2. **内心戏**：视角角色的想法、回忆、犹豫、判断。极大丰富人物厚度。
3. **细节对话**：把原文的"说了几句"展开成有节奏的对话。
4. **次要互动**：和路人、随从、动物的简短交集。可加色彩。
5. **支线场景**（最危险）：插入一段独立小事件。**只在确认不打断主线节奏时做**。

## 不加什么

- **新主线事件** —— 扩写不是改写，主线骨架不能动。
- **新主要人物** —— 除非源文已有伏笔，不要无中生有。
- **新设定 / 新规则** —— 会推翻源文的设定一律不加。
- **结局变化** —— 绝对禁止。

## 比例参考

| 扩写倍率 | 加什么为主 |
|---|---|
| 1× → 4× | 内心戏 + 环境 + 对话细节 |
| 1× → 8× | 同上 + 次要互动 + 1 个支线 |
| 1× → 15×+ | 同上 + 多个支线，必须在章纲层规划 |

超过 10× 应当 plan-then-execute：先 outline 扩写后的结构，再逐段执行。

## 工作流（与具体段落配合）

1. 把原文拆为**事件骨架**（每一句对应一个事件 / 一段对话 / 一段心理）。
2. 决定每个骨架点要扩成多少。
3. 一段一段扩，每段保持原句的**骨**（人物 / 动作 / 结果），换**肉**（描写 / 内心 / 细节）。
4. 全文扩完，回读一次。如果支线插太多打散节奏 → 先精简再交付。

## 反模式

- **均匀扩写**：每段都扩 4× → 节奏均一变沉闷。要有详略对比。
- **加新关键决定**：扩写不是给人物加新选择，是把已有选择写厚。
- **新的"其实"**："其实他一直爱她"这种内心新揭示属于改写不是扩写。
