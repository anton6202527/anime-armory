# 精简法

把长文压缩成短文。**保主线 + 砍枝节**。

## 砍什么（按砍的安全性从高到低）

1. **景物描写**（最安全）：长段写景几乎都可砍到 1-2 句。
2. **次要支线**：和主线无关的短章 / 小桥段，砍。
3. **重复内心戏**：同一情绪反复表达的段落，留一处删其它。
4. **路人 / 工具人对话**：与主角推进无关的对话，砍。
5. **回忆 / 闪回**：若主线已能展示，删除回忆段。
6. **多余的反转**：一章里反转 ≥ 3 次的删 / 合并。

## 保什么（绝不砍）

- 主角的**关键决定** + **后果**
- 所有**锚点事件**（如果是基于源作品的）
- **反转点**（哪怕只剩一句话也要保留）
- **钩子**（章末或段末的悬念）
- **关键人物首次出场**
- **大高潮的情绪击穿**

## 比例参考

| 原文 → 压缩 | 推荐策略 |
|---|---|
| 1.5–2× | 砍描写 + 砍重复内心。基本不动情节 |
| 3–5× | 上 + 合并相邻章 + 砍次要支线 |
| 10× | 上 + 大刀阔斧合章 + 用一句话概述长段。结构会变 |
| 20×+ | 不是精简了，是"提取大纲" |

漫剧友好压缩：每集 90-120 秒 ≈ 1500-2500 字脚本 → 100k 字原作通常压到 30-50 集脚本。

## 工作流

1. 标**主线 / 锚点 / 反转点**（不能动）。
2. 决定每一段的去留：保 / 削 / 删。
3. **合章**：相邻同主题章合并为一章，标题与节奏重排。
4. 一段段重写：保留骨架，砍细节。
5. 全本回读：节奏是否仍连贯 / 钩子是否仍存在。

## 反模式

- **均匀压缩**：每段都砍 50% → 高潮段会失去击穿力。要重点保高潮。
- **砍了反转点凑字数** → 反转点是钩子来源，必须保。
- **漫剧版仍按原章节切** → 漫剧要按戏剧节拍切，跨章合理。
- **大段照搬原作高潮段** → 即便是高潮段也要重写，不搬文本。
