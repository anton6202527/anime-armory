# 章纲编织法

适用于 novel-spinoff / novel-expand / novel-condense 等需要章纲的场景。

## 三幕结构

适用于 ≥ 12 章长度。< 12 章不强套三幕。

- **第一幕 ~1/4 章数** —— 建立人物与世界。
- **第二幕 ~1/2 章数** —— 主线推进，锚点密集，中段必有一次大反转。
- **第三幕 ~1/4 章数** —— 高潮与收束。

## 反转点

- 至少两次。
- 一次在第二幕中段（约 40% 进度），一次在第三幕入口（约 75% 进度）。
- 反转 = 人物身份 / 关系 / 目标的实质改变，不是情节波动。

## 钩子

每章末尾留一个"下一章你想看的东西"——悬念 / 误会 / 决定 / 新人物。**没有钩子的章是失败章。**

## 锚点织网（spinoff / expand 用）

- 每个锚点至少一章覆盖。
- 锚点章前后做一对"赴约章 + 复盘章"——加情绪密度。
- 早期锚点稀疏 → 前几章可以无锚点纯建视角。
- 末期锚点密集 → 靠章纲拉节奏，不要每章都对锚点。

## 输出格式

```markdown
# 章纲 — 《书名》

## 总体弧线
（一段话）

## 三幕结构
- 第一幕（第 1 - N 章）：...
- 第二幕（第 N+1 - M 章）：...
- 第三幕（第 M+1 - 末章）：...

## 锚点-章节映射
| 锚点 ID | 原作位置 | 本作章节 | 对齐点 |

## 逐章
- 第 1 章 《标题》 — 主线事件 / 转折 / 锚点 / 情绪 / 钩子
- ...
```

## 反模式

- **流水账**：每章只推一点点情节，没起伏
- **无反转**：只一条直线，没有身份/目标的实质改变
- **钩子断章**：钩子和下一章无关
- **锚点全堆中段**：节奏跳变
