# Specialized Agent Trio Pipeline (三步迭代创作管线)

为了彻底消除"AI味"并极大提升文学质感，长篇小说的单章写作从"单步直出"升级为"三步独立角色接力"（Architect → Ghostwriter → Senior Editor）。

在使用 `draft_packets.py` 时，通过传入 `--step` 参数，为每一章生成专属于该角色的任务包。

## 1. Architect Pass（剧情架构师）
**目标**：将骨感的章纲转化为「节拍与潜台词（Beats & Subtext）」脚本。
**输入**：前文摘要、本章章纲、Tension Ledger。
**禁止**：禁止写优美的文学散文，禁止堆砌辞藻。
**输出**：纯干货的节拍列表（Bullet points）。
- 每个节拍必须包含：
  - **动作/事件**（发生了什么）
  - **POV 内心活动/潜台词**（角色没说出口的真实想法）
  - **情绪锚点**（这个节拍的张力是上升还是释放？）

## 2. Ghostwriter Pass（代笔枪手）
**目标**：将 Architect 的节拍脚本转化为流畅的小说正文，严格复刻文风指纹。
**输入**：Architect 产出的节拍脚本、文风指纹（Style Fingerprint）、N2D 资产标签要求。
**禁止**：禁止擅自增加或删减 Architect 定好的剧情节拍；禁止使用套路化成语（如"倒吸一口凉气"）。
**输出**：第一版初稿正文（Draft）。

## 3. Senior Editor Pass（资深主编润色）
**目标**：专门进行「五感丰富」与「去AI味」的滤网打磨。
**输入**：Ghostwriter 初稿正文。
**任务指令**：
- **杀掉抽象词**：将"他感到非常恐惧"改为生理性描述（"喉咙里泛起铁锈般的血腥味，指尖冰凉"）。
- **动词升级**：扫灭平庸动词（"走"、"看"），换成有性格的动态词。
- **环境映衬**：将场景中的光影、气味与人物的潜台词融合（Show, don't tell）。
**输出**：最终交付正文（Polish）。
