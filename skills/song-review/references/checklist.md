# 作品质检 —— 全维度清单（看什么 · 怎么判 · 定级）

> 落到 song 写歌产线。**机** = `song_check.py` 确定性查；**判** = LLM 语义判（词）/ 用户试听判（曲+演唱）。容错铁律：只记真问题。
> 正向标尺：词 = `song-lyrics/references/songcraft.md`；曲 = `song-compose/references/backends.md` + `创作蓝图.md`。

## A. 词 · 可唱性与质量（对 `songcraft.md`）

| 维度 | 机/判 | 怎么查 | 定级 |
|---|---|---|---|
| 占位未精修（`待`/`TODO`/`（待`） | 机 | 扫 `词/lyrics.md` 全文 | 🔴 |
| 缺副歌 / hook | 机+判 | 机查有无 `[chorus]` 段；判副歌是否有 1 句最抓耳、可重复、易跟唱的 hook | 无副歌 🔴 / hook 弱 🟡 |
| 字数贴旋律（同段字数相近） | 机+判 | 机查同段各行字数离散度（极差过大标 🟡）；判是否拗口难唱 | 离散大 🟡 |
| 押韵 | 判 | 一段一韵脚（或规律换韵），不为押而凑、不乱跳；宁可不押也别拗口 | 乱跳/拗口 🟡 |
| 顺口 / 口语 | 判 | 念出来顺，无堆典、无现代俚语乱入（除非曲风需要） | 🟡/🟢 |
| 结构完整 | 机+判 | 机比段落数 vs `_meta.structure`；判主歌蓄力→副歌爆发→桥段反转是否成立 | 段数不符 🟡 |
| 平台/曲风套路 | 判 | 抖音前 8 秒出 hook、副歌前置、洗脑重复；说唱密押 flow；国风意象——对 `_meta.target_platform`/`genre` | 🟡 |
| 与蓝图一致 | 判 | 主题/情绪/视角/禁忌是否守 `创作蓝图.md` | 漂 🟡 |
| 原文照搬 / 抄袭 | 判 | 非原创、照搬已发行歌词 | 🔴 |
| AI 味 / 同质化 | 判 | 空泛口水、万能比喻、无识别度 | 🟡 |

## B. 曲 + 演唱 · 听感（**用户试听清单** —— LLM 听不到 wav）

> 本 skill 生成下表交用户播放 `歌/song.wav` 逐条打钩；客观指标（采样率/削波/静音/时长）交机检。**报告里把用户反馈的不合格项落成 finding。**

| 试听项 | 听什么 | 定级 |
|---|---|---|
| 曲风贴合 | 出来的曲风/配器是否对 `创作蓝图.md`（如国风武侠 vs 出了纯电子） | 跑偏 🟡 |
| 旋律记忆点 | 副歌旋律是否抓耳、可哼唱、有 hook | 平淡 🟡 |
| 人声清晰 / 咬字 | 唱词听得清、咬字准、不糊不飘 | 糊/听不清 🟡 |
| 人声 ↔ 伴奏平衡 | 人声没被伴奏盖住，也不突兀干贴 | 失衡 🟡 |
| 演唱情绪 | 情绪与词/蓝图贴（燃/悲/甜），不平不尬 | 🟡 |
| 结构对齐词 | 段落顺序、副歌出现位置与 `lyrics.md` 一致 | 错位 🟡 |
| 整体听感 | 有无明显机器感/拼接断裂/跑调 | 严重失真 🔴 / 一般 🟡 |

## C. 音频 · 客观指标（确定性 → 机检）

| 维度 | 机/判 | 怎么查 | 定级 |
|---|---|---|---|
| `歌/song.wav` 存在 | 机 | 未作曲（读 `_进度.md`）则正常，不报 | 缺且应有 🟡 |
| 全静音 / 近静音 | 机 | 峰值 < ~ -40dBFS | 🔴 |
| 损坏 / 不可解析 | 机 | wave 解析失败（非 WAV？）| 🔴 |
| 削波 clipping | 机 | 峰值贴满量程（≥0.995）且占比高 → 失真 | 🟡（严重 🔴）|
| 采样率 | 机 | < 44100Hz 投放偏低 | 🟡 |
| 时长与平台 | 机+判 | 抖音神曲过长 / demo 片段当成品 | 🟡 |

## D. 合规（非交涉项，每次必查）

| 维度 | 机/判 | 怎么查 | 定级 |
|---|---|---|---|
| 演唱音色来源 | 机+判 | `_meta.vocal_source` 须为 自有/授权/合成；缺失或标真人未授权 | 缺/未授权 🔴 |
| 声音克隆授权 | 判 | 克隆真人嗓有授权（2026 opt-in） | 未授权 🔴 |
| 翻唱词曲版权 | 判 | 翻唱已发行曲，商用需原作者授权（自有原创随意） | 未授权商用 🔴 |
| 词曲权利状态 | 机 | `_meta.rights_status`（original/licensed/…）已记 | 缺 🟡 |

## E. 完整性 / 对账（机检）

| 维度 | 定级 |
|---|---|
| `创作蓝图.md` / `_进度.md` / `_meta.json` 齐全 | 缺 🟡 |
| `_meta.structure` 段落数 vs `lyrics.md` 实际段数一致 | 不符 🟡 |
| 进度表勾选 vs 实际产物（song.wav / lyrics）一致 | 不符 🟡 |
| 产物快照（段落数 / 字数 / 时长 / 采样率 / 声道） | 🟢 信息 |
