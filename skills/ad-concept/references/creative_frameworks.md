# 广告创意工艺库（ad-concept 参考）

不是模板套路清单，是帮 AI 把 brief 转成有记忆点创意的思考脚手架。

## 创意路线（选一条主轴，可混）

| 路线 | 适用 | 钩子机制 | 风险 |
|---|---|---|---|
| **功能卖点** | 产品有硬差异（参数/技术/价格） | demo 可视化「之前 vs 之后」 | 易平庸、像说明书 → 要有一个戏剧化呈现 |
| **情感共鸣** | 品牌片/快消/服务，差异不在参数 | 把产品接到一个真实情绪/关系/场景 | 易煽情空洞 → 情绪要落到具体细节 |
| **幽默** | 年轻受众/社交传播 | 反差/夸张/自嘲制造笑点 | 笑点盖过品牌 → 品牌/产品要是笑点的一部分 |
| **悬念反转** | 信息流前 3 秒留人 | 先抛反常/悬念，结尾揭晓=产品 | 反转与产品无关=骗点击 → 揭晓必须是产品 |
| **名人/达人代言** | 有授权代言人 | 借代言人信任与流量 | 肖像/言论需授权；代言人盖过产品 |
| **场景种草** | 电商/带货/小红书 | 真实使用场景 + 痛点解决 | 像硬广 → 场景要真、痛点要准 |

## 黄金 3 秒钩子（信息流/短视频必备）

前 3 秒决定划走还是看完。钩子类型：
- **冲突/反常**：打破预期的画面或台词。
- **痛点直击**：「你是不是也……」对号入座。
- **利益前置**：「3 秒教你……」「9.9 拿下……」。
- **悬念**：抛一个不揭晓不舒服的问题。
- 钩子要与产品相关，别为留人而留人（揭晓必须回到产品/主张）。

## 段落级故事线模板（按主片时长伸缩）

```
[0–3s]  钩子        ← 黄金3秒，反常/痛点/利益/悬念
[3–Xs]  情境/痛点    ← 放大「没有它」的代价或情绪
[X–Ys]  产品/方案    ← 产品登场，hero shot，USP 可视化
[Y–Zs]  证据/记忆点  ← demo/数据/对比/金句，建立信任
[尾]    CTA/品牌包装 ← slogan + logo + 行动指令(end card)
```
- 6s：钩子 + 产品 + slogan（极压缩，一个记忆点）。
- 15s：钩子 + 痛点 + 产品 + CTA。
- 30s：完整五段。
- 60s/品牌片：情感线可拉长，证据段可加叙事。

## big idea 自检

- 能不能用一句话说清「这片靠什么让人记住」？
- 换成竞品还成立吗？成立=不够独占，要再逼近品牌真理。
- 一句话主张能落到 KV 和片尾 slogan 吗？落不到=策略与执行脱节。

## mood & reference 写法

描述**气质**而非抄**作品**：光色（冷暖/高低调）、质感（胶片/数字/CG/手持）、节奏（慢推/快切）、色彩主调（呼应品牌色）。给 `ad-image` 当 `基础视觉风格` 与 KV 锚，不复制他人具体镜头。

## 与广告法的前置默契

功效/对比/数据/排名类 claim 在创意阶段就标注「需依据」。绝对化用语（最/第一/国家级/顶级/唯一）、医疗保健极限词在脚本阶段会被 `ad-script/ad_law_check.py` 拦截——创意阶段就用合规替代表述（如「更」而非「最」、「之一」而非「第一」、具体可证数据而非笼统夸大）。
