# 改写工艺：改动spec + 新设定圣经

改写区别于外传的两件核心产物。外传靠"锚点表"锁住原作；改写靠这两份文件**既放开手脚、又不写崩**。

---

## 一、改动spec（`设定/改动spec.md`）—— 这部改写的"宪法"

三栏，动笔前必须和用户敲定。每条要**具体、可判定**，不要"让它更精彩"这种空话。

```markdown
# 改动spec — 《<原作>》→《<新书名>》

## 一句话改动方向
（把"X 的故事"改成"Y 的故事"——一句话说清这次重做的灵魂）

## ① 保留的内核（魂，不许丢）
- 主角人设内核：（如"表面怂、底里轴"——事件可换，这个魂不变）
- 情感主线：（如"兄弟/师徒情"——载体可换，情不变）
- 世界观底色 / 基调：（如"清苦修真 + 黑色幽默"）
- 必须保留的标志性桥段/意象：（可选，几个原作里不能丢的高光）

## ② 改的部分（事件 / 设定 / 结局）
- 改主线走向：（原作是…→ 改成…）
- 改/删的事件：（哪些原作情节砍掉或重写）
- 改的设定：（原作设定 X → 改成 X'；注意：改设定要在 ③ 里登记新值）
- 改的结局：（如有）

## ③ 加的新料（原创设定 / 材料 / 势力 / 金手指 / 人物）
- 列出本次新增的所有原创元素（详细体系进"新设定圣经"，这里只列清单 + 一句话）
- 新金手指/系统：
- 新势力/组织：
- 新人物：
- 新地理/秘境/物品：
```

> **铁律**：凡 ② 改了某条原作设定、③ 加了某个新元素，**都要在"新设定圣经"里登记成可复用的条目**，否则后面逐章写必然漂。

---

## 二、新设定圣经（`设定/新设定.md`）—— 改写第一生产资料

用户说"会增加各种其他设定、材料"——它们全进这里，并且**带一致性约束**（continuity bible 思路：每条设定写明"是什么 + 规则/代价 + 不可违反点"）。

```markdown
# 新设定圣经 — 《<新书名>》

> 本作相对原作【新增/改写】的所有设定。逐章写作时是硬约束，回扫时逐条核。
> 与原作旧设定冲突的，以本表为准；本表内部不许自相矛盾。

## 体系（金手指/力量/修炼等）
### <名称>
- 是什么：
- 规则 / 边界：
- **代价 / 限制**（防无敌流崩盘）：
- 首次出现：第N章 ｜ 复用：
- 不可违反点：

## 势力 / 组织
### <名称>　立场/目的/与主角关系/首现章

## 新人物
### <名称>　身份/外貌锚定/性格/动机/说话习惯/首现章/复用范围

## 地理 / 秘境 / 物品
### <名称>　描述/规则/首现章

## 改写后的旧设定（覆盖原作的）
| 原作设定 | 改成 | 影响范围 |
|---|---|---|
```

### 一致性追踪做法
- 新名词/新规则一旦定，**全书统一**（名称、数值、代价不许漂）——交给 `novel-review` 的术语统计 + 人判核。
- 新金手指**必须有代价/限制**，否则改写极易滑向无敌爽崩。
- 改了的旧设定，要全篇排查"有没有地方还按原作旧值写"（回扫重点）。

---

## 三、与 novel-spinoff 的边界（写进路由判断）

| 维度 | novel-spinoff（外传） | novel-rewrite（改写） |
|---|---|---|
| 原作事件 | **锁定**（锚点对齐，绝不改） | **可改**（按改动spec ②） |
| 新设定 | 不发明能推翻原作的新规则 | **主动新增/覆盖**（新设定圣经） |
| 视角 | 固定某配角 POV | 任意（常是新主角/重塑主角） |
| 地基文件 | `锚点表.json` | `改动spec.md` + `新设定.md` |
| 一句话 | 换人看同一个故事 | 拿这个故事重做一个 |

**临界判断**：
- "只在原作某节点分叉、之后走别的路，但人物/世界基本沿用" → 其实是 spinoff 的 **branch 模式**，未必要 rewrite。
- "换主角 + 大改主线 + 塞一套新金手指/新势力" → 是 rewrite。
- 拿不准时看**有没有新设定圣经要建**：要大量新增/覆盖设定 → rewrite。

---

## 四、回扫重点（rewrite 专属，喂给 novel-review）

除 novel-review 通用维度外，改写额外查：
1. **新设定一致性**：新设定圣经每条，全篇是否前后一致（名称/数值/代价）。
2. **没跑回旧设定**：被 ② 改掉的原作设定，全篇有没有地方还按原作旧值写。
3. **内核没丢**：改动spec ① 的"保留内核"，在成稿里是否真的还在（人设魂/情感主线）。
4. **没照搬原文**：改写比外传更需重写，原文照搬检查照跑（机检 n-gram vs 原作.txt）。
5. **新金手指不崩**：是否有代价、有没有滑向无敌流硬伤。
