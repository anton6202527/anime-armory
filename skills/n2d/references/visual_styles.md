# 基础视觉风格（选择点 `基础视觉风格`）

本文件只定义可选风格和派生规则。用户的实际选择写入 `<作品根>/_设置.md`，不写进 skill 源码。新项目在写 `设定库/global_style.md` 前先定本项；已有项目按 `_设置.md` / `global_style.md` 沉默沿用。脚本侧预设落在 `skills/common/n2d_visual_styles.py`，改菜单或预设时需要同步本页与脚本。

## 使用规则

1. 读取 `基础视觉风格`：`_设置.md` → 全局默认 → 首次给菜单让用户选一次。
2. 将选择写进 `设定库/global_style.md` 的「基础视觉风格」和「基础视觉风格契约（style_contract 源头）」。
3. 阶段2 分镜时在 `storyboard.json` 写 `style_contract`，字段固定为：`风格名 / 视觉基调 / 镜头与构图 / 光色策略 / 运动边界 / 风格禁忌`。
4. `n2d-image` 和 `n2d-video` 只继承并细化 `style_contract`，不得把某一种风格（如写实电影感）重新追加成硬规则。
5. 旧项目已有 `cinematic_contract` /「本集真实电影感契约」时可以继续通过 gate；下次修改故事板或总览时迁到 `style_contract` /「本集基础视觉风格契约」。

## 菜单

| 值 | 适用 | `style_contract` 方向 |
|---|---|---|
| 写实电影感 | 真人感、古装短剧、悬疑、宫斗、现代短剧 | 写实电影剧照、可信焦段、动机光、低到中饱和、克制运镜；禁塑料皮肤/过磨皮/游戏CG/插画化 |
| 国漫写实 | 仙侠、玄幻、东方幻想、爽文漫剧 | 国漫角色比例 + 写实材质和场景；光效可更戏剧化但来源清楚；禁欧美脸漂移/页游塑料/随机霓虹 |
| 二次元赛璐璐 | 校园、轻喜、青春、偶像、轻奇幻 | 清晰线稿、平涂分层、干净高光、镜头语言仍按分镜走；禁照片皮肤/重胶片颗粒/3D塑料 |
| 水墨国风 | 古风、武侠、诗意段落、片头片尾 | 水墨留白、宣纸纹理、墨色层次、构图重气韵；运动边界偏慢；禁厚重写实皮肤/高饱和霓虹/现代CG感 |
| 厚涂幻想 | 高魔、史诗、怪物、世界观展示 | 厚涂笔触、体积光、强材质、宏大景别；运动可有史诗推拉但仍服务节奏；禁低幼Q版/薄平涂/廉价页游UI感 |
| 赛博霓虹 | 未来、都市异能、科幻、夜景动作 | 高对比霓虹、玻璃/金属/雨夜反射、冷暖撞色；运镜可更锐利但不乱甩；禁古风材质串味/脏灰低能见度 |
| Q版轻喜 | 搞笑、萌系、低龄向、番外小剧场 | 低头身、圆润造型、明亮色块、表情夸张；运动更弹性；禁恐怖写实血腥/成人写实脸/复杂脏纹理 |
| 自定义 | 用户给参考图、作品名、平台样片或口头描述 | 先复述为 `style_contract` 六字段让用户确认，再写入 `_设置.md` 和 `global_style.md` |

默认建议是 `写实电影感`，但它只是预选，不是规范铁律。若用户说“更像国漫”“二次元”“水墨”“赛博”“Q版”等，立即覆盖本选择点并同步 `global_style.md`。

## 派生示例

```json
{
  "style_contract": {
    "风格名": "国漫写实",
    "视觉基调": "东方幻想国漫，角色比例略理想化，场景和服装材质写实，高细节但不照片化",
    "镜头与构图": "保留影视景别和轴线；可用更强剪影、广角压迫和法术特写，但不随机变透视",
    "光色策略": "青金/墨黑为主，法术高光只在爽点出现；强光来自月光、烛火、符阵或兵器反光",
    "运动边界": "慢推、固定、跟摇为主；爽点可短促环绕或轻甩，禁止无理由飞行镜头",
    "风格禁忌": ["欧美脸漂移", "页游塑料盔甲", "随机霓虹", "过度磨皮", "背景像贴图", "低幼Q版"]
  }
}
```
