# mv-image 出图 prompt 格式（两层 + 锚点）

## 两层架构
- **共享层** `出图/共享/图片/`：
    - `prompt/` 子目录：角色定妆.md、场景定妆.md
    - `图片/` 子目录：定妆_<角色>.png、定妆_<场景>.png
- **Clip 层** `出图/段落/图片/`：
    - `prompt/` 子目录：`Clip_XXX.md`（由 `mv-plan` 生成）
    - `图片/` 子目录：`Clip_XXX.png` 首帧；按需 `Clip_XXX_end.png` 尾帧

## 角色卡锚点句（每张分镜 prompt 末尾必拼）
`设定/characters/<主角>.md` 写：固定外貌/发型/服装/色卡 + **锚点句**（3-5 不可漂特征压成一句）。

## 分镜 prompt = 导演视角八维（不是画师视角的"好看主角图"）
详见 `mv/references/导演视角prompt.md`。按八维装配：
```
[①镜头][②机位]，[③歌手·锚点句·本段妆造] 正在 [④情绪最强一瞬],
置身 [⑤场景+环境氛围]，[⑥演出光+色胶+调性]，[⑦本段情绪+张力+色调]，[⑧MV画风+渲染+画幅][锚点句]
```
MV 最易漏三维：**②机位**（副歌大胆机位/荷兰角）、**⑥光影**（演出光/色胶/逆光剪影，别均匀打亮）、**⑦张力**（对齐 beatgrid 段落）。**定妆图是中性档案**（正面/均匀光），戏剧光只上分镜图。

## Clip ↔ 画面（按 clip_plan + 视觉蓝图 + beatgrid）
`分镜/clip_plan.json` 是出图任务真值源：读每个 clip 的 `image_prompt_path`、`image_path`、`section`、`lyric_hint`、`visual_style`、`continuity` 和 `need_end_frame`。不要在出图阶段重新拆 clip；若缺 plan，先跑 `mv-plan`。

| 段落 | 画面取向 | 分镜疏密 |
|---|---|---|
| intro | 氛围/空镜/主角入场 | 疏 |
| verse | 叙事推进、生活/情绪场景 | 中 |
| pre-chorus | 蓄力、镜头逼近 | 中→密 |
| chorus | 高能、主角高光、强色彩 | 密（踩鼓点切，多备镜） |
| bridge | 反转/换景/换色 | 中 |
| outro | 收束/留白/定格 | 疏 |

## 生图调用
- **生图后端是选择点（默认 Codex）**：阶段1 放行官方多参考后端（Seedream/可灵主体库/Nano Banana/Sora Cameo）。一支 MV **统一一个官方后端、不混用**；**禁即梦/Dreamina 逆向出图**（官方 Seedream API 可用）。后端清单见 `mv-craft/scripts/contract.py`。
- 跨段一致性靠：共享定妆参考图 + 锚点句 + 统一 global_style；选支持原生主体的后端（Seedream/可灵/Sora Cameo）可注册主体按 ID 引用更稳。
- 竖屏 MV 9:16 / 横屏 16:9，与 视觉蓝图 画幅一致。

## 出图前自查（prompt 装配）
- [ ] 共享定妆先出、锚点句已定？
- [ ] 每张 clip 首帧拼了锚点句，并写到 `clip_plan.json` 指定的 `image_path`？
- [ ] 段落取向 + 疏密对（副歌密、verse 疏）？
- [ ] 全程画风统一，且整支 MV 只用同一个官方生图后端（不混用、非即梦逆向）？
- [ ] `need_end_frame=true` 的 clip 已补 `Clip_XXX_end.png`？
- [ ] **导演视角八维齐**？尤其 ②机位（非默认正面平视）/ ⑥光影（演出光/色胶非均匀打亮）/ ⑦张力（对齐 beatgrid 段落）三维没偷懒？
- [ ] 定妆图保持中性档案（没打戏剧光/色胶）？

## 生成后自检（每张过 · 落档闸门）
> 与上面"出图前自查"互补：这是**生图后**的过/重抽判定。按**筛选宽容铁律**——轻微偏差直接放行，只命中硬伤才重抽；重抽上限由 `重抽预算策略` 档位决定（见 mv-image SKILL.md「重抽预算铁律」三档表）。每张分镜建议在出图清单/段落记录里带一行：
```markdown
**自检**（轻微偏差放行，只命中硬伤才重抽）：
- [ ] 核心人/物/场景无错位 — 本镜须含：<该出现的主角/场景/道具>
- [ ] 主角脸/画风未漂移（对照 出图/共享/图片/定妆_<主角>.png；主角零漂移容忍）
- [ ] 无硬性禁忌（景别须=<本镜景别>；演出光/色胶到位非均匀打亮；无字幕/logo/多余手指）
- 重抽预算：<主角/副歌高光镜 or 配角/普通段镜，按当前策略档填上限> ｜ 实抽__次 → ⬜过 ⬜重抽 ⬜到顶取一致性最优版
```
- **副歌高光镜 / 爽点 / 反转 / 封面候选** = 高优先镜，套用更宽的重抽上限（同 n2d 的 🔑关键镜）。
