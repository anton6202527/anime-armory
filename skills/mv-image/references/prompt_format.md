# mv-image 出图 prompt 格式（两层 + 锚点）

## 两层架构
- **共享层** `出图/common/`：主角定妆（A-pose 干净背景 + 脸特写）、关键场景定妆。全曲复用。
- **分段层** `出图/段落/`：各段落分镜，基于共享定妆（参考图/图生图）+ 锚点句。

## 角色卡锚点句（每张分镜 prompt 末尾必拼）
`设定/characters/<主角>.md` 写：固定外貌/发型/服装/色卡 + **锚点句**（3-5 不可漂特征压成一句）。
分镜 prompt = `主体 + 动作/表情 + 段落场景 + 景别 + 画风词 + [锚点句]`。

## 段落 ↔ 画面（按 视觉蓝图 + beatgrid）
| 段落 | 画面取向 | 分镜疏密 |
|---|---|---|
| intro | 氛围/空镜/主角入场 | 疏 |
| verse | 叙事推进、生活/情绪场景 | 中 |
| pre-chorus | 蓄力、镜头逼近 | 中→密 |
| chorus | 高能、主角高光、强色彩 | 密（踩鼓点切，多备镜） |
| bridge | 反转/换景/换色 | 中 |
| outro | 收束/留白/定格 | 疏 |

## 生图调用
- 同一首 MV 全程**同一生图工具**（即梦/Flux/Gemini 三选一，别中途换，防画风跳）。
- 跨工具/跨段一致性靠：共享定妆参考图 + 锚点句 + 统一 global_style。
- 竖屏 MV 9:16 / 横屏 16:9，与 视觉蓝图 画幅一致。

## 自查
- [ ] 共享定妆先出、锚点句已定？
- [ ] 每张分镜拼了锚点句？
- [ ] 段落取向 + 疏密对（副歌密、verse 疏）？
- [ ] 全程画风/工具统一？
